# Human Reviews

## Purpose
Human reviews apply the V0.14 quality rubric to narrative_tool drafts after technical QA passes.

## Review status scale
- technical_pass_only — passes structure checks but needs editorial review or revision.
- needs_editorial_revision — has clear human-quality issues to fix.
- editorial_pass — good enough editorially, with only minor notes.
- validated_reference_quality — strong reference for future generation.
