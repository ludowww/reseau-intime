# Narrative Tool Docs

Documentation de travail pour la continuité mémoire et les patterns de contrats de scène.

## Documents disponibles

- `memory_continuity_model.md` — modèle de continuité mémoire, couches, callbacks et resets interdits.
- `scene_contract_patterns.md` — patterns de contrats de scènes, champs requis et checklist de revue humaine.

## Usage

Ces documents servent de base éditoriale avant d’écrire ou d’itérer sur une nouvelle route.
Ils expliquent la fonction narrative attendue, pas une scène canonique à recopier.
