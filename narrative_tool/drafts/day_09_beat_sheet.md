J8 a recentré le joueur : Raphaëlle puis Marie.

## Emoji and phone-language rule
Les emojis de J9 doivent être présents là où la voix le justifie.
Ils ne doivent pas être uniformes.
Ils ne doivent pas être ajoutés mécaniquement.
Leur fréquence dépend du personnage, du stade d’intimité, du risque et de l’état de route.
L’absence d’emoji peut être un effet narratif si elle est assumée.

## Scene 1 — Sandra / relance du manque

### Function
relancer Sandra après J8 sans la rendre disponible.

### Entry trigger
prétexte concret lié à une photo, un ancien déjeuner, une trace oubliée, un détail qui revient.

### Emotional movement
banal → gêne → phrase trop vraie → esquive → promesse douce ou relance retenue.

### Player choice function
le joueur peut répondre avec prudence, curiosité ou trouble assumé léger.
Pas de choix de confession lourde.
Pas de choix de bascule sexuelle directe.

### Visual hook
au moins 1 photo Sandra douce, cadrée comme trace relationnelle, pas preuve, pas adulte.

### Emoji direction
rares, retenus, mais plus chargés que J8 si la gêne monte.
🙂 ou 😅 peuvent servir de masque.

### Beat outline
1. Sandra revient avec un prétexte concret.
2. Elle mentionne une photo / trace / souvenir sans en faire un appât évident.
3. Player peut répondre prudemment, sincèrement ou ironiquement.
4. Sandra laisse apparaître qu’elle a aussi attendu.
5. Une photo douce ou trace visuelle nourrit le manque.
6. Le fil frôle une phrase trop vraie.
7. Sandra esquive par humour ou changement léger.
8. La scène se ferme sur une envie de se revoir / se reparler, sans promesse explicite.

### Exit state
Player a envie de revoir / réentendre Sandra.
Sandra reste touchée mais prudente.
La tension monte, rien n’est verrouillé.

### Guardrails
- pas de disponibilité totale ;
- pas de confession lourde ;
- pas de bascule sexuelle directe ;
- pas de route verrouillée ;
- pas de panel.

## Optional Scene 2 — Marie / coût indirect

### Function
rappeler que Player a une vie de couple, sans transformer Marie en obstacle.

### Entry trigger
message domestique, fatigue, photo quotidienne, détail de présence/absence.

### Emotional movement
quotidien → douceur fatiguée → petit manque → réparation possible.

### Player choice function
répondre présent, esquiver doucement, ou compenser maladroitement.
Pas de mensonge lourd.
Pas de confrontation Sandra.
Pas de jalousie explicite.

### Visual hook
photo quotidienne ou trace domestique.
Pas de preuve.
Pas de scène de suspicion.

### Emoji direction
modérés ; 🙂 / 🙄 / 😅 ou absence assumée selon la fatigue.

### Beat outline
1. Marie écrit pour un détail concret.
2. Le quotidien paraît simple mais un peu usé.
3. Player choisit une forme de présence.
4. Marie répond avec tendresse ou fatigue légère.
5. Une trace visuelle quotidienne peut garder le couple incarné.
6. La scène se ferme sans reproche frontal.

### Exit state
Marie existe encore comme lien vivant.
Le coût émotionnel de Sandra est sensible mais indirect.

### Guardrails
- pas de confrontation ;
- pas de reproche frontal ;
- pas d’obstacle moral simplifié ;
- pas de vol de l’axe Sandra.
