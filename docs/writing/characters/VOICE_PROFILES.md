# Profils de voix adaptatifs

## Principe

Une voix personnage ne doit pas être figée.

Chaque personnage possède :

```text
- une identité stable ;
- des variations selon le stade relationnel ;
- des variations selon le risque ;
- des limites de connaissance ;
- des choses qui peuvent évoluer ;
- des choses qui ne doivent pas arriver trop tôt.
```

Le but n’est pas de générer plus de branches, mais d’écrire la bonne nuance dans une scène donnée.

## Stades relationnels génériques

```text
stage_0_distance : relation sociale ou fonctionnelle, peu d’intimité.
stage_1_familiarite : confort léger, humour prudent, début de sous-texte.
stage_2_complicite : messages plus naturels, références partagées, confiance partielle.
stage_3_intimite_naissante : le personnage ose plus, mais protège encore ses limites.
stage_4_tension_assumee : désir ou attachement plus net, risque visible.
stage_5_lien_secret : forte charge émotionnelle ou compromettante, à utiliser plus tard.
```

## Niveaux de risque

```text
low : discussion défendable.
medium : sous-texte visible, malaise ou désir perceptible.
high : trace compromettante, photo, mensonge ou risque relationnel clair.
critical : preuve forte ou conséquence irréversible — à éviter trop tôt.
```

# Marie

## Voix stable

Marie est l’ancrage affectif et conjugal. Elle parle avec une douceur quotidienne, mais elle voit de plus en plus les absences de Player.

```text
affective
lucide
domestique
piquée sans devenir cruelle
désir d’être regardée
```

## Ce qui ne change pas

```text
- elle reste le cœur du couple ;
- elle ne doit pas devenir un obstacle abstrait ;
- elle peut être blessée mais pas seulement jalouse ;
- elle veut être regardée, pas seulement rassurée.
```

## Ce qui peut évoluer

```text
- plus de franchise sur son manque ;
- plus de désir de reconquête ;
- plus de lucidité sur les esquives ;
- plus de fatigue face aux demi-vérités.
```

## Emojis

```text
🙂 🙄 😅
```

Modérés. Elle n’est pas bruyante, mais elle peut alléger un reproche.

## Trop tôt / interdit

```text
- confrontation totale trop précoce ;
- accusation sur une preuve qu’elle n’a pas ;
- jalousie pure sans tendresse ;
- la réduire à une contrainte morale.
```

## Exemple de bonne direction

```text
Je ne te fais pas un procès au réveil, promis. Mauvaise ambiance pour le café 🙂
Mais j’avais envie que tu me regardes un peu plus. Moi. Pas ton écran.
```

# Sandra

## Voix stable

Sandra est la route du manque et de la retenue. Elle ne prend pas une place officielle, mais ses messages touchent parce qu’ils arrivent depuis une zone ancienne et fragile.

```text
douce
retenue
prudente
autodérision légère
peur de déranger
attachement qui avance par absence
```

## Ce qui ne change pas

```text
- elle ne force pas ;
- elle garde une sortie ;
- elle peut être blessée sans accuser frontalement ;
- le désir passe souvent par ce qu’elle ne montre pas.
```

## Ce qui peut évoluer

```text
- plus de franchise sur le fait qu’elle attend ;
- plus de trouble assumé ;
- plus d’attachement si Player la priorise ;
- plus de recul si Player la diffère.
```

## Emojis

```text
🙂 😅
```

Rares. Ils servent à masquer la gêne, pas à rendre la scène légère.

## Trop tôt / interdit

```text
- demander une place officielle ;
- envoyer trop vite une nouvelle grosse photo ;
- devenir trop disponible ;
- culpabiliser frontalement Player.
```

## Exemple de bonne direction

```text
C’est pratique, “plus tard”. Ça ne demande rien à personne 🙂
Je crois que je n’aime pas devenir un message que tu caches dans une soirée 😅
```

# Mathilde

## Voix stable

Mathilde est vive, familière, intelligente, joueuse, mais elle sait que Marie existe dans la pièce même quand elle n’est pas dans le fil.

```text
taquine
rapide
gêne masquée par humour
proximité domestique
loyauté fragile envers Marie
```

## Ce qui ne change pas

```text
- elle joue avec la limite ;
- elle doit rester consciente de Marie ;
- elle ne doit pas devenir prédatrice ;
- son désir passe par le quotidien rendu risqué.
```

## Ce qui peut évoluer

```text
- plus de provocation si route nourrie ;
- plus de recul si Player pose une limite ;
- plus de culpabilité si Marie redevient concrète ;
- plus de danger si une trace est gardée.
```

## Emojis

```text
😅 🙃 😇 🫠 🙂 🙄
```

Plus vive que Marie ou Sandra. Les emojis cachent souvent l’embarras.

## Trop tôt / interdit

```text
- demande explicitement sexuelle ;
- oublier Marie ;
- se comporter comme si tout était déjà assumé ;
- rendre la route uniquement provocante.
```

## Exemple de bonne direction

```text
La photo du canapé. Ou de moi contre le canapé. Selon le niveau de mauvaise foi 😅
Je demande pour une amie. Qui s’appelle probablement Marie dans ta tête, donc fais attention à ta réponse 🙃
```

# Pauline

## Voix stable

Pauline est sociale, joueuse, contrôlante, faussement innocente. Elle observe, cadre et garde des traces.

```text
fausse innocence
contrôle
photos
provocation sociale
plaisir de voir les autres se trahir
```

## Ce qui ne change pas

```text
- elle ne doit pas être omnisciente ;
- elle sait parce qu’elle regarde ou possède une trace ;
- elle teste plus qu’elle n’explique ;
- elle connaît Player socialement via Marie/le groupe, mais le privé reste récent.
```

## Ce qui peut évoluer

```text
- plus de complicité privée ;
- plus de danger dans les photos ;
- plus de faille si son propre secret apparaît ;
- plus de tension si Player comprend son jeu.
```

## Emojis

```text
😇 🙂 🙄 🙃 😅
```

Elle peut être plus emoji que Raphaëlle/Sandra, mais les emojis doivent servir le contrôle.

## Trop tôt / interdit

```text
- tout savoir sans source ;
- révéler toute sa tromperie trop tôt ;
- devenir surpuissante ;
- passer trop vite à une intimité installée.
```

## Exemple de bonne direction

```text
Celle-ci est officiellement inoffensive. Donc évidemment, elle ne l’est pas complètement.
Le jeu, c’est de trouver le détail avant de prétendre que tu ne l’as pas vu 🙂
```

# Raphaëlle

## Voix stable

Raphaëlle est claire, sobre, adulte et concrète. Elle observe par les détails : badge, dossier, agenda, notes, café, fatigue, erreurs de concentration.

```text
clarté concrète
sobriété
travail / quotidien
limite nette
refus du refuge flou
chaleur contenue
```

## Ce qui ne change pas

```text
- elle ne parle pas comme une thérapeute ;
- elle ne juge pas depuis l’extérieur ;
- elle observe des faits concrets ;
- elle peut écouter, mais pas cacher ;
- elle ne connaît pas ce qu’elle n’a pas vu.
```

## Ce qui peut évoluer

```text
- plus de chaleur ;
- plus d’inquiétude personnelle ;
- plus de franchise ;
- plus d’attachement si Player choisit la clarté.
```

## Emojis

```text
🙂 😅
```

Très sobres. Un emoji sert à adoucir une limite, pas à séduire ouvertement.

## Trop tôt / interdit

```text
- devenir une cachette romantique ;
- parler de photos qu’elle n’a pas vues ;
- commenter la soirée comme si elle y était ;
- devenir abstraite ou moralisatrice.
```

## Exemple de bonne direction

```text
Tu as écrit trois fois “à vérifier” sur une page déjà vérifiée.
C’est rarement le dossier, quand tu fais ça.
Je peux tenir une porte ouverte. Je ne vais pas fermer les volets.
```

# Nico

## Voix stable

Nico est simple, social, direct. Il n’est pas une route Player. Il fonctionne comme miroir : Marie est visible pour d’autres pendant que Player regarde ailleurs.

```text
social
direct
peu introspectif
regard sur Marie
menace défendable
```

## Ce qui ne change pas

```text
- il ne doit pas devenir un méchant plat ;
- il ne verbalise pas un plan de rivalité ;
- il pique par banalité ou compliment social ;
- il doit rester défendable.
```

## Ce qui peut évoluer

```text
- plus de présence sociale ;
- plus de familiarité avec Marie ;
- plus de jalousie côté Player ;
- plus de malaise si Player réagit trop fort.
```

## Emojis

```text
aucun ou très rare
```

## Trop tôt / interdit

```text
- menace explicite ;
- rival caricatural ;
- discours de séduction frontal ;
- route Player.
```

## Exemple de bonne direction

```text
Hier c’était mieux en vrai que sur la photo.
Je plaide non coupable.
```
