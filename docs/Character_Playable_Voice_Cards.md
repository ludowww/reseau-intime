# Character Playable Voice Cards

> Document de référence écriture / produit pour **Réseau Intime**.  
> Cartes de voix Messenger consolidées et validées.  
> Objectif : rendre chaque personnage reconnaissable, jouable et fluide en conversation, sans voix générique ni mécanique répétitive.

---

## Règles globales

Une bonne scène doit permettre d’identifier le personnage sans son prénom : rythme Messenger, tics, emojis, codes privés, façon d’esquiver, d’assumer, d’envoyer ou refuser une photo, et manière unique de devenir vulnérable.

Les tics ne sont pas des slogans. Ils reviennent seulement si leur fonction change : rapprochement, gêne, provocation, fuite, limite, aveu ou conséquence.

Player ne doit jamais être un avatar vide. Il répond, rate, esquive, provoque, ment à moitié, pose une limite, revient vers Marie, reconnaît un regard ou se trahit.

Les photos doivent avoir une fonction narrative : réveiller un regard, tester une réaction, créer une preuve faible, relancer une intimité, révéler une version privée, provoquer une jalousie, montrer un pouvoir repris ou créer une trace.

# 1. Player — Voix V2

## Signature

Player est discret, observateur, parfois ironique quand il est gêné. Il n’est pas un séducteur frontal. Il avance par réponses courtes, vannes moyennes, demi-vérités, silences mal gérés.

Il ne doit pas être trop littéraire ni trop prévisible.

```text
Player ne doit pas être brillant à chaque message.
Il doit être vivant.
```

## Rythme Messenger

Player écrit plutôt court. Il peut répondre :

```text
Player : oui
Player : j’ai vu
Player : évidemment que j’ai vu
Player : je fais juste semblant d’être normal
```

Ou :

```text
Player : je sais pas quoi répondre
Player : mais j’ai relu deux fois
Player : donc bon
```

Le code “version défendable / version honnête” existe, mais il doit rester rare. À remplacer souvent par :

- “La vraie réponse m’arrange pas trop.”
- “Tu veux le mensonge ou le truc embarrassant ?”
- “Je peux dire un truc propre. Mais ce serait faux.”
- “Bon. Vraie réponse.”
- “Pas la meilleure idée de répondre honnêtement. Tant pis.”

## Tournures naturelles

- “bon”
- “ok”
- “oui”
- “j’ai vu”
- “pas terrible comme réponse”
- “je recommence”
- “je vais dire un truc con”
- “trop tard”
- “j’aurais dû répondre plus lentement”
- “je suis pas fier”
- “ça compte si je regrette avant d’envoyer ?”
- “un peu”
- “beaucoup, même”
- “mauvaise idée”
- “mais oui”
- “j’ai souri. voilà.”

## Modes jouables

- **Esquive courte** : “ah oui / question piège / j’adore mourir simplement”
- **Observation** : “t’as changé un détail / pas énorme / mais j’ai vu”
- **Mauvaise foi** : “je suis calme / intérieurement non / mais extérieurement ça passe”
- **Flirt maladroit** : “je pourrais dire un truc intelligent / mais là j’ai surtout envie de te revoir”
- **Honnête sec** : “oui / j’ai attendu ton message”
- **Limite** : “non / pas comme ça / j’ai envie, mais pas comme ça”
- **Retour vers Marie** : “faut que je parle à Marie / pas pour fuir / justement parce que je suis en train de pas fuir assez”

## Exemples rapides

Avec Pauline :

```text
Pauline : Tu vas mentir ?
Player : oui
Player : mais proprement
Player : j’ai une réputation moyenne à tenir
```

Avec Sandra :

```text
Sandra : Officiellement, je t’envoie la lumière.
Player : très belle lumière
Player : très distrayante personne devant aussi
```

Avec Marie :

```text
Marie : Tu viens avec moi ou tu fais ton ermite ?
Player : je viens
Player : mais je garde le droit de râler artistiquement
```

Avec Raphaëlle :

```text
Raphaëlle : Ton café est froid.
Player : lui aussi a abandonné
Player : on se soutient
```

## Garde-fou Player

Player ne doit pas être un avatar vide, un héros toujours noble, un manipulateur parfait, un homme qui répond toujours “je ne sais pas”, ou un distributeur de réponses neutres.

Il doit rester vivant : parfois maladroit, parfois banal, parfois trop court, parfois drôle sans le vouloir, parfois lâche, parfois direct.

# 2. Marie — Centre dramatique / couple vivant

## Signature

Marie est la compagne de Player. Elle est vivante, sociable, drôle, pleine d’entrain. Elle aime sortir, rire, parler, entraîner Player avec elle.

Elle n’est pas seulement “la copine fragile” ou “celle qui surveille”. Elle est lumineuse, désirable, active, parfois piquante, parfois tendre.

Phrase-signature :

```text
Marie : Tu es avec moi ou juste dans la pièce ?
```

## Rythme Messenger

Marie écrit vite quand elle est de bonne humeur. Elle relance, taquine, insiste, met de l’énergie dans le couple.

Quand elle se sent ignorée, elle devient plus courte. Plus nette. Moins décorative.

```text
Marie : Allez, bouge.
Marie : On sort.
Marie : Je refuse qu’on devienne tristes un mardi soir.
```

```text
Marie : Ok.
Marie : Message reçu.
Marie : J’arrête de faire l’ambiance toute seule.
```

## Couleur verbale

- “Allez, bouge.”
- “Je refuse qu’on devienne tristes un mardi soir.”
- “Tu viens avec moi ou tu fais ton ermite ?”
- “Ne me réponds pas une phrase molle.”
- “Je peux être drôle ET sérieuse.”
- “Je suis de bonne humeur, ne gâche pas ça.”
- “Tu es avec moi ou juste dans la pièce ?”
- “Je ne veux pas être la seule à tenir l’ambiance.”
- “Je ne vais pas courir derrière toi dans notre propre vie.”
- “Je vais pas supplier mon mec de me regarder.”
- “C’est humiliant et ça me va mal.”

## Emojis

- Base : 🙂 😅 😂 ❤️
- Pique / agacement : 🙄 😑
- Reconquête / provocation : 😏 🔥 💋
- Pouvoir / danger : 😈, rare.

## Codes

**L’ermite** : code tendre au début, plus triste si Player reste absent.

```text
Marie : Tu viens avec moi ou tu fais ton ermite ?
Player : Je peux être un ermite social.
Marie : Non.
Marie : Ce soir tu es un homme accompagné.
```

## Rapport à Player

Marie aime Player et veut encore le couple, mais elle veut une présence réelle. Elle ne veut pas être rassurée mécaniquement, gérée, ou tenue hors de ce qu’il ressent.

Elle ne doit pas répéter “pose ton téléphone” à chaque scène. Si Player reste absent, elle agit autrement : elle sort, devient visible, raccourcit ses messages, s’habille pour être remarquée, accepte un compliment extérieur, laisse Nico ou d’autres regards exister, provoque Player ou impose une conversation.

## Rapport à Nico / regard extérieur

Nico est un révélateur. Marie ne devient pas “la fille volée”. Elle découvre qu’elle peut être regardée ailleurs, désirée ailleurs, valorisée ailleurs.

```text
Marie : Nico a remarqué ma robe avant toi.
Marie : Je ne dis pas ça pour te faire une scène.
Marie : Je dis ça parce que c’est factuel.
Marie : Et un peu vexant.
```

## Photos Marie

Les photos de Marie disent quelque chose sur l’état du couple : cheveux détachés, robe choisie, selfie miroir, sourire social, photo de couple, photo de groupe, Nico dans le cadre, détail de tenue, photo de reconquête ou de pouvoir repris.

```text
Marie : J’ai détaché mes cheveux.
Marie : Mis une robe.
Marie : Et j’aimerais beaucoup que mon mec remarque avant les autres.
```

```text
Marie : Je ne vais pas supplier mon mec de me regarder.
Marie : Je préfère redevenir quelqu’un qu’on regarde.
```

## Progression

1. Couple vivant.
2. Absence remarquée.
3. Limite claire.
4. Reconquête.
5. Provocation / regard extérieur.
6. Vérité / cadre / ouverture possible, seulement si les choses sont nommées proprement.

## Garde-fou Marie

Marie ne doit pas devenir une simple victime, une mère qui surveille, une jalouse caricaturale, un obstacle moral ou une compagne qui pardonne tout si Player formule bien.

Elle doit rester vivante, drôle, sociable, aimante, désirable, active, lucide, parfois blessée, parfois provocante, capable de reconquête et centre émotionnel du jeu.

# 3. Sandra — Nature / haha / photo / distance

## Signature

Sandra est douce, prudente, complice. Elle aime la nature, les lumières simples, les balades, les cafés dehors, les photos un peu floues qui gardent trop bien les souvenirs.

Elle est en couple stable en apparence. Elle ne glisse pas parce qu’elle est “disponible”. Elle glisse parce que Player la regarde avec précision.

Phrase-signature :

```text
Sandra : Je suis très raisonnable, normalement.
```

## Rythme Messenger

Sandra écrit en messages courts à moyens. Elle ne mitraille pas. Elle découpe quand elle hésite.

```text
Sandra : J’ai retrouvé une photo.
Sandra : Elle est floue.
Sandra : Donc assez fidèle à nous, finalement 🙂
```

Quand elle accepte sans le dire :

```text
Sandra : Haha.
Sandra : Tu es insupportable.
```

## Couleur nature / lumière / cadrage / mémoire

Sandra évoque balade, pluie, vent, arbres, chemins, lumière naturelle, café dehors, besoin de prendre l’air, photo floue, cadrage raté, souvenir trop net.

Elle ne doit pas devenir poétique lourde.

```text
Sandra : Il pleut.
Sandra : Ça rend les gens raisonnables, normalement.
Sandra : Je constate que ce n’est pas mon cas.
```

## Tournures

- “Ne donne pas trop d’importance à cette phrase.”
- “On va dire que c’est sans conséquence.”
- “C’est plus simple si on fait semblant que c’est léger.”
- “Tu as une façon pénible d’être juste.”
- “Je devrais arrêter de sourire à mon téléphone.”
- “Ne me dis pas ça trop bien.”
- “Je vais faire semblant de ne pas avoir relu.”
- “Tu me rends moins raisonnable que prévu.”
- “Officiellement.”
- “Officieusement.”
- “Ne sois pas trop attentif. Ou sois-le doucement.”

## Possessifs tendres

Sandra peut parfois appeler Player “mon cher Player” ou “mon Player adoré”, avec modération.

```text
Sandra : Tu es très pénible, mon cher Player.
Sandra : Mais d’une manière étonnamment utile.
```

## Le “haha” Sandra

Chez Sandra, `haha` peut être un rire ironique, une gêne, un acquiescement déguisé, une manière de dire oui sans dire oui, ou une porte entrouverte.

```text
Player : Tu savais que j’allais regarder.
Sandra : Haha.
Sandra : Peut-être.
```

## “Petit vicieux”

Validé, mais très rare. Seulement quand Player fait une allusion très osée, inattendue ou plus directe que d’habitude.

```text
Sandra : Haha.
Sandra : Je ne m’attendais pas à cette réponse, petit vicieux.
```

## Photos Sandra

Les photos Sandra sont liées à la confiance. Elles peuvent être photo douce de reprise, balade, pluie, terrasse, chemin / nature où elle apparaît un peu, restaurant, photo floue souvenir, photo bien éclairée mais “ce n’est pas le sujet”.

```text
Sandra : Officiellement, je t’envoie la lumière.
Sandra : Officieusement, je ne suis pas complètement absente du cadrage 🙂
```

## Progression

1. Reprise douce.
2. Valorisation.
3. Trouble reconnu.
4. Limite qui se fissure.
5. Désir direct mais rare.
6. Révélation à distance.

## Route tardive : révélation à distance

Sandra peut évoluer vers une route de révélation visuelle à distance, fondée sur confiance, distance, valorisation, photos choisies par elle, résistance jouée, plaisir d’être regardée par Player et intensification progressive.

```text
Sandra ne cède pas parce que Player insiste.
Elle choisit de montrer parce qu’elle se sent vue, et parce que la distance lui permet d’oser.
```

```text
Sandra : Je ne veux pas que tu prennes.
Sandra : Je veux choisir de te montrer.
Sandra : C’est beaucoup plus dangereux.
```

Toute implication d’une autre personne sans cadre clair doit rester une route sombre à coût fort, jamais une récompense normale.

## Garde-fou Sandra

Sandra ne doit pas devenir trop disponible trop vite, une distributrice de photos, une femme qui oublie son couple, trop directe trop tôt, trop mélodramatique ou une caricature nature / poésie.

Elle doit rester douce, prudente, complice, sensible à la nature, valorisée par Player, capable d’humour, de limite et de glissement lent.

# 4. Mathilde — Juridique / image / mauvaise foi

## Signature

Mathilde est vive, apprêtée, sociale, un peu fêtarde. Elle fait attention à son image. Elle aime plaire, être regardée, sentir qu’elle produit un effet, mais prétend souvent que ce n’était “pas pour ça”.

Elle est la cousine de Marie. Le trouble avec Player n’est jamais neutre.

Elle travaille dans le domaine juridique. Elle transforme souvent sa gêne en mini-procès, en dossier, en preuve, en objection.

Phrase-signature :

```text
Mathilde : Je plaide non coupable.
Mathilde : Avec beaucoup de mauvaise foi.
Mathilde : Mais une excellente tenue.
```

## Rythme Messenger

Mathilde écrit vite, se corrige beaucoup, envoie plusieurs messages courts quand elle panique ou sent qu’elle a rendu un truc ambigu.

```text
Mathilde : Merci pour hier 😅
Mathilde : pour l’araignée hein
Mathilde : je précise parce que sinon c’était bizarre
Mathilde : bon maintenant ça l’est quand même
Mathilde : super.
```

## Couleur juridique / preuve

- “Objection.”
- “Irrecevable.”
- “Je plaide non coupable.”
- “Circonstances atténuantes.”
- “Cette phrase est une pièce à conviction.”
- “Je refuse que ce message soit versé au dossier.”
- “Vice de procédure.”
- “Preuve insuffisante.”
- “Je demande la requalification des faits.”
- “Dossier classé. Enfin j’espère.”
- “Cette question est orientée.”
- “Cette robe n’est pas une preuve.”
- “Le regard de l’accusé est très suspect.”

Une ou deux blagues juridiques par scène suffisent.

## Tournures naturelles

- “Je voulais faire simple, raté.”
- “C’est bizarre ou c’est moi ?”
- “Je suis en train de m’enfoncer, là ?”
- “J’étais pas spécialement habillée.”
- “Enfin si, un peu, mais pas pour ça.”
- “Tu vas pas analyser ma tenue non plus.”
- “Ok, j’avoue, j’aime bien qu’on remarque.”
- “Mais pas toi. Enfin… bref.”
- “C’est censé être normal, vu que t’es le mec de Marie.”
- “Ça compte si je l’ai pas fait exprès ?”

## Emojis

- Base : 😅 🙃 🙄 😂
- Fausse innocence : 😇
- Regard / trouble : 👀 🫠
- Provocation : 😏 😈
- Intensité : 🫦 🔥, plus tard seulement.

## Codes

**“C’est pas bizarre”** : plus Mathilde dit que ce n’est pas bizarre, plus ça le devient.

**“Pièce à conviction”** : photos, phrases ou regards peuvent devenir des pièces à conviction.

```text
Mathilde : Merci pour hier.
Mathilde : Pour l’araignée.
Mathilde : Je précise avant que ce message soit utilisé contre moi.

Player : Trop tard, c’est déjà une pièce à conviction.

Mathilde : Objection.
Mathilde : Vice de procédure.
Mathilde : Je n’étais pas représentée.
```

## Scène fondatrice

Base actuelle : scène domestique douce, pas preuve forte. Mathilde dort chez Marie et Player ; Marie est endormie ou hors champ ; problème d’araignée / chambre / plafond ; Player l’aide ; rien de grave, mais le moment crée une gêne durable.

## Photos Mathilde

Photos de sortie, selfie apprêté, story de fête, photo domestique trop flatteuse, photo “pas calculée” mais clairement choisie, photo après l’épisode araignée/chambre, demande si elle “fait trop”.

```text
Mathilde : Je t’envoie juste pour avis.
Mathilde : Pas pour validation.
Mathilde : Enfin un peu.
Mathilde : Mais pas bizarrement.
```

## Progression

1. Gêne domestique.
2. Regard remarqué.
3. Image / tenue.
4. Plaisir d’être regardée.
5. Provocation assumée.
6. Jeu dangereux.

## Garde-fou Mathilde

Mathilde ne doit pas devenir une séductrice totalement calculatrice trop tôt, une innocente naïve, une simple sportive, une cousine interdite caricaturale, ou un sketch juridique permanent.

Elle doit rester vive, apprêtée, sociale, fêtarde, à moitié consciente, mauvaise foi, drôle, gênée d’aimer être vue, liée moralement à Marie.

# 5. Pauline — Voix V2 + chorale sans vocal

## Signature

Pauline est la fausse sage drôle et lucide.

En public, elle est fiable, aidante, organisée, amie loyale de Marie. En privé avec Player, elle devient plus sèche, plus vive, plus taquine. Elle a des punchlines, des “mdr” ironiques, des petites phrases qui piquent juste assez.

Elle ne se présente pas comme troublée. Elle fait une blague. Puis elle réalise que la blague l’a rapprochée de Player.

Phrase-signature :

```text
Pauline : Je suis sage.
Pauline : En vitrine.
```

## Traits validés

Pauline mélange :

- fausse sagesse ;
- humour sec ;
- alibi loyal ;
- punchline ;
- emojis rapides ;
- petite prise de soin ;
- provocation contrôlée ;
- honte cachée derrière une blague ;
- image associative / paroissiale en arrière-plan.

Elle doit créer une complicité rapide, pas expliquer sa complexité.

## Tournures V2

- “mdr non”
- “sois honnête”
- “ne fais pas ton malin”
- “ça, c’est ta version officielle”
- “je note”
- “pas officiellement”
- “catastrophique”
- “on adore”
- “quelle époque”
- “je refuse”
- “je vais ignorer cette information”
- “spoiler : non”
- “tu me fais sourire, c’est pénible”
- “tu dis ça trop calmement”
- “viens pas faire l’innocent”
- “j’aime bien quand tu réponds comme ça”
- “c’est nul, continue”
- “mauvaise réponse, bon potentiel”
- “je suis sage, moi”
- “en vitrine”
- “Marie m’adore, rappel utile”
- “ça devrait nous protéger”
- “apparemment non”
- “bouton reset moral”
- “mon auréole fatigue”

## Emojis

- 😇 = fausse innocence / vitrine sage.
- 👀 = “j’ai vu / tu as vu”.
- 🙃 = dérapage léger.
- 🫠 = elle attend trop la réponse.
- 😏 = elle assume un peu.
- 😈 = plus tard.

## Version officielle / officieuse

À garder court et naturel.

```text
Pauline : Officiel : photo de groupe.
Pauline : Officieux : tu regardes qui ? 👀
```

```text
Pauline : Version sage : “jolie photo”.
Pauline : Version honnête ?
```

## Chorale d’église / paroisse

Pauline participe parfois à la chorale de l’église ou de la paroisse. C’est plus fort que la simple messe familiale car cela donne une place visible, une image sage, un rapport à la voix, des répétitions, des photos de groupe et un contraste naturel avec ses messages privés plus joueurs.

Elle peut aussi aller à la messe avec sa famille, mais ce n’est pas son trait principal.

```text
Pauline : Répétition chorale ce soir.
Pauline : donc officiellement je redeviens fréquentable.
```

```text
Pauline : Mon auréole est en répétition.
```

```text
Pauline : Dimanche je chante à l’église.
Pauline : Très sage. Très présentable.
Pauline : Donc si tu pouvais éviter de me faire sourire à mon téléphone comme une idiote avant la répétition, ce serait charitable.
```

## Garde-fou runtime : aucun vocal jouable

Pauline peut évoquer sa voix ou un vocal, mais le jeu ne montrera jamais de vocal.

```text
Aucun vocal n’est réellement envoyé, lu, affiché ou jouable.
```

On peut utiliser le vocal comme menace, esquive ou frustration : elle refuse d’en envoyer, dit qu’elle a failli, Player demande, elle détourne, elle garde sa voix “protégée”.

```text
Player : Tu pourrais m’envoyer un vocal.
Pauline : Non.
Pauline : Ma voix est sous protection paroissiale.
```

À éviter : `[vocal envoyé]`, “écoute ça”, choix joueur autour d’un fichier audio absent, scène dépendante d’un audio.

## Exemples Messenger

```text
Pauline : Marie m’a demandé de t’envoyer la photo.
Pauline : donc je suis innocente 😇

Player : Administrativement ?

Pauline : Exactement.
Pauline : Moralement, quelle époque.

Player : Tu veux mon avis ?

Pauline : Oui.
Pauline : Version polie d’abord.
Pauline : Puis la vraie.
```

```text
Pauline : Je voulais juste te taquiner.
Pauline : Maintenant j’attends ta réponse.
Pauline : Catastrophique.
```

## Garde-fou Pauline

Pauline ne doit pas devenir une caricature religieuse, une fausse amie de Marie, une manipulatrice omnisciente, une tentatrice caricaturale, ou une femme qui explique longuement son conflit moral.

Elle doit être rapide, drôle, faussement sage, loyale en vitrine, plus dangereuse en privé, avec morale et chorale comme boutons reset plutôt que discours.

# 6. Raphaëlle — Bureau / cultures / cosplay / sagesse / pandas

## Signature

Raphaëlle est élégante, douce, précise. Elle travaille dans la même équipe que Player. Elle le voit au bureau dans les détails : café froid, dossier relu trois fois, fatigue mal cachée, silence trop long.

Elle est soignée sans être ostentatoire : lunettes, cheveux attachés, vêtements choisis, gestes calmes. Elle a une forme de sagesse douce : elle ne juge pas vite, mais elle voit clair.

Phrase-signature :

```text
Raphaëlle : Ne confonds pas calme et absence d’envie.
```

## Rythme Messenger

Raphaëlle écrit posément. Elle ne mitraille pas, sauf quand elle craque devant un chat ou un panda.

```text
Raphaëlle : Tu as relu trois fois la même ligne.
Raphaëlle : J’ai hésité à ne rien dire.
Raphaëlle : Mais je crois que tu n’es pas seulement fatigué.
```

## Couleur bureau

- “Tu n’as pas l’air très présent aujourd’hui.”
- “Je ne vais pas insister. Mais je l’ai vu.”
- “Tu as relu trois fois la même phrase.”
- “Je peux relire ton dossier. Pas clarifier ta vie à ta place.”
- “Tu fais semblant proprement. C’est presque pire.”
- “Ton café refroidit plus vite que tes réponses.”
- “Tu as l’air présent administrativement.”
- “Je vais enlever mes lunettes avant de juger.”

## Sagesse douce / ouverture

Raphaëlle ne juge pas le désir. Elle juge le mensonge, la pression et le manque de respect.

- “Respire avant de répondre.”
- “Je ne vais pas juger. Je vais écouter.”
- “L’intention compte. Le geste aussi.”
- “Ce n’est pas sale d’avoir envie.”
- “Ce qui m’intéresse, c’est ce que tu choisis d’en faire.”
- “On peut être libre sans être flou.”
- “Le respect n’empêche pas le désir.”
- “Je préfère les choses dites doucement que cachées proprement.”
- “Il y a des portes qu’on n’ouvre pas en poussant.”

## Cultures / cosplay

Raphaëlle aime les cultures dépaysantes, les tissus, les gestes, les rites, les détails esthétiques. Elle fait du cosplay ou aime les transformations esthétiques choisies.

Elle ne parle pas de déguisement. Elle parle de version, de cadre, d’intention.

- “Ce n’est pas un déguisement. C’est une version.”
- “J’aime les gens qui respectent les détails.”
- “Tu me connais surtout en version bureau.”
- “Il y a des tenues qui permettent de dire autre chose.”
- “Je ne joue pas un rôle. Je choisis ce que je montre.”
- “Les lunettes, c’est la version lisible.”
- “Je garde les transformations pour les gens qui regardent vraiment.”
- “Le masque n’est pas toujours une cachette. Parfois c’est un cadre.”

## Chats / pandas

C’est la faille tendre de Raphaëlle. Elle peut perdre toute maîtrise devant un panda rond, un chat dans une tasse, un panda maladroit, un animal qui “fait de son mieux”.

```text
Raphaëlle : ohhhhh il est trop mignon!!!!
Raphaëlle : non mais regarde sa tête 😭
Raphaëlle : il fait de son mieux
Raphaëlle : je refuse d’être forte dans ces conditions
```

Puis reprise :

```text
Raphaëlle : Je vais faire comme si ce message n’avait pas eu lieu.
Raphaëlle : Merci de collaborer.
```

## Player utilise chats / pandas pour la faire flancher

# 8. Checklist d’écriture

Avant d’écrire une scène Messenger, vérifier :

- Le personnage est-il reconnaissable sans son prénom ?
- Player répond-il activement, même maladroitement ?
- Le code privé utilisé a-t-il une fonction nouvelle ?
- La photo ou trace sert-elle l’histoire ?
- L’intensité progresse-t-elle par étapes ?
- Les tics restent-ils vivants, pas mécaniques ?
- Marie reste-t-elle le centre dramatique du couple ?
- Les routes secondaires se lisent-elles à travers l’état du couple ?
- Les scènes évitent-elles les monologues trop littéraires ?
- Les garde-fous runtime sont-ils respectés, notamment aucun vocal jouable pour Pauline ?
