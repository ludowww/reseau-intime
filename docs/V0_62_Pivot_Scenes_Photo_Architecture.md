# V0.62 — Pivot Scenes & Photo Architecture

## 1. Intention

V0.62 corrige la trajectoire J3+ pour éviter une progression trop plate.

À partir de J3, chaque journée doit contenir :

* au moins une scène pivot mémorable ;
* au minimum 3 photos ou contenus visuels exploitables ;
* une décision significative de Player ;
* une conséquence visible ou différée.

Les scènes ne doivent pas seulement colorer une route : elles doivent faire avancer l’histoire.

## 2. Pas de vocal

Le runtime narratif actuel ne doit pas prévoir de vocal.

Interdit :

* message vocal ;
* scène basée sur un audio ;
* progression narrative dépendant d’un vocal.

Autorisé :

* SMS ;
* messages Messenger ;
* photos ;
* captures ;
* notifications ;
* appels mentionnés narrativement si nécessaire, mais non joués comme audio ;
* appels manqués représentés par message ou notification si le système le permet.

Conséquence :
Les anciennes idées de “vocal Sandra” doivent être remplacées par :

* message supprimé ;
* message hésitant ;
* appel manqué ;
* échange SMS tardif ;
* photo envoyée puis commentée ;
* phrase que Sandra regrette presque d’avoir écrite.

## 3. Règle des 3 photos minimum par jour

Chaque jour doit prévoir au minimum 3 photos ou contenus visuels.

Les messages, captures, notifications ou appels manqués peuvent accompagner la scène, mais ne remplacent pas ces 3 photos.

Chaque photo doit avoir une fonction narrative.

Types de photos :

1. Photo affective ou quotidienne.
2. Photo de trouble, désir ou ambiguïté.
3. Photo de conséquence, dette ou preuve faible.

Une photo ne doit pas être un simple décor.
Elle doit pouvoir produire désir, gêne, mémoire, jalousie, soupçon, dette ou conséquence.

## 4. Catégories de photos

### Photo consentie

La personne sait qu’elle est prise ou envoyée.

Usage :

* couple ;
* sortie ;
* souvenir ;
* jeu social ;
* regard assumé.

Risque :
faible à moyen.

### Photo ambiguë

La personne sait peut-être, laisse peut-être faire, ou la photo change de sens hors contexte.

Usage :

* trouble ;
* vu / pas vu ;
* interprétation ;
* dette.

Risque :
moyen à fort.

### Photo non consentie / volée / diffusée

La personne ne consent pas clairement à la prise, au partage ou à la diffusion.

Usage :

* route sombre ;
* faute ;
* chantage ;
* perte de contrôle ;
* rupture possible.

Règle :
Une photo non consentie ne doit jamais être traitée comme une simple récompense. Elle appartient aux routes sombres et doit pouvoir créer des conséquences fortes.

## 5. Scènes pivot

Une scène pivot doit contenir :

1. Une action concrète de Player.
2. Une tension relationnelle.
3. Au moins 3 photos ou contenus visuels.
4. Une conséquence.

Exemples d’action :

* Player se déplace ;
* Player ment ;
* Player cache ;
* Player montre une photo ;
* Player supprime ou garde une photo ;
* Player répond ou ignore ;
* Player rejoint quelqu’un ;
* Player accepte ou refuse un pacte.

## 6. Sortie Marie / Pauline / Mathilde — sauna, piscine ou plage

Ajouter une deuxième sortie entre Marie, Pauline et Mathilde.

Contexte :
Après une première sortie Marie / Pauline plus sociale, une deuxième sortie est prévue avec Mathilde, dans un cadre plus visuel :

* sauna ;
* piscine ;
* plage ;
* spa ;
* hammam ;
* journée détente.

Fonction narrative :

* produire des photos plus sexy mais consenties ;
* montrer Marie dans une liberté sociale ;
* rapprocher Pauline et Mathilde ;
* créer une jalousie ou excitation chez Player ;
* donner à Pauline un prétexte crédible pour écrire à Player ;
* faire comprendre que Marie et Mathilde peuvent exister hors du regard direct de Player.

Photos possibles :

1. Photo de groupe Marie / Pauline / Mathilde.
2. Photo Marie plus flatteuse ou sexy soft.
3. Photo Pauline + Marie.
4. Photo Mathilde dans le contexte détente.
5. Photo de lieu : piscine, plage, serviettes, transats, vestiaire non explicite.
6. Photo envoyée par Pauline à Player.

Choix Player :

* complimenter Marie seulement ;
* commenter le groupe ;
* demander si Mathilde est avec elles ;
* répondre à Pauline ;
* rester jaloux ;
* plaisanter sur le fait qu’elles ont l’air de trop s’amuser ;
* demander une autre photo, ce qui ouvre un axe plus trouble.

Conséquences :

* Marie peut aimer se sentir regardée ;
* Pauline comprend que Player regarde ;
* Mathilde devient plus présente dans l’imaginaire de Player ;
* Nico peut plus tard demander à voir les photos ;
* une photo de sortie peut revenir comme dette ou preuve faible.

Placement recommandé :

* J4 : première sortie Marie / Pauline, sociale et pas trop sexy.
* J7 : deuxième sortie Marie / Pauline / Mathilde, piscine / plage / sauna, photos sexy soft consenties.

## 7. Progression J3→J12 corrigée

### J3 — Mathilde appelle Player la nuit

Pivot :
Mathilde appelle ou écrit à Player la nuit pour une araignée ou un problème domestique.

Fonction :
première intimité domestique cachée.

Photos minimum :

1. photo de l’araignée ou du coin de chambre ;
2. photo du couloir / porte / nuit ;
3. photo du lendemain ou détail domestique qui rappelle la scène.

Conséquence :
Marie dort pendant que Player devient le secours de Mathilde.

### J4 — Marie sort avec Pauline

Pivot :
Marie sort avec Pauline.

Fonction :
Marie devient visible hors du couple.

Photos minimum :

1. Marie avant la sortie ;
2. selfie Marie / Pauline ;
3. photo du lieu.

Conséquence :
Pauline entre doucement dans le réseau. Player ressent fierté, jalousie ou curiosité.

### J5 — Nico remarque Marie

Pivot :
Nico complimente Marie et teste Player.

Fonction :
Nico devient miroir du désir.

Photos minimum :

1. photo Marie montrable ;
2. photo ou capture liée à Nico ;
3. photo contexte : bar, message, sortie ou écran.

Conséquence :
Player choisit de protéger, rire ou ouvrir une porte.

### J6 — Jeu Mathilde pendant absence de Marie

Pivot :
Marie est dehors. Mathilde propose un jeu à Player.

Fonction :
rapprochement domestique actif.

Photos minimum :

1. jeu / verres / cartes ;
2. photo Mathilde soft ou drôle ;
3. photo ou message de Marie pendant la scène.

Conséquence :
Player et Mathilde partagent une complicité que Marie ne voit pas.

### J7 — Piscine / plage / sauna Marie + Pauline + Mathilde

Pivot :
Deuxième sortie entre Marie, Pauline et Mathilde.

Fonction :
montée visuelle et sociale.

Photos minimum :

1. photo groupe ;
2. photo Marie sexy soft ;
3. photo Pauline / Mathilde / lieu.

Conséquence :
Pauline a un vrai prétexte pour écrire à Player ; les photos peuvent devenir dette ou tentation.

### J8 — Sandra devient concrète

Pivot :
Sandra propose un restaurant ou une rencontre concrète.

Fonction :
la route Sandra passe du SMS au réel.

Photos minimum :

1. photo du restaurant / lieu ;
2. photo Sandra soft ;
3. photo trace de retour : table, ticket, message, extérieur.

Conséquence :
Player doit choisir transparence ou secret vis-à-vis de Marie.

### J9 — Sandra en vulnérabilité

Pivot :
Sandra se dispute avec son copain. Player peut la réconforter par messages ou aller chez elle.

Fonction :
pivot émotionnel fort.

Photos minimum :

1. photo extérieure ou lieu chez Sandra ;
2. photo canapé / tasse / ambiance ;
3. photo Sandra vulnérable soft ou trace de la soirée.

Conséquence :
Sandra devient prioritaire dans un moment que Marie ignore.

### J10 — Urgence Sandra + photo compromettante

Pivot :
Sandra demande un service urgent alors que Player est avec Nico ou Pauline. Player ment pour partir. Il est suivi ou pris en photo dans une situation embarrassante.

Fonction :
mensonge + preuve ambiguë.

Photos minimum :

1. photo ou capture liée au prétexte de départ ;
2. photo chez Sandra ;
3. photo prise en cachette par Nico ou Pauline.

Conséquence :
Nico ou Pauline possède une preuve ambiguë.

### J11 — Le détenteur de preuve agit

Pivot :
Nico utilise la preuve pour pousser Player, ou Pauline l’utilise pour se rapprocher / enquêter.

Fonction :
première conséquence informationnelle forte.

Photos minimum :

1. photo-preuve qui revient ;
2. réaction ou capture ;
3. nouvelle photo demandée, refusée ou gardée.

Conséquence :
Player perd une partie du contrôle.

### J12 — Première mise au point Marie

Pivot :
Marie pose une vraie question, ou une contradiction devient visible.

Fonction :
pré-lock émotionnel.

Photos minimum :

1. photo Marie / couple / maison ;
2. photo qui revient comme conséquence ;
3. photo ou trace liée à la route dominante.

Conséquence :
les routes commencent à se différencier plus fortement après J12.

## 8. Scènes en réserve

À garder pour variantes ou J10+ :

* Marie demande à Player de la prendre en photo ;
* Mathilde enfermée dehors ;
* Mathilde malade ou angoissée ;
* Mathilde demande à Player de mentir pour elle ;
* Marie rentre plus tard que prévu ;
* Marie parle d’un homme qui l’a complimentée ;
* Pauline demande un avis à Player pour Marie ;
* Pauline découvre un secret de Player ;
* Nico propose de couvrir Player contre photo ;
* Nico invite Marie secrètement ;
* sorties parallèles Player/Nico et Marie/Mathilde ;
* Raphaëlle remarque que Player ment mal ;
* Raphaëlle couvre une absence ;
* séminaire Raphaëlle.

## 9. Garde-fous

* Une scène pivot forte par jour suffit.
* Les autres scènes doivent être conséquences, échos ou respirations.
* Les photos doivent avoir origine, statut, détenteur et risque.
* Marie reste la grille de lecture principale.
* Mathilde doit avoir de l’agence, pas seulement être observée.
* Nico doit évoluer progressivement.
* Pauline doit entrer via Marie et les photos sociales.
* Sandra reste le danger de l’intimité émotionnelle.
* Raphaëlle reste différée et extérieure.
* Aucun lock fort avant J10/J12.

## 10. Conclusion

V0.62 corrige l’approche J3+.

À partir de J3, l’histoire doit avancer par scènes pivot, photos, secrets, conséquences et retours émotionnels.

Minimum :

* 3 photos par jour ;
* 1 pivot fort ;
* 1 choix significatif ;
* 1 conséquence.

Les routes restent ouvertes, mais chaque journée doit devenir mémorable.
