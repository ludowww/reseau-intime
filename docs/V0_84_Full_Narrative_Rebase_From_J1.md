5. Nouveau critère d’audit jour par jour

Pour chaque jour existant J1→J6, poser les questions suivantes :

1. Le jour sert-il encore la meilleure version v1.0 ? 2. La structure est-elle à garder ? 3. Les personnages sont-ils encore justes ? 4. Les voix sont-elles obsolètes ? 5. Le jour est-il trop legacy ? 6. Peut-on garder seulement une scène ou une idée ? 7. Faut-il déplacer le beat plus tard ? 8. Faut-il supprimer ou remplacer ? 9. Quel impact sur J7→J12 ? 

Catégories de verdict :

Conserver : structure et voix globalement bonnes.

Conserver avec polish : structure bonne, voix à moderniser.

Recycler partiellement : idée utile, jour à reconstruire.

Déplacer : beat intéressant mais mauvais timing.

Remplacer : meilleure scène à écrire.

Supprimer : legacy nuisible ou redondant.

6. Audit J1

6.1 Fonction attendue J1 v1.0

J1 doit être la vraie promesse du jeu.

Il doit installer :

un couple Marie / Player vivant ;

un Player incarné ;

une première vibration Sandra ;

Mathilde en indirect seulement si utile ;

la logique smartphone / messagerie privée ;

une envie de continuer ;

une tension intime crédible, pas encore lourde.

J1 ne doit pas seulement “présenter des personnages”.

Il doit faire sentir :

Je suis dans une vie intime. Les messages ont du poids. La banalité peut devenir dangereuse. Marie compte vraiment. Sandra revient trop doucement pour être neutre. Player n’est pas vide. 

6.2 Marie J1

Marie doit apparaître dès J1 comme :

compagne vivante ;

drôle ;

sociale ;

désirable ;

tendre ;

parfois piquante ;

pas fragile par défaut ;

pas gardienne morale ;

pas simple obstacle au désir.

Objectif :

Marie doit donner envie à Player de vivre avec elle, pas seulement peur de la perdre. 

Si le J1 actuel ne donne pas assez cette énergie, il doit être réécrit.

6.3 Player J1

Player doit être :

discret ;

observateur ;

parfois ironique quand il est gêné ;

maladroit ;

actif ;

capable de répondre court ;

capable de se trahir.

Interdit :

Player neutre ;

Player absent ;

Player trop brillant ;

Player simple support de choix ;

Player toujours noble.

6.4 Sandra J1

Sandra peut être un excellent moteur J1 si elle arrive par :

ancienne complicité ;

photo ;

lumière ;

souvenir ;

distance ;

haha défensif ;

tendresse prudente.

Elle ne doit pas être trop disponible.

Son intérêt vient du fait qu’elle se sent regardée précisément, pas du fait qu’elle serait immédiatement prête à glisser.

6.5 Verdict J1

Verdict recommandé :

J1 doit être réévalué fortement. Il peut être conservé seulement si Marie est assez vivante, Player assez incarné, et Sandra assez spécifique. Sinon, créer un nouveau J1 source pack v1.0. 

Priorité : très haute.
