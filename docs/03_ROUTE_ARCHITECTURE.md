Règles actuelles :

- les routes sont invisibles pour le joueur ;
- elles décrivent une transformation durable, pas un score ou un propriétaire ;
- elles se construisent par accumulation d’actes, de limites, de promesses, de silences et de conséquences ;
- les scènes servent des routes définies en amont ;
- les séquences sont conçues avant leur distribution en journées ;
- les états runtime sont bornés et sourcés.

L’ancien contenu reste consultable dans l’historique Git uniquement.
