V0.64 keeps the structure of V0.61 but applies the requested naturalization pass:

- fewer perfect punchlines;
- more ordinary Messenger replies;
- Player less polished;
- Marie still lively, but less like a machine à réparties;

- low-level emojis added sparingly;
- no route lock;
- no runtime integration.

## 10. Changes from V0.61

### Reduced over-cleverness

Removed or softened several lines that sounded too written, including:

- overly polished Player jokes;
- repeated conceptual punchlines;
- excessive back-and-forth symmetry;
- some lines where Marie sounded like she was already diagnosing the whole couple.

### Added ordinary Messenger texture

Added or preserved:

- `ok`;
- `oui`;
- `désolé`;
- `pas faux`;
- `j'y vais`;
- small confirmations;
- shorter Player replies;
- slight flatness in some branches.

### Added low-level emojis sparingly

Used only low-intensity J1-compatible emojis:

- Marie: `😅`, `🙂`
- Sandra: `🙂`

No high-intensity emoji.

### Preserved J1 structure

No structural change from V0.60:

- Marie opening;
- Marie color beat;
- Sandra trace re-entry;
- Marie presence check;
- Sandra soft boundary;
- Marie end close.
