## Problème identifié

La première intégration J3 était fonctionnelle, mais certains échanges restaient trop plats ou impersonnels :

- débuts de conversation trop abrupts ;
- manque d’accroches naturelles comme `Coucou` ou `Bonjour` ;
- densité d’emojis trop faible par rapport à J1/J2 ;
- voix parfois trop proches entre personnages ;
- rythme de messagerie pas assez humain.

## Rappel de règle

Le guide d’écriture indique que les emojis doivent rester caractérisés, mais cela ne veut pas dire qu’ils doivent presque disparaître.

Pour J3, la bonne règle est :

```text
assez d’emojis pour que la messagerie paraisse vivante,
mais jamais au point d’uniformiser les voix.
```

## Densité cible J3

La cible recommandée pour J3 est :

```text
Marie matin : 8–12 emojis
Mathilde matin : 10–14 emojis
Raphaëlle midi : 4–6 emojis
Sandra soir : 10–14 emojis
Marie nuit : 5–8 emojis
```

Raphaëlle reste volontairement plus sobre. Mathilde et Sandra peuvent être plus expressives, mais pas de la même manière.

## Emojis par voix

### Marie

Emojis privilégiés :

```text
🙂 🙄 😅 😂 ☕
```

Usage : domesticité, couple, ironie douce, fatigue tendre, petite pique intime.

Marie ne doit pas devenir légère gratuitement : ses emojis doivent souvent adoucir une remarque lucide.

### Mathilde

Emojis privilégiés :

```text
😅 😇 🙄 😂 🫠 🙃
```

Usage : insolence, mauvaise foi, gêne masquée, provocation défendable, recul comique.

Mathilde peut utiliser plus d’emojis que Marie, mais ils doivent renforcer son mélange de jeu et de culpabilité légère.

### Raphaëlle

Emojis privilégiés :

```text
🙂 😅
```

Usage : chaleur sobre, mini-humour professionnel, clarté sans flou.

Raphaëlle ne doit pas sonner comme Sandra ou Mathilde. Elle peut sourire, mais elle reste mesurée.

### Sandra

Emojis privilégiés :

```text
🙂 😅 🫠
```

Usage : embarras, douceur, hésitation, autodérision, message retenu.

Sandra utilise les emojis pour se protéger, pas pour provoquer frontalement.

## Accroches naturelles

J3 doit éviter les débuts trop secs.

Exemples préférés :

```text
Coucou de la cuisine ☕
Coucou les propriétaires du canapé 😅
Bonjour 🙂
Coucou 🙂
```

Les accroches doivent varier selon le personnage et le moment de journée.

## Garde-fous

Ne pas ajouter :

```text
❤️ 😍 😘 😏 🥵 🍑 🍆 💦
```

Ne pas transformer chaque message en ligne emoji.

Ne pas compenser un dialogue plat par des emojis : d’abord ajouter détail concret, rythme, sous-texte, puis emoji si utile.

## Principe final

```text
Un bon emoji ne remplace pas une intention.
Il rend visible la façon dont le personnage essaie de contrôler, adoucir ou masquer cette intention.
```
