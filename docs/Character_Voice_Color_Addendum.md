# Character Voice Color Addendum

> Document de cadrage produit / écriture.  
> Addendum à `docs/Character_Voice_Messenger_Intensity_Bible.md`.  
> Référence pour renforcer les couleurs verbales, codes privés et vulnérabilités des personnages.  
> Aucun runtime, aucun JSON narratif, aucune conversation jouable, aucun test ne sont modifiés ici.

## 1. Statut

Ce document capture les décisions validées après la première bible Messenger / intensité.

Il doit être lu comme une couche de personnalité supplémentaire, à intégrer ensuite dans les fiches complètes des personnages.

Règle :

```text
Ces couleurs ne doivent pas devenir des gimmicks répétés à chaque message.
Elles servent à rendre chaque voix identifiable et à fournir des codes privés exploitables en messagerie.
```

## 2. Synthèse des couleurs validées

| Personnage | Couleur verbale / privée | Fonction |
|---|---|---|
| Marie | vie sociale / énergie / scène | Elle refuse que le couple s'éteigne |
| Sandra | nature / lumière / photo / possessifs tendres / haha ambigu | Elle transforme le désir en regard doux, précis et à moitié assumé |
| Mathilde | juridique / mauvaise foi / image / preuve | Elle transforme sa gêne en mini-procès complice |
| Pauline | loyauté publique / associatif / morale / double visage | Elle apparaît fiable, puis révèle une zone cachée et contradictoire |
| Raphaëlle | bureau / élégance / cultures dépaysantes / cosplay / sagesse douce / chats-pandas | Elle offre clarté, transformation maîtrisée et vulnérabilité tendre |
| Nico | jeu / pari / mauvaise idée / vérité confortable | Il nomme ce que Player évite |
| Player | version défendable / version honnête | Il esquive, puis se trahit |

## 3. Sandra — nature, possessifs, rire ambigu et route de révélation à distance

### 3.1 Couleur nature

Sandra aime la nature.

Cela doit colorer ses messages, photos et respirations :

- balade ;
- pluie ;
- arbres ;
- lumière naturelle ;
- café dehors ;
- vent ;
- chemins ;
- besoin de prendre l'air ;
- nature comme tentative de redevenir raisonnable.

Cette couleur doit rester simple, pas poétique lourde.

Exemples :

```text
Sandra : Je suis allée marcher.
Sandra : Très saine décision.
Sandra : Air frais, arbres, tout ça.
```

```text
Sandra : Il pleut.
Sandra : Ça rend les gens raisonnables, normalement.
Sandra : Je constate que ce n'est pas mon cas.
```

### 3.2 Possessifs tendres

Sandra emploie parfois des possessifs affectueux avec Player.

Formes validées :

```text
mon cher Player
mon Player adoré
```

À utiliser avec modération, surtout en complicité, gêne ou ironie tendre.

Exemples :

```text
Sandra : Tu es très pénible, mon cher Player.
Sandra : Mais d'une manière étonnamment utile.
```

```text
Sandra : Attention, mon Player adoré.
Sandra : Là tu réponds beaucoup trop bien pour quelqu'un censé rester raisonnable.
```

### 3.3 Le "haha" Sandra

Sandra utilise souvent `haha` comme un marqueur très personnel.

Ce n'est pas seulement un rire. Cela peut signifier :

- rire ironique ;
- gêne ;
- acquiescement déguisé ;
- oui sans dire oui ;
- manière d'accepter de franchir une limite tout en se protégeant ;
- défense avant une phrase plus intime.

Exemples :

```text
Player : Tu savais que j'allais regarder.
Sandra : Haha.
Sandra : Peut-être.
```

```text
Sandra : Haha.
Sandra : Je ne devrais pas répondre à ça.
Sandra : Mais je vais le faire quand même.
```

```text
Player : Tu as envie que je te dise ce que j'ai vu ?
Sandra : Haha.
Sandra : Tu es insupportable.
```

Règle :

```text
Chez Sandra, un "haha" peut être une porte entrouverte.
```

### 3.4 Usage rare de "petit vicieux"

`Petit vicieux` est validé, mais doit rester très rare.

Ce n'est pas un surnom régulier. Ce n'est pas un tic. Cela marque une bascule.

À utiliser seulement quand Player fait une allusion très osée, inattendue ou beaucoup plus directe que d'habitude.

Forme recommandée : incluse dans une phrase.

Exemples :

```text
Sandra : Je ne m'attendais pas à ça, petit vicieux !
```

```text
Sandra : Haha.
Sandra : Je ne m'attendais pas à cette réponse, petit vicieux.
```

```text
Sandra : Je vois très bien ce que tu veux dire, petit vicieux.
Sandra : C'est justement le problème.
```

À éviter pour les simples compliments ou sous-entendus doux.

### 3.5 Route tardive Sandra — révélation à distance

Sandra peut évoluer plus tard vers une route de révélation visuelle à distance, fondée sur :

- confiance ;
- distance ;
- valorisation ;
- photos choisies par elle ;
- résistance jouée ;
- plaisir d'être regardée par Player ;
- intensification progressive.

Principe :

```text
Sandra ne cède pas parce que Player insiste.
Elle choisit de montrer parce qu'elle se sent vue et parce que la distance lui donne le courage d'oser.
```

Formule de référence :

```text
Sandra : Je ne veux pas que tu prennes.
Sandra : Je veux choisir de te montrer.
Sandra : C'est beaucoup plus dangereux.
```

La route doit distinguer :

- photos choisies par Sandra ;
- secret émotionnel avec Player ;
- toute implication d'une autre personne sans cadre clair, qui doit rester une route sombre à coût fort et non une récompense normale.

Exemple de progression :

```text
Sandra : J'ai failli prendre une photo.
Sandra : Pas une photo "jolie".
Sandra : Une photo que je n'aurais pas su expliquer.
Sandra : Et j'ai pensé à toi en hésitant.
```

## 4. Mathilde — humour juridique et complicité de preuve

### 4.1 Décision

Mathilde travaille dans le domaine juridique.

Cela ne doit pas la rendre froide. Ce détail doit renforcer sa mauvaise foi, sa gêne et sa complicité avec Player.

Elle utilise parfois des tournures juridiques pour plaisanter, esquiver, flirter ou transformer une gêne en jeu.

### 4.2 Fonction

Le vocabulaire juridique sert à :

- masquer la gêne ;
- faire de la mauvaise foi ;
- transformer une phrase ambiguë en dossier ;
- plaisanter sur les preuves, photos et regards ;
- créer un code privé avec Player ;
- augmenter la tension sans devenir trop directe trop vite.

### 4.3 Tournures juridiques

- "Objection."
- "Irrecevable."
- "Je plaide non coupable."
- "Circonstances atténuantes."
- "Je demande à consulter mon avocate."
- "Cette phrase est une pièce à conviction."
- "Je refuse que ce message soit versé au dossier."
- "Je retire ma déclaration."
- "Le tribunal appréciera."
- "Vice de procédure."
- "Preuve insuffisante."
- "Témoin hostile."
- "Je demande la requalification des faits."
- "Dossier classé. Enfin j'espère."
- "Cette question est orientée."
- "Je conteste formellement."
- "C'est un aveu ou une tentative de provocation ?"

### 4.4 Exemples

```text
Mathilde : Merci pour hier.
Mathilde : Pour l'araignée.
Mathilde : Je précise avant que ce message soit utilisé contre moi.

Player : Trop tard, c'est déjà une pièce à conviction.

Mathilde : Objection.
Mathilde : Vice de procédure.
Mathilde : Je n'étais pas représentée.
```

```text
Player : Tu savais que cette photo ferait réagir.

Mathilde : Je plaide circonstances atténuantes.
Mathilde : Il y avait une bonne lumière.
Mathilde : Et peut-être une mauvaise intention.
Mathilde : Mais très légère.
Mathilde : Presque défendable.
```

Règle :

```text
Une ou deux blagues juridiques par scène suffisent.
Ne pas transformer chaque message de Mathilde en sketch juridique.
```

Phrase de référence :

```text
Mathilde : Je plaide non coupable.
Mathilde : Avec beaucoup de mauvaise foi.
Mathilde : Mais une excellente tenue.
```

## 5. Pauline — double visage, loyauté publique et refuge moral

### 5.1 Décision

Pauline doit être perçue d'abord comme 100% loyale, fiable, sociale, presque irréprochable.

Son lien associatif / caritatif / paroissial doit rester une couleur de fond, pas une caricature permanente.

Elle peut être impliquée dans :

- association caritative ;
- collectes ;
- organisation d'événements solidaires ;
- réseau paroissial discret ;
- actions de quartier ;
- communication associative ;
- événements familiaux ou sociaux.

### 5.2 Double visage

En public, Pauline est la femme fiable, aidante, respectée, amie de Marie.

En privé, elle cache :

- fantasmes ;
- secrets ;
- passé ou secret de tromperie ;
- goût du risque ;
- rapport fort aux photos ;
- envie d'être vue autrement ;
- honte après coup ;
- attirance grandissante pour Player.

Elle ne doit pas être cynique. Elle alterne entre honte, provocation, alibi, désir, culpabilité, loyauté et envie de recommencer.

### 5.3 Refuge moral / associatif

Quand Pauline se sent honteuse, elle peut se raccrocher à son image propre.

Exemples :

```text
Pauline : J'ai passé l'après-midi à trier des cartons pour la collecte.
Pauline : Donc normalement je suis redevenue une personne correcte.
Pauline : Statistiquement.
```

```text
Pauline : Je ne vais pas répondre tout de suite.
Pauline : Je vais faire un truc utile.
Pauline : C'est mon mécanisme officiel de survie morale.
```

### 5.4 Quand elle assume

Elle peut jouer avec son image de fille sage, mais rarement.

Exemples :

```text
Pauline : Version officielle : je suis sage.
Pauline : Version officieuse : ta réponse m'a beaucoup trop plu 😇
```

```text
Pauline : Ne me donne pas envie de perdre mon auréole.
Pauline : Elle me sert encore en public.
```

### 5.5 Phrases clés

- "Version officielle : je suis sage."
- "Version officieuse : tu réponds beaucoup trop bien."
- "Je ne suis pas très bien placée pour juger."
- "Marie me fait confiance. C'est très contrariant."
- "Je vais faire un truc utile pour redevenir fréquentable."
- "Mon auréole me sert encore en public."
- "Je contrôle très bien l'image. Moins bien l'envie."
- "Je ne veux pas lui faire de mal. C'est vrai. Et ça n'arrange rien."
- "Je pensais gérer le jeu. Mauvaise estimation."
- "Je vais me réfugier dans l'associatif. C'est plus sûr que toi."
- "Je ne cautionne pas. Je constate. Avec intérêt."
- "C'est moralement discutable. Donc évidemment je réfléchis trop."

Règle :

```text
Ne pas faire de Pauline une caricature religieuse.
Le caritatif / paroissial sert de refuge moral, de contraste et d'image publique fiable.
```

## 6. Raphaëlle — cultures, transformation, sagesse douce et pandas

### 6.1 Décision générale

Raphaëlle combine :

- élégance quotidienne ;
- clarté professionnelle ;
- soin discret ;
- lunettes ;
- cheveux attachés ;
- passion pour les cultures dépaysantes ;
- cosplay / transformation maîtrisée ;
- sagesse douce, respectueuse et ouverte ;
- faiblesse chats / pandas.

### 6.2 Cultures dépaysantes et cosplay

Raphaëlle aime les cultures, les tissus, les gestes, les rites, les détails esthétiques.

Elle fait du cosplay de manière discrète mais sérieuse.

Elle ne parle pas de déguisement. Elle parle de version, d'intention, de rôle choisi.

Tournures :

- "Ce n'est pas un déguisement. C'est une version."
- "J'aime les gens qui respectent les détails."
- "Un costume raté, c'est souvent une intention floue."
- "Je ne voyage pas toujours loin, alors je fabrique parfois le détour."
- "Tu me connais surtout en version bureau."
- "Il y a des tenues qui permettent de dire autre chose."
- "Je ne joue pas un rôle. Je choisis ce que je montre."
- "Les lunettes, c'est la version lisible."
- "Je garde les transformations pour les gens qui regardent vraiment."
- "Le masque n'est pas toujours une cachette. Parfois c'est un cadre."

### 6.3 Sagesse douce / ouverture d'esprit

Raphaëlle peut ramener une forme de sagesse douce, sans caricature spirituelle.

Elle ne juge pas le désir. Elle juge le mensonge, la pression et le manque de respect.

Tournures :

- "Respire avant de répondre."
- "Je ne vais pas juger. Je vais écouter."
- "L'intention compte. Le geste aussi."
- "Ce n'est pas sale d'avoir envie."
- "Ce qui m'intéresse, c'est ce que tu choisis d'en faire."
- "On peut être libre sans être flou."
- "Le respect n'empêche pas le désir."
- "Je préfère les choses dites doucement que cachées proprement."
- "Il y a des portes qu'on n'ouvre pas en poussant."
- "Tu peux avoir une envie. Tu n'es pas obligé de la salir."
- "Ne confonds pas calme et absence de désir."
- "Je vois très bien le trouble. Je ne suis pas obligée de le fuir."

Règle :

```text
Raphaëlle ne rend pas le désir plus faible.
Elle le rend plus conscient.
```

### 6.4 Faiblesse chats / pandas

Raphaëlle craque pour les chats et surtout les pandas.

Cette faiblesse :

- casse sa maîtrise ;
- crée une complicité avec Player ;
- devient un code privé ;
- la rend vulnérable ;
- peut servir de respiration après tension.

Phrase de référence :

```text
Raphaëlle : Ne te sers pas des pandas contre moi.
Raphaëlle : C'est une arme déloyale.
```

Elle peut avoir une rupture totale de registre face à un panda ou un chat trop mignon.

Exemples :

```text
Raphaëlle : ohhhhh il est trop mignon!!!!
Raphaëlle : non mais regarde sa tête 😭
Raphaëlle : il fait de son mieux
```

```text
Raphaëlle : non.
Raphaëlle : non mais non.
Raphaëlle : il est dans une tasse 😭
Raphaëlle : comment veux-tu que je reste sérieuse ?
```

Après cette rupture, elle tente généralement de reprendre le contrôle.

```text
Raphaëlle : Je vais faire comme si ce message n'avait pas eu lieu.
```

### 6.5 Player utilise les chats / pandas pour la faire flancher

À terme, Player peut comprendre que chats/pandas sont une vraie faille de Raphaëlle.

Il peut l'utiliser pour :

- la faire sourire ;
- désamorcer une tension ;
- reprendre contact ;
- obtenir une réponse ;
- la rendre plus vulnérable ;
- la draguer avec douceur.

Cette stratégie doit rester douce et attentionnée.

Bon usage : Player envoie un panda pour la faire sourire, puis dit une phrase vraie.

Mauvais usage : Player envoie un panda pour éviter une limite ou une réponse claire.

Exemple :

```text
Raphaëlle : Tu utilises les pandas pour contourner ma défense.

Player : Je préfère dire que je respecte tes points d'entrée émotionnels.

Raphaëlle : Formulation beaucoup trop propre pour une manœuvre aussi basse.

Player : Et efficace ?

Raphaëlle : Malheureusement.
```

Phrase de référence :

```text
Raphaëlle : Je sais que tu fais ça pour me désarmer.
Player : Oui.
Raphaëlle : Merci d'être honnête. C'est beaucoup plus dangereux que le panda.
```

## 7. Règle finale

Ces couleurs doivent être utilisées comme des points d'ancrage, pas comme des slogans.

Une scène réussie doit pouvoir faire reconnaître le personnage par :

- une tournure ;
- un emoji ;
- un code privé ;
- une façon de fuir ;
- une façon d'assumer ;
- une façon d'envoyer ou refuser une photo ;
- une manière unique de devenir vulnérable.
