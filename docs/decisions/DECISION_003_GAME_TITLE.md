# DECISION 003 — Titre du jeu

## Statut

Validé.

## Décision

Le titre officiel du projet est :

```text
Réseau Intime
```

## Nom du dépôt

```text
reseau-intime
```

## Pourquoi

Le titre évoque directement :

- les liens entre personnages ;
- les messages privés ;
- les secrets ;
- la circulation du désir ;
- les ramifications relationnelles ;
- l’interface smartphone et sociale du jeu.

Il est suffisamment large pour couvrir les routes de couple, tromperie, libertinage, polyamour, NTR, harem et enquête intime, sans réduire le jeu à une seule mécanique.

## Titres écartés

- Entre les lignes
- Vu à 23:47
- Messages supprimés
- Mode silencieux
- Confidences dangereuses

Ces titres restent utiles comme inspirations possibles pour des chapitres, fins, succès, ou sous-titres de routes.

## Conséquence

Tous les documents, futurs écrans de titre, fichiers marketing et références internes doivent utiliser **Réseau Intime** comme nom officiel.
