Ce document est la **source canonique narrative J3**.

Règle de lecture appliquée :

```text
Brut narratif validé > JSON runtime.

Toute divergence future doit être auditée avant correction.
```

---

## 1. Intention J3

J3 est la journée où la maison change vraiment.

Le mouvement n’est pas une montée sexuelle ni une bascule de route : c’est un pivot domestique, intime et discret, centré sur la présence de Mathilde dans l’espace commun.

La tension de J3 vient de quatre choses :
- la proximité ;
- le soin ;
- la gêne légère ;
- le secret doux.

Marie reste la grille affective principale : elle structure le couple, garde la mesure morale, et sert de référence émotionnelle même quand elle est hors champ la nuit.

Sandra reste un écho doux à midi : elle apporte une respiration, un ailleurs feutré, sans concurrence directe avec Marie.

Mathilde devient le vrai pivot de J3, mais par le domestique : traces, déplacements, aide, plafond, chambre, ajustements, petit trouble de présence.

Nico, Pauline et Raphaëlle restent absents de J3 actif.

## 3. Structure canonique J3

Ordre canonique :

1. **Marie matin** — cuisine, chaussettes, biscuits, traces de Mathilde déjà présentes.
2. **Sandra midi** — roman idiot, lac, écho doux, respiration extérieure.
3. **Marie soir** — dormir tôt, rituels du couple, préparation naturelle du motif araignée.
4. **Mathilde nuit** — araignée administrative, plafond, chambre, aide domestique.

Lecture produit :
- J3 n’ouvre pas une nouvelle route ;
- J3 ne verrouille rien ;
- J3 prépare le passage domestique vers le futur, sans le formaliser ;
- J3 doit rester une journée de proximité et de frottement léger, pas de confrontation frontale.

Le premier vrai pivot Mathilde se situe ici, pas plus tôt.

J3 est canonique en tant que **pivot domestique nocturne de Mathilde**.

- matin = traces de présence ;
- midi = respiration douce ;
