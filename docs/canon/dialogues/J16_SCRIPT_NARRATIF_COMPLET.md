# 1. Décision principale de la journée

J16 devient :

```text
Mercredi — Ce qui doit revenir avant la vérité
```

Sa fonction est :

```text
payer ou constater explicitement l’échec
de la conséquence la plus urgente encore active
avant l’entrée dans les résolutions
```

J16 ne demande pas :

```text
quelle relation Player souhaite poursuivre
```

Elle demande :

```text
quelle règle, suppression, sécurité,
responsabilité ou vérité minimale
doit être rétablie avant que les personnages
puissent décider ce qu’ils veulent encore conserver
```

Une relation ne peut pas entrer en résolution tant qu’une personne attend encore :

- d’être mise en sécurité ;
- de savoir où existe son image ;
- d’obtenir une suppression promise ;
- de connaître l’heure réelle ;
- de recevoir un travail déjà dû ;
- de récupérer son espace ;
- de ne plus servir d’alibi ;
- de voir un mensonge reconnu.

# 2. Question dramatique

> Que faut-il rendre, retirer, confirmer ou arrêter avant que les personnages puissent enfin parler de ce qu’ils veulent ?

Ce qui doit « revenir » peut être :

- une clé ;
- un objet ;
- une distance ;
- un fichier professionnel ;
- le contrôle d’une image ;
- une heure réelle ;
- une responsabilité ;
- une parole que Player avait déplacée sur quelqu’un d’autre ;
- la possibilité pour une personne d’agir sans être observée ;
- une place dans le foyer ;
- une version qui appartient de nouveau à son autrice.

J16 ne promet aucune réparation complète.

Elle produit un état praticable.

---

# 3. Différence avec J15

J15 rendait les priorités visibles par les horaires.

J16 traite ce que cette priorité a produit.

J15 pouvait créer :

- une heure tenue ;
- une heure abandonnée honnêtement ;
- une personne laissée attendre ;
- un mensonge ;
- une preuve ;
- une image retirée ;
- un travail perdu ;

```text

```

# 4. Ordre de priorité invisible

Le joueur ne voit jamais cette liste.

Le pivot J16 est sélectionné selon :

```text
1. sécurité ou consentement
2. audience compromise ou suppression
3. après-coup adulte encore impayé
4. responsabilité professionnelle
5. foyer et logement
6. alibi, horaire ou promesse
7. clarification du couple
8. fermeture propre si aucune dette grave ne subsiste
```

## 4.1 Sécurité ou consentement

Passe toujours en premier si :

- Mathilde a demandé à ne pas rester seule avec Player ;
- une distance explicite a été ignorée ;
- Player est revenu dans un lieu pendant une fenêtre interdite ;
- un refus ou un stop a été contesté ;
- le logement a été utilisé comme accès ;
- une personne ne se sent plus libre de rester dans le foyer.

Aucune autre relation ne peut offrir une compensation.

## 4.2 Audience ou suppression

Passe ensuite si :

- une image privée a été vue hors audience ;
- la personne représentée n’a pas encore reçu les informations exactes ;
- une suppression promise n’a pas été effectuée ;
- Player a menti sur l’état du contenu ;
- une version privée dépend encore d’un faux accord ;
- une personne ne sait pas qui possède encore la trace.

## 4.3 Après-coup adulte

Passe avant toute résolution si une intimité antérieure produit encore :

- insécurité ;
- silence imposé ;
- confusion sur la permission actuelle ;
- dette conjugale ;
- organisation du logement ;
- demande de retrait.

J16 ne produit aucune nouvelle intimité.

## 4.4 Travail

Passe avant la simple déception relationnelle si :

- un rendu a été manqué ;
- une personne a dû terminer seule ;
- un nom professionnel a été utilisé comme faux alibi ;
- une promesse de correction reste active ;
- une collaboration a été exposée à un risque réel.

## 4.5 Foyer

Passe avant les routes extérieures lorsque :

- Mathilde doit changer de chambre ou de lieu ;
- Marie doit organiser l’espace ;
- Player n’a plus un accès ordinaire au domicile ;
- la fin du séjour doit être préparée ;
- un objet, une clé ou des affaires doivent être rendus.

## 4.6 Horaire, alibi ou promesse

Nico, Marie ou un tiers peut exiger que :

- l’heure réelle soit fixée ;
- un alibi soit retiré ;
- une promesse soit reconnue comme perdue ;
- Player cesse d’utiliser l’incertitude comme version alternative.

## 4.7 Couple

La clarification conjugale devient le pivot seulement si aucun état plus urgent ne réclame d’abord :

- sécurité ;
- audience ;
- suppression ;
- logement ;
- travail.

J17 définira le couple.

J16 crée seulement les conditions dans lesquelles cette définition pourra être honnête.

# 8. Modèle du choix Player

Les formulations varient selon la conséquence, mais trois postures restent possibles.

## P16-A — Restituer précisément

Player :

- reconnaît le fait exact ;
- rend ce qui doit être rendu ;
- confirme l’état du contenu ;
- accepte une limite ;
- distingue ce qu’il sait de ce qu’il suppose ;
- ne demande pas immédiatement une récompense.

## P16-B — Réparer pratiquement sans tout résoudre

Player peut :

- organiser une remise d’objet ;
- confirmer une suppression ;
- accepter une exclusion temporaire ;
- rendre un travail ;
- fixer une information exacte ;
- proposer une action bornée.

La réparation pratique ne restaure pas automatiquement la confiance.

## P16-C — Contester, minimiser ou déplacer

Player peut :

- discuter la légitimité de la règle ;
- transformer une demande en accusation ;
- attribuer la responsabilité au témoin ;
- prétendre qu’un retrait suffit à effacer la connaissance ;
- exiger qu’une personne le rassure ;
- maintenir une version contredite.

Cette posture ferme ou durcit l’état concerné.
