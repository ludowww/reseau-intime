# Réseau Intime — Lot D — Polish des voix et naturel J07–J21

## Statut

**Catégorie : Guide éditorial canonique — passe finale pré-runtime**

Ce document encadre la passe de naturel appliquée directement aux scripts consolidés.

Il ne modifie jamais :

- les choix ;
- les promesses ;
- les heures ;
- les audiences ;
- les traces ;
- les connaissances ;
- les conditions de consentement ;
- les états de sortie ;
- la reachability.

---

# 1. Objectif

Réduire les signes de dialogue trop écrit :

- symétries parfaites ;
- séries de punchlines ;
- formulations qui expliquent la règle du système ;
- diagnostics complets sur Player ;
- personnages toujours capables de nommer exactement leur contradiction ;
- réponses trop propres après une gêne réelle.

Le résultat attendu conserve les décisions fortes mais ajoute :

- banalité ;
- hésitation ;
- corrections ;
- messages incomplets ;
- détails concrets ;
- silences ;
- une seule ligne mémorable par beat.

---

# 2. Règles communes

```text
1 ligne forte maximum par beat
30–50 % de messages ordinaires ou partiels
pas de trio de phrases parfaitement parallèles
pas de résumé moral après chaque décision
pas de vocabulaire canonique visible sauf voix compatible
```

Mots à employer avec parcimonie dans les dialogues :

```text
audience
permission
version
cadre
responsabilité
preuve
dette
règle générale
```

Ils restent possibles pour Pauline et Raphaëlle, mais ne doivent pas être répétés comme un lexique partagé par tout le casting.

---

# 3. Filtres de voix

## Marie

- concret avant abstraction ;
- heures, pain, clés, manteaux, cartons, travail ;
- une émotion apparaît par ce qu’elle fait ou cesse de faire ;
- elle ne récite pas le contrat du couple.

## Sandra

- faible volume ;
- gêne et détours ;
- elle retire une phrase plutôt qu’elle n’en ajoute trois ;
- une limite peut être très courte.

## Mathilde

- rapide ;
- pratique ;
- corrections en cours de phrase ;
- humour ou mauvaise foi légère ;
- elle ne devient pas experte thérapeutique.

## Pauline

- surface, timing, fil, suppression, compartiment ;
- phrases sèches possibles ;
- moins de démonstrations conceptuelles ;
- ses actes de fermeture comptent davantage que son analyse.

## Raphaëlle

- fabrication, fichier, lumière, matière, délai ;
- questions simples ;
- moins de diagnostics moraux ;
- silence possible après une réponse insuffisante.

## Nico

- court ;
- bar, chaise, service, heure ;
- provocation puis question réelle ;
- aucune leçon générale.

## Player

- minuscules ;
- court ;
- parfois maladroit ;
- ne formule pas toujours parfaitement son intention ;
- ses actes et heures complètent ses mots.

---

# 4. Tests

Après la passe :

- chaque personnage reste identifiable sans nom ;
- aucune décision canonique ne change ;
- aucun identifiant de contrat ne change ;
- aucune phrase polie ne crée une nouvelle promesse ;
- les limites restent nettes même si leur formulation devient plus humaine ;
- l’après-coup et les conséquences restent lisibles ;
- les personnages peuvent laisser une question partiellement ouverte.

---

# 5. Règle finale

> **Le naturel ne consiste pas à rendre les personnages confus. Il consiste à leur laisser une vie, une gêne et un rythme autour des décisions qu’ils ont réellement prises.**
