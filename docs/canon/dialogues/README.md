# Réseau Intime — Dialogues canoniques

## Statut

Ce dossier contient les scripts narratifs complets de la saison 1 et leurs audits canoniques.

Ordre obligatoire :

```text
plan détaillé de journée
→ script narratif complet
→ audit de conformité
→ validation de Ludovic
→ corrections documentaires validées
→ contrat d’état et de traçabilité
→ seulement ensuite adaptation JSON/runtime
```

## Autorités correctives

```text
J01_J09_AUDIT_CONFORMITE_NARRATIVE.md
J01_J21_AUDIT_GLOBAL_DIALOGUES_CONTINUITE.md
```

Le premier document corrige les écarts identifiés dans J01–J09.

Le second audit, validé après J21, fixe les blocages transversaux à corriger avant toute reprise technique.

## Règles

- messagerie texte uniquement ;
- aucun appel joué ;
- aucun message vocal ;
- aucune scène audio ;
- aucun asset conçu dans les scripts ;
- aucune mécanique créée dans cette couche ;
- les événements physiques restent hors téléphone ;
- toute promesse importante est un vrai choix ;
- une seule relation, conséquence ou découverte reçoit le pivot majeur ;
- les conséquences prioritaires passent avant les nouvelles occasions ;
- toute trace utilisée pour une découverte doit déjà exister ;
- le témoin ne connaît que ce que l’écran affiche ;
- suppression du contenu et connaissance du témoin restent distinctes ;
- tout passage adulte est négocié, révocable et suivi d’un après-coup ;
- les images possèdent origine, auteur, audience et circulation.
