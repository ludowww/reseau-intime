## Statut

**Catégorie : Canon validé narrativement — source pré-runtime consolidée**

**Périmètre : J15 uniquement**

**Surface : messagerie texte uniquement**

Ce document définit la version narrative complète de J15 avant toute adaptation technique.

Il s’appuie sur :

- `J14_SCRIPT_NARRATIF_COMPLET.md` ;
- les obligations réellement créées en J10–J14 ;
- `S28 — Les horaires ne tiennent plus` ;
- les règles de conséquence, dette, preuve, audience et consentement ;
- le canon text-only.

Il ne contient :

- aucun JSON ;
- aucun flag runtime définitif ;
- aucun script Godot ;
- aucune migration ;
- aucun asset ;
- aucun prompt ComfyUI ;
- aucune scène audio ;
- aucun appel joué ;
- aucun message vocal.

---

# 1. Décision principale de la journée

J15 devient :

```text
Mardi — Les horaires ne tiennent plus
```

Sa fonction est :

```text
rendre incompatibles au moins deux obligations déjà créées
afin que la priorité réelle de Player devienne visible
```

J15 ne demande pas :

```text
quelle relation veux-tu choisir ?
```

J15 demande :

```text
quelle obligation vas-tu réellement payer
lorsque tu ne peux plus maintenir toutes les versions de ta journée ?
```

La priorité n’est jamais affichée comme une route.

Elle apparaît dans :

- l’heure tenue ;
- l’heure déplacée ;
- la personne prévenue ;
- la personne laissée attendre ;
- le mensonge construit ;
- le lieu où Player se rend réellement ;
- l’appareil qu’il décide de poser ;
- la personne qui agit sans lui.

---

# 2. Question dramatique

> Quelle vérité devient visible lorsque Player ne peut plus tenir simultanément les heures, les versions et les attentes qu’il a lui-même créées ?

Une réponse honnête peut coûter :

- une occasion ;
- un accès privé ;
- une confiance professionnelle ;
- une soirée de couple ;
- une présence au foyer ;
- une relation qui refuse d’être replacée plus tard.

Un mensonge peut préserver quelques minutes ou une occasion.

Il doit alors créer :

```text
une dette
+
une trace
+
une preuve possible
+
une conséquence J16
```

La sincérité ne répare pas automatiquement.

Le mensonge n’est pas seulement un choix de ton.

Il modifie ce que les personnages peuvent raisonnablement croire et décider.

---

# 3. Différence obligatoire avec J08

J08 montrait une première superposition.

Les attentes pouvaient encore être :

- amendées facilement ;
- reformulées ;
- différées sans accumulation grave ;
- abandonnées avant de devenir des obligations.

J15 arrive après :

- plusieurs promesses choisies ;
- une convergence sociale ;
- une conséquence payée en J13 ;
- une découverte partielle en J14 ;
- une personne représentée éventuellement prévenue ;
- un témoin qui sait désormais qu’une autre vérité existe.

J15 doit donc montrer :

```text
deux obligations antérieures minimum
+
une fenêtre horaire réellement incompatible
+
une priorité accomplie dans les faits
+
une personne autonome qui cesse d’attendre
```

J15 ne peut pas être résolu par :

- `je verrai plus tard` ;
- une réponse vague envoyée aux deux personnes ;
- une absence sans conséquence ;
- une confession générale ;
- une nouvelle scène adulte ;
- une image séduisante de compensation.

---

# 4. Conditions d’entrée canoniques

J15 exige au minimum :

## Obligation A — conséquence directe de J14

Une obligation doit venir de la découverte ou de la clarification J14.

Exemples :

- expliquer à Marie ce que Player avait accepté avec Pauline ;
- informer Sandra de ce que Mathilde a réellement vu ;
- garantir une règle de sécurité ou de distance à Mathilde ;
- expliquer à Marie la place réelle de Raphaëlle ;
- reconnaître l’heure réelle connue par Nico ;
- expliquer une notification privée apparue sur la photographie publique du couple.

## Obligation B — engagement antérieur encore actif

La seconde obligation doit avoir été créée avant J15.

Elle peut être :

- une heure de couple choisie avec Marie ;
- une présence au foyer acceptée ;
- une distance demandée par Mathilde ;
- une tâche professionnelle promise à Raphaëlle ;
- une réponse sur l’audience due à Sandra ;
- un compte rendu demandé par Pauline ;
- une rencontre extérieure réellement confirmée ;
- une suppression promise ;
- une absence du foyer acceptée ;
- une vérité sur l’heure encore due ;
- un engagement précis avec Nico.

Une simple attirance n’est pas une obligation.

Une possibilité de route n’est pas une obligation.

Un personnage disponible n’est pas une obligation.

Une photographie reçue n’est pas une obligation tant qu’aucune règle, réponse ou conséquence n’est due.

# 5. Validateur narratif invisible

Avant de jouer J15, l’histoire doit pouvoir répondre :

```text
obligation_1_source
obligation_1_personne
obligation_1_action_due
obligation_1_heure_ou_fenêtre
obligation_1_acceptée_par_player

obligation_2_source
obligation_2_personne
obligation_2_action_due
obligation_2_heure_ou_fenêtre
obligation_2_acceptée_par_player
```

Les deux obligations doivent être attribuables à des messages ou à des actes antérieurs.

J15 ne peut pas transformer rétroactivement :

- une phrase polie en rendez-vous ;
- une disponibilité en promesse ;
- une attirance en dette ;
- une image en autorisation ;
- un silence en acceptation.

Si aucune seconde obligation réelle n’existe, le canon amont doit être corrigé.

J15 ne fabrique pas un rendez-vous artificiel pour sauver son titre.

Une compatibilité historique éventuelle peut payer l’unique dette existante, mais elle ne constitue pas la version canonique complète de S28.

---

# 6. Budget de la journée

Dans chaque partie :

```text
2 obligations antérieures minimum
1 collision horaire principale
1 choix majeur
1 priorité rendue visible par un acte
1 personne non servie qui agit seule
0 nouvelle route dominante
0 progression adulte
0 image-récompense
3 contenus visuels minimum
1 dette majeure ou réparation partielle en sortie
```

Une seule conséquence devient foreground.

Les autres personnages peuvent agir, fermer une attente ou laisser une trace, mais ils ne reçoivent pas chacun une scène complète.

# 7. Architecture générale

```text
08:00–09:00
confirmation d’une obligation déjà due

11:30–14:00
la seconde personne fixe sa propre fenêtre réelle

16:30–18:30
les deux horaires deviennent objectivement incompatibles

avant la première fenêtre
Player peut encore être honnête, amender ou refuser

pendant la collision
une seule obligation est réellement accomplie

après la collision
la personne non servie agit sans Player

soir
retour textuel de la conséquence principale

fin de journée
dette précise transmise à J16
```

Les heures exactes varient selon les obligations actives.

Elles ne sont jamais choisies uniquement pour produire une collision.

Elles découlent :

- d’un poste ;
- d’un départ ;
- du retour de Bastien ;
- de la présence de Marie ;
- d’un besoin de sécurité au foyer ;
- d’une fermeture professionnelle ;
- d’un fichier envoyé par Maud ;
- d’un service de Nico ;
- d’une heure déjà proposée et acceptée.

---

# 8. Règle des scènes

## Scène 1 — Confirmation

Fonction unique :

```text
rappeler qu’une obligation existe encore
```

Elle ne crée pas une nouvelle intimité.

## Scène 2 — Verrouillage

Fonction unique :

```text
donner une fenêtre horaire réelle à la seconde obligation
```

La personne agit selon sa propre journée.

## Scène 3 — Collision

Fonction unique :

```text
obliger Player à accomplir, amender ou trahir un engagement
```

## Scène 4 — Autonomie

Fonction unique :

```text
montrer que la personne non servie n’attend plus passivement
```

## Scène 5 — Retour

Fonction unique :

```text
transformer l’horaire en conséquence relationnelle
```

Aucune scène ne doit simultanément :

- ouvrir une route ;
- livrer une image intime ;
- résoudre le couple ;
- créer un nouveau secret ;
- produire une progression adulte.

---

# 9. Modèle du choix principal

Le joueur ne reçoit jamais :

```text
Choisir Marie
Choisir Pauline
Choisir Sandra
Choisir Mathilde
Choisir Raphaëlle
Choisir Nico
```

Les fils arrivent dans l’ordre crédible de la journée.

Le choix apparaît dans la manière de répondre à une obligation déjà visible.

Trois postures restent possibles.

## H15-A — Dire que les deux horaires ne tiennent pas

Player reconnaît la collision avant qu’une personne commence à attendre.

Il doit préciser :

- quelle heure il peut réellement tenir ;
- quelle heure il ne peut plus tenir ;
- ce qu’il refuse de promettre ;
- qui sera prévenu ;
- quelle conséquence il accepte.

Une première réponse Player peut rester imparfaite :

> j’ai dit oui trop vite

Puis être précisée :

> je ne peux pas être avec toi de 19 h à 20 h et répondre à Pauline après notre discussion avant 19 h 25. je dois arrêter de faire comme si les deux tenaient

Cette posture ne répare rien automatiquement.

## H15-B — Maintenir une obligation et amender l’autre

Player garde une heure et propose une alternative précise.

L’autre personne peut :

- accepter ;
- refuser ;
- annuler ;
- agir sans lui ;
- fermer l’attente.

Un refus clair ferme réellement la fenêtre.

## H15-C — Maintenir deux versions incompatibles

Player affirme aux deux personnes qu’il sera disponible.

Il peut utiliser :

- un faux horaire ;
- un faux lieu ;
- une fausse annulation ;
- un faux accord ;
- une suppression prétendue ;
- une personne comme alibi.

Le mensonge doit être contredit par une trace crédible :

- horodatage ;
- message déjà envoyé ;
- réponse directe d’un témoin ;
- publication dans un fil partagé ;
- retrait autonome d’une image ;
- départ réel ;
- retour au foyer ;
- fichier rendu à une heure visible.

La preuve démontre la contradiction.

Elle ne révèle pas automatiquement toute la relation.

# 12. Variante Mathilde — La distance du foyer entre en collision avec une autre obligation

## 12.1 Éligibilité

Cette variante existe si Marie a vu une notification Mathilde en J14 et si le foyer exige :

- une règle de non-répétition ;
- une discussion séparée ;
- une distance ;
- une protection de Mathilde ;
- une réorganisation du logement.

Une seconde obligation extérieure doit réellement être active.

Elle peut concerner :

- une tâche Raphaëlle ;
- une réponse Sandra ;
- un retour Pauline ;
- une rencontre déjà confirmée ;
- une présence professionnelle ;
- une heure donnée à Nico.

La variante ne choisit pas arbitrairement la relation la plus séduisante.

Elle utilise l’engagement antérieur le plus précis.

## 12.2 Priorité sécurité

Si Mathilde a demandé de ne pas rester seule avec Player ou si une limite a été ignorée :

```text
sécurité du foyer
>
toute obligation extérieure
```

Le joueur peut encore refuser cette priorité.

Le récit ne présente pas les deux choix comme moralement équivalents.

| Mensonge | Preuve possible |
|---|---|
| fausse heure | message ou horodatage réel |
| faux lieu | publication ou fil professionnel partagé |
| faux accord | message direct de la personne concernée |
| fausse suppression | état réel du contenu |
| faux silence | message autonome du témoin |
| faux alibi | réponse factuelle de Nico |

- révéler un fil entier ;
- montrer une image privée à une nouvelle audience ;
- rendre Marie omnisciente ;
- transformer un soupçon en connaissance totale ;
- inventer une capture secrète.
