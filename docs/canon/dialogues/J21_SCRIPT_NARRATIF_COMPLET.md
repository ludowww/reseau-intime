**Catégorie : Canon validé narrativement — source pré-runtime consolidée**

Ce document définit la finale narrative complète de la première saison avant toute adaptation technique.

# 1. Décision principale de la journée

J21 devient :

```text
Lundi — La dernière photo change de sens
```

Sa fonction est :

```text
montrer une vie ordinaire transformée
et recontextualiser une trace déjà connue
sans ajouter de révélation,
sans sélectionner une route
et sans annuler les conséquences précédentes
```

J21 ne demande pas :

```text
qui choisis-tu ?
```

Elle demande :

```text
que fais-tu maintenant
avec la règle, la perte ou la contradiction
que toute la saison a rendue visible ?
```

La finale repose sur trois éléments :

```text
une vie ordinaire
+
une trace ancienne
+
un comportement actuel
```

Elle ne résume pas les vingt journées précédentes.

Elle montre leur effet.

# 2. Question dramatique

> Quelle image du réseau reste lorsque les mêmes personnes, les mêmes lieux et les mêmes visages ne peuvent plus être regardés comme au début ?

L’image finale ne révèle pas un secret caché.

Elle montre qu’une trace connue possède désormais :

- une autre audience ;
- une autre permanence ;
- une autre absence ;
- une autre dette ;
- une autre limite ;
- ou une autre place dans la vie de Player.

# 4. Budget de la finale

Dans chaque partie :

```text
1 vie ordinaire transformée
1 trace principale déjà existante
1 dernier choix de posture
1 conséquence visible
0 nouvelle route
0 nouvelle révélation
0 nouvelle image intime
0 permission rétroactive
0 confession exhaustive
0 écran de résultat
3 contenus visuels minimum
```

La finale ne doit pas devenir plus longue uniquement parce que plusieurs routes ont été ouvertes.

Une seule trace reçoit le premier plan.

Les autres relations existent par leurs nouveaux comportements, leurs silences ou leurs absences.

# 5. Sélection invisible de la trace principale

Le joueur ne choisit pas une photographie dans une galerie.

La trace est sélectionnée par l’histoire.

Ordre de résolution :

```text
1. trace liée à une dette consciemment maintenue
2. trace liée à la route dominante
3. trace liée à la transformation du couple
4. trace ayant changé d’audience ou de permanence
5. trace sociale du réseau
6. fallback Marie / vie commune
```

## 5.1 Interdictions

La sélection ne peut pas :

- privilégier automatiquement l’image la plus sexuelle ;
- faire revenir un fichier supprimé ;
- montrer à Player une image dont il n’est plus l’audience ;
- fabriquer une nouvelle photographie pour symboliser la saison ;
- classer les personnages ;
- transformer la galerie en palmarès ;
- permettre à Player de choisir la femme correspondant à l’image.

## 5.2 Une absence peut être la trace

Si l’image la plus importante a été retirée, J21 peut montrer :

```text
contenu retiré
image inaccessible
emplacement vide
message annonçant la conservation privée
absence de la photo attendue
```

L’absence est alors l’état final de l’image.

Elle ne devient pas un prétexte pour la restaurer.

Les trois significations cachées sont :

```text
A — agir selon la nouvelle règle

B — reconnaître une perte ou une dette

C — maintenir consciemment une contradiction
```

```text
final_trace_id doit référencer J01_J21_TRACE_REGISTRY.md
```


# 29. Effet de l’accumulation sur le dernier choix

Le choix final ne peut pas effacer les vingt journées précédentes.

## 29.1 Une réponse A ne produit pas automatiquement une reconquête

Si Player a :

- menti de manière répétée ;
- violé une audience ;
- contesté une sécurité ;
- utilisé plusieurs alibis ;
- refusé les discussions ;

alors agir correctement en J21 peut seulement :

- éviter une aggravation ;
- confirmer une fermeture ;
- commencer une réparation future.

## 29.2 Une réponse B ne produit pas automatiquement un pardon

Reconnaître une perte peut préserver la dignité d’une relation.

Cela ne restaure pas :

- l’image ;
- le couple ;
- le foyer ;
- la confiance ;
- l’accès privé.

## 29.3 Une réponse C n’annule pas les résolutions

La saison reste complète.

Elle se termine avec une contradiction identifiée qui pourra produire un nouvel arc.

# 49. Résumé canonique candidat

J21 commence par une matinée ordinaire.

Marie parle :

- de pain ;
- de café ;
- d’une heure ;
- d’une liste ;
- de cartons ;
- ou d’une règle déjà décidée.

Ce quotidien n’est plus celui du début.

Le couple peut être :

- en reconquête ;
- en suspension ;
- en négociation ;
- en double vie ;
- fracturé ;
- ou séparé.

Une trace ancienne revient ensuite.

Elle peut être :

- la photographie du déjeuner Sandra ;
- une photographie Marie / La Verrière ;
- une image du foyer Mathilde ;
- une version publique ou privée Pauline ;
- une image choisie Raphaëlle ;
- une photographie sociale liée à Nico ;
- ou l’absence d’un contenu retiré.

La trace ne révèle rien de nouveau.

Elle possède seulement un sens différent.

Une photographie peut désormais être :

- un souvenir sans accès ;
- une confiance précise ;
- une dette ;
- une version publique toujours vraie ;
- un processus sans permission corporelle ;
- une image sociale qui ne peut plus être sexualisée ou transférée automatiquement ;
- une preuve qu’une personne a cessé d’attendre ;
- ou un contenu inaccessible dont l’absence doit être respectée.

Le dernier choix ne demande pas à Player quelle femme il choisit.

Il lui permet de :

- agir selon la nouvelle règle ;
- reconnaître une perte ou une dette ;
- maintenir consciemment une contradiction déjà existante.

Le résultat dépend de toute la saison.

Une bonne réponse finale n’efface pas les mensonges, les absences, les refus ou les audiences violées.

Une réponse imparfaite n’efface pas non plus les actes de présence, les retraits respectés ou les vérités déjà assumées.

La saison se termine sur une vie qui continue.

Marie peut encore demander du pain.

Sandra peut garder une photographie pour elle.

Mathilde peut regarder une ancienne image comme une trace familiale plutôt qu’un accès domestique.

Pauline peut maintenir ou fermer un compartiment sans que sa vie publique devienne fausse.

Raphaëlle peut conserver une version tout en fermant l’accès.

Nico peut rester un ami, un rival, un témoin ou un complice sans devenir propriétaire du désir de qui que ce soit.

Les mêmes images existent.

Les personnes qui les regardent ne sont plus les mêmes.

> **La dernière photo ne change pas. Ce sont les permissions, les absences, les dettes et les vies autour d’elle qui lui donnent enfin un autre sens.**
