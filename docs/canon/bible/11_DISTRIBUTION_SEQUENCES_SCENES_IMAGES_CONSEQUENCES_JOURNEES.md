# Réseau Intime — 11 Distribution des séquences, scènes, images et conséquences en journées

## Statut du document

**Catégorie : Canon**

**Niveau : Calendrier narratif et rythme de production de la première saison**

Ce document transforme les actes, les séquences, les règles de scènes, la progression visuelle et la carte des conséquences en un calendrier de saison jouable.

Il définit :

- le nombre cible de journées narratives ;
- leur fonction dramatique ;
- leur densité ;
- les séquences fixes et variables ;
- la place de Marie ;
- le nombre de progressions majeures possibles ;
- les fenêtres réservées aux conséquences ;
- la distribution des images ;
- les absences et retours ;
- les règles de mutation en cas de refus ou d’occasion manquée ;
- la compatibilité avec l’ouverture runtime actuelle.

Il complète :

- `04_TRAME_PRINCIPALE_ET_ACTES_SAISON_1.md` ;
- `05_ROUTES_MACRO_SAISON_1.md` ;
- `06_EVOLUTION_EROTIQUE_DES_ROUTES.md` ;
- `07_BIBLIOTHEQUE_DE_SEQUENCES_SAISON_1.md` ;
- `08_REGLES_DES_SCENES_MODULAIRES.md` ;
- `09_PROGRESSION_VISUELLE_ET_PHOTOGRAPHIQUE.md` ;
- `10_CARTE_CONSEQUENCES_DETTES_SECRETS_OBLIGATIONS.md` ;
- les canons complets des personnages.

Il remplace comme autorité principale de densité quotidienne :

`docs/narrative/DAILY_RHYTHM_AND_CONTENT_DENSITY.md`

Ce document historique reste utile pour comprendre l’origine de certains garde-fous, mais ne gouverne plus le calendrier de la saison.

Ce document ne contient :

- aucun dialogue définitif ;
- aucune scène directement intégrable ;

- aucun asset ;
- aucun schéma JSON ;
- aucune variable runtime obligatoire ;
- aucune validation automatique des journées existantes.

# 1. Principe directeur

> **Une journée ne sert pas à faire avancer le plus de routes possible. Elle sert à rendre un changement principal lisible, à montrer ses traces et à préparer ou payer ce qu’il crée.**

Une journée réussie possède :

- une fonction dominante ;
- un état d’entrée ;
- un pivot principal ;
- éventuellement un pivot secondaire ;
- une respiration ;
- trois contenus visuels distincts au minimum ;
- un état de sortie ;
- des obligations mises à jour ;
- une raison claire pour les personnages absents.

Une journée ne doit pas ressembler à :

```text
message Marie
→ photo Sandra
→ scène Mathilde
→ crop Pauline
→ costume Raphaëlle
→ proposition Nico
→ tout le monde monte d’un niveau
```

Elle doit plutôt produire :

```text
un mouvement principal
→ une réaction ou un contraste
→ une trace ou une conséquence
→ un nouvel équilibre provisoire
```

# 2. Forme canonique de la première saison

## 2.1 Durée

La première saison utilise un cadre de production de :

```text
21 journées narratives consécutives
```

Ce volume permet :

- une ouverture lisible ;
- plusieurs répétitions avant les bascules ;
- une vraie place aux absences ;
- une progression visuelle régulière ;
- des conséquences entre les scènes intenses ;
- une convergence sociale ;
- cinq journées de résolution ;
- la fin du séjour temporaire de Mathilde dans sa fenêtre de dix à quinze jours ;
- une saison complète sans transformer chaque journée en catalogue.

## 2.2 Semaine de départ

Le calendrier narratif commence un :

```text
mardi
```

Il se termine le lundi de la troisième semaine.

```text
J01 mardi
J02 mercredi
J03 jeudi
J04 vendredi
J05 samedi
J06 dimanche
J07 lundi
J08 mardi
J09 mercredi
J10 jeudi
J11 vendredi
J12 samedi
J13 dimanche
J14 lundi
J15 mardi
J16 mercredi
J17 jeudi
J18 vendredi
J19 samedi
J20 dimanche
J21 lundi
```

Les dates calendaires exactes restent à fixer lors du plan d’intégration.

Les jours de la semaine sont canoniques pour :

- les horaires de travail ;
- les événements de La Verrière ;
- les services de Nico ;
- les horaires décalés de Sandra ;
- les week-ends de groupe ;
- la durée du séjour de Mathilde ;
- la plausibilité des obligations.

## 2.3 Journée charnière

J12 ferme l’acte III et ouvre l’acte IV.

Une journée peut appartenir à deux mouvements dramatiques successifs lorsqu’un événement :

- paie les lignes privées ;
- puis crée immédiatement la convergence.

Les actes restent des états dramatiques, pas des boîtes horaires rigides.

# 3. Répartition des actes

```text
Acte I   — La réouverture              J01 à J04
Acte II  — Les lignes privées          J05 à J08
Acte III — Les vies parallèles         J09 à J12
Acte IV  — La convergence              J12 à J16
Acte V   — La vérité supportable       J17 à J21
```

## Acte I — 4 jours

Fonction : faire exister tout le réseau dans des contextes ordinaires.

Aucune route adulte active.

## Acte II — 4 jours

Fonction : donner à plusieurs relations une continuité propre et créer la première superposition.

Au moins deux attentes privées existent à la fin.

## Acte III — 4 jours

Fonction : transformer une ou deux lignes en réalité coûteuse.

Une route extérieure dominante peut apparaître.

## Acte IV — 5 jours, dont J12 charnière

Fonction : faire entrer les versions privées dans un même espace et payer les collisions.

## Acte V — 5 jours

Fonction : définir le couple, fermer le séjour de Mathilde, donner un état à chaque relation et recontextualiser la dernière image.

# 4. Journée, fenêtre et scène

Une journée contient plusieurs fenêtres temporelles.

Exemples :

- matin ;
- trajet ;
- midi ;
- après-midi ;
- début de soirée ;
- soirée ;
- après-séparation.

Une fenêtre peut contenir :

```text
1 scène de premier plan
0 à 2 échos
```

Une journée contient généralement :

```text
2 à 4 fenêtres utiles
```

Elle ne doit pas remplir chaque moment de la journée.

## Règle de premier plan quotidien

Par défaut :

```text
1 pivot principal de premier plan
```

Une journée peut comporter un second premier plan si celui-ci est :

- une conséquence ;
- un contraste ;
- une scène de couple liée au pivot principal ;
- une résolution courte ;
- une partie du même événement.

Le second premier plan ne doit pas constituer une seconde grande progression de route indépendante.

## Exception convergence et résolution

J12, J17, J19 et J20 peuvent contenir plusieurs scènes importantes.

Elles doivent néanmoins conserver :

- une fonction de journée unique ;
- une seule bascule dominante ;
- des états de sortie lisibles.

# 5. Types de journées

## 5.1 Journée légère

```text
1 pivot
1 à 2 conversations ou scènes fortes
0 à 2 échos
3 contenus visuels
```

Utilisée pour :

- ouverture ;
- respiration ;
- après-coup ;
- retrait ;
- préparation.

## 5.2 Journée normale

```text
1 pivot principal
1 réaction ou contraste
2 à 4 conversations / scènes utiles
3 à 4 contenus visuels
```

C’est le format par défaut.

## 5.3 Journée riche

```text
1 pivot principal
1 pivot secondaire lié
1 conséquence ou respiration
3 à 5 scènes / conversations utiles
4 à 6 contenus visuels
```

Utilisée pour :

- événement social ;
- première superposition ;
- scène adulte avec après-coup ;
- convergence ;
- résolution multiple.

## 5.4 Journée de conséquence

```text
aucune nouvelle escalade obligatoire
1 conséquence majeure
1 retour ordinaire
1 espace de respiration
3 contenus visuels
```

Elle peut être très forte sans créer de nouvelle opportunité.

## 5.5 Journée charnière

Ferme un acte et ouvre le suivant.

J12 est la journée charnière principale.

# 6. Budget des routes

## 6.1 Marie n’est pas une route optionnelle

Marie reste la relation centrale.

Elle possède :

- des scènes propres ;
- des images propres ;
- des actions indépendantes ;
- des obligations ;
- une transformation complète.

Son espace narratif ne dépend pas du fait que le joueur poursuive ou non une route extérieure.

## 6.2 Route extérieure dominante

Une seule route extérieure peut devenir dominante par défaut :

```text
Sandra
ou
Mathilde
ou
aucune route extérieure dominante
```

La saison autorise une bascule complète de Sandra ou Mathilde, mais ne doit pas supposer que les deux atteignent automatiquement leur profondeur maximale dans une même partie.

Une double bascule extérieure reste possible uniquement si :

- elle a été écrite comme variante spécifique ;
- les obligations et collisions sont payées ;
- le couple et le réseau réagissent ;
- elle ne devient pas le chemin de production par défaut.

## 6.3 Routes secondaires

Pauline, Raphaëlle et Nico doivent obtenir :

- une ligne identifiable ;
- une scène ou décision autonome ;
- une conséquence ;
- un état final.

Ils ne doivent pas tous atteindre une profondeur équivalente à Sandra ou Mathilde.

## 6.4 Capacité par acte

### Acte I

```text
Marie : active
Toutes les routes extérieures : R0 ou R1
Aucune permission adulte
```

### Acte II

```text
maximum 2 routes extérieures à R2
maximum 1 début de R3
aucun cadre adulte complet
```

### Acte III

```text
maximum 1 route extérieure dominante à R4
maximum 1 route secondaire à R3
les autres restent R1/R2, refroidissent ou ferment
```

### Acte IV

```text
maximum 1 bascule extérieure profonde par défaut
les autres routes produisent conséquence, position ou hook
```

### Acte V

```text
aucune nouvelle route ouverte
résolution, payoff, retrait ou futur concret uniquement
```

## 6.5 Propriétaire de progression dominante

Chaque bloc de deux à quatre jours doit savoir quelle relation possède la progression dominante.

Exemples :

```text
J05–J08 : Marie + premières attentes, aucune route verrouillée
J09–J12 : Marie + route extérieure dominante provisoire
J13–J16 : convergence et conséquences
J17–J21 : résolution
```

# 7. Sélection de la route dominante

La route dominante ne doit jamais être choisie par un menu.

Elle émerge de :

- temps accordé ;
- attention répétée ;
- promesses tenues ;
- limites respectées ;
- images reçues et utilisées ;
- occasions acceptées ;
- choix de retour vers Marie ;
- conséquences assumées.

## Étapes

### Fin J04

Aucune route dominante.

### Fin J08

Une ou deux continuités extérieures sont plus actives.

Aucun verrou définitif.

### Fin J10

Une route dominante provisoire peut être lisible.

### Fin J12

Une route dominante est généralement établie par :

- une intention reconnue ;
- une promesse coûteuse ;
- une image destinée ;
- une rencontre ;
- un cadre ;
- un refus qui ferme la concurrence.

### Après J12

Le changement de route dominante reste possible seulement par :

- fermeture ;
- violation ;
- retrait ;
- choix explicite de renoncer ;
- conséquence majeure.

La nouvelle route ne récupère pas automatiquement les niveaux de l’ancienne.

# 8. Centralité quotidienne de Marie

Après J04 :

> **Deux journées complètes consécutives ne doivent pas s’écouler sans présence directe de Marie ou conséquence active sur le couple.**

Présence directe signifie :

- scène ;
- conversation ;
- action hors téléphone ;
- image ;
- invitation ;
- retour au foyer.

Une simple mention ne suffit pas toujours.

## Fonctions Marie réparties dans la saison

- J01 : vie partagée et attention ;
- J02 : accueil de Mathilde ;
- J03 : retour du travail / du foyer ;
- J04 : réseau social et fermeture de l’ouverture ;
- J05 : avis et présence choisie ;
- J08 : conséquence de la première superposition ;
- J09 : visibilité hors cadre ;
- J12 : présence centrale à La Verrière ;
- J14–J16 : réaction aux collisions ;
- J17 : définition du couple ;
- J21 : vie ordinaire après la bascule.

Marie peut apparaître davantage selon les choix.

# 9. Présence et absence des autres personnages

## Règle générale

Tous les personnages n’écrivent pas chaque jour.

Une absence doit être :

- naturelle ;
- choisie ;
- conséquence ;
- cooldown ;
- attente ;
- vie propre.

## Longueur d’absence recommandée

### Sandra

Peut être absente deux à trois jours après :

- image forte ;
- phrase trop précise ;
- limite ;
- rencontre.

### Mathilde

Peut être absente du téléphone tout en restant présente physiquement dans le foyer.

### Pauline

Peut rester silencieuse plusieurs jours lorsque la surface officielle doit reprendre le dessus.

### Raphaëlle

Peut disparaître de la couche privée tout en restant visible au travail.

### Nico

Peut être absent du chat mais présent dans une sortie, une image sociale ou une obligation d’alibi.

## Plafond d’absence

Aucun personnage principal ne doit disparaître plus de cinq journées narratives consécutives sans :

- trace ;
- conséquence ;
- mention signifiante ;
- vie propre visible ;
- fermeture explicitement assumée.

# 10. Contrat visuel des journées

Toute journée complète propose au minimum :

```text
3 contenus visuels distincts
```

Ils peuvent être :

- nouveaux ;
- variantes de branche ;
- images de scène ;
- images publiques ;
- conséquences visuelles ;
- images antérieures recontextualisées.

## Réutilisation

Une image antérieure peut occuper un beat visuel seulement si :

- son audience change ;
- sa signification change ;
- elle est retirée ;
- elle devient preuve ;
- elle devient souvenir ;
- elle est comparée à une nouvelle version.

Elle ne devient pas un nouvel asset ou un nouvel identifiant.

## Progression dominante

Une seule famille visuelle domine généralement la journée.

Les autres images servent :

- contexte ;
- contraste ;
- groupe ;
- conséquence ;
- respiration.

## Journée sociale

J04 et J12 peuvent contenir :

```text
4 à 6 contenus visuels
```

avec :

- un set social ;
- une image individuelle dominante ;
- une trace ou version privée éventuelle.

## Journée explicite

Une scène sexuelle ou pornographique recommande :

```text
approche
→ acte
→ après-coup
```

Les trois images du jour ne doivent pas toutes être des variations de l’acte.

---

# 11. Distribution du contenu adulte

Aucun quota d’actes sexuels n’est imposé.

Le calendrier impose seulement :

- aucune scène explicite en acte I ;
- acte II limité à désir, images choisies et cadres émergents ;
- première scène explicite possible en J10 ou J11 sur une route suffisamment avancée ;
- aucune nouvelle scène explicite le jour suivant sans après-coup ;
- une scène de convergence adulte possible uniquement si tous les participants sont séparément éligibles ;
- un payoff tardif possible en J18 ou J19 s’il remplace la résolution standard de la route ;
- aucune nouvelle route sexuelle ouverte en J20 ou J21.

## Anti-clustering

Deux scènes explicites ne doivent pas apparaître sur deux journées consécutives, sauf si la seconde est :

- un après-coup ;
- une continuation du même événement ;
- une scène conjugale directement liée à une négociation déjà ouverte.

## Route Sandra

Une intimité sexuelle exige avant elle :

- continuité ;
- image choisie ou confiance équivalente ;
- limite respectée ;
- possibilité d’après-coup ;
- distinction entre moment vécu et fichier.

## Route Mathilde

Une intimité sexuelle exige :

- intention reconnue ;
- capacité de retrait ;
- risque domestique ;
- conséquence avant la fin du séjour.

## Marie

La reconquête sexuelle doit modifier le lendemain et les comportements ordinaires.

---

# 12. Priorité quotidienne des conséquences

Avant de remplir une journée, appliquer l’ordre du document `10` :

```text
sécurité / consentement / retrait
→ après-coup adulte ou conflit majeur
→ obligation à échéance dure
→ mensonge ou alibi en collision
→ retour Marie / conséquence de couple
→ obligation relationnelle souple
→ continuation de route
→ nouvelle opportunité
→ respiration
```

## Réservation de fenêtres

À partir de J08, toute journée doit réserver conceptuellement une fenêtre disponible pour :

- obligation due ;
- retour Marie ;
- image retirée ;
- clarification ;
- conséquence d’un refus ;
- après-coup.

Cette fenêtre peut rester une respiration lorsqu’aucune conséquence n’est due.

## Règle

Le calendrier n’est jamais rempli entièrement par les opportunités avant que les obligations soient calculées.

# 14. Acte II — J05 à J08

Une séquence parmi :

Journée normale.

## J06 — Dimanche — La première attention répétée

### Fonction

Donner un second cycle à deux personnages sans créer deux pivots majeurs.

### Pivot principal variable

Une séquence non utilisée parmi :

- S09 ;
- S10 ;
- S11 ;
- S12.

### Pivot secondaire

Une scène plus courte parmi les mêmes familles ou une respiration Marie / foyer.

### Règle

- une seule route progresse réellement ;
- la seconde scène crée accès, contexte ou attente ;
- un personnage peut rester absent.

### Visuels

Minimum 3, une seule progression dominante.

### Sortie

- deux relations extérieures peuvent désormais posséder une attente ;
- aucune route verrouillée.

Minimum 3.

# 15. Acte III — J09 à J12

### Fonction

Rendre l’autonomie sociale et le désir de Marie impossibles à réduire à une simple réaction.

### Séquence fixe

S15 — Marie sort du cadre habituel.

### Fixe

- Marie agit pour elle-même ;
- Player peut rejoindre, désirer, surveiller ou s’absenter ;
- une image sociale existe ;
- les obligations de J08 sont prises en compte.

### Visuels

4 possibles :

- préparation ;

# 20. Règles de substitution

## Route indisponible

Si une scène prévue n’est pas éligible :

1. vérifier la conséquence due ;
2. utiliser sa mutation ;
3. donner une autonomie au personnage ;
4. utiliser un autre slot uniquement si cela ne crée pas une progression artificielle.

## Route refusée

Le refus peut produire :

- respect ;
- refroidissement ;
- image publique ;
- action indépendante ;
- disparition temporaire ;
- fermeture.

Il ne produit pas automatiquement une autre femme disponible le même soir.

## Occasion manquée

La scène ne reste pas suspendue indéfiniment.

Exemples :

- Sandra retire l’image ;
- Mathilde demande à quelqu’un d’autre ;
- Marie profite seule ;
- Pauline restaure la surface ;
- Raphaëlle résout avec Maud ;
- Nico va travailler.

## Route dominante fermée

Une route secondaire peut gagner du temps à partir de la journée suivante.

Elle ne reçoit pas rétroactivement :

- les scènes manquées ;
- la confiance ;
- le niveau adulte ;
- les permissions de l’ancienne route.

# 23. Charge visuelle et production

Le calendrier impose trois contenus visuels par journée de partie.

Il n’impose pas de multiplier mécaniquement :

```text
21 jours × toutes les variantes × toutes les routes
```

La production doit distinguer :

- assets communs ;
- sets publics ;
- variantes de route ;
- images de scène ;
- contenus recontextualisés ;
- accès perdu ;
- assets exclusifs à une branche.

## Règle de mutualisation

Un contenu peut être utilisé dans plusieurs filtres ou contextes seulement si :

- son origine reste identique ;
- son identifiant reste unique ;
- sa signification peut raisonnablement changer ;
- il n’est pas dupliqué pour gonfler le volume.

## Règle de personnage

La mutualisation ne doit pas rendre les progressions visuelles interchangeables.
