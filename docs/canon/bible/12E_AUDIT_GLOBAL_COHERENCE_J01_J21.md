# 6. Cohérence de la communication textuelle

## Décision validée

```text
aucune scène audio
aucun appel joué
aucune note vocale
aucun message vocal
aucune scène doublée
```

Tout dialogue interactif est :

- message texte ;
- choix textuel ;
- notification ;
- commentaire d’image ;
- message de groupe.

## Hors téléphone

Les événements physiques restent possibles.

Ils sont représentés par :

```text
messages avant
→ arrêt du chat
→ temps / état / image
→ messages après
```

Cette règle reste crédible avec :

- repas ;
- café ;
- événements ;
- intimité ;
- départ ;
- rencontre ;
- discussion de couple.

## Risque principal

Certaines résolutions, notamment J17, pourraient devenir artificielles si une longue conversation conjugale était forcée dans le chat pendant la co-présence.

Réponse canonique :

- lancer le choix lorsque les personnages sont séparés ;
- arrêter le chat à la rencontre ;
- rendre le résultat par messages ultérieurs ;
- ne jamais créer une scène audio.

# 7. Cohérence des routes

## Route dominante

Une seule route extérieure devient dominante par défaut.

Cela reste compatible avec :

- Sandra ;
- Mathilde ;
- aucune route extérieure.

Pauline et Raphaëlle peuvent devenir plus importantes selon les choix, mais leur profondeur saison 1 reste bornée.

## Refus

Un refus ne déclenche pas automatiquement une autre route.

La saison conserve des jours où :

- aucun personnage extérieur ne progresse ;
- une conséquence prend le premier plan ;
- Marie ou la vie ordinaire domine.

## Changement de route

Possible seulement après :

- fermeture ;
- violation ;
- retrait ;
- renoncement ;
- conséquence majeure.

Aucun transfert automatique de niveau.
