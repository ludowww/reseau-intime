# 2. Budget J09–J12

Une seule relation extérieure peut recevoir le principal développement sur J10–J11.

- J09 comme journée forte ;

# 3. J09 — Mercredi — Marie sort du cadre habituel

## Identité

```text
day_id: J09
day_type: RICH

main_sequence: S15
principal_relationship: Marie / Player
runtime_status: NOT_IMPLEMENTED
```

## Question dramatique

> Player rejoint-il la vie visible de Marie, la regarde-t-il seulement, ou lui laisse-t-il enfin une autonomie réelle sans la transformer en reproche ?

## Fonction

J09 déplace Marie hors du foyer sans la présenter comme une réaction jalouse.

Elle :

- possède une raison professionnelle et sociale ;
- se prépare pour elle-même ;
- demande éventuellement un acte précis ;
- existe si Player ne vient pas ;
- revient ensuite dans la relation par texte.

## Matériau réutilisable

`chapter_03_marie_event_offer` et `chapter_03_marie_event_joined` peuvent être adaptés ici.

À conserver :

- La Verrière ;
- besoin logistique réel ;
- envie personnelle que Player vienne ;
- distinction entre aide et présence ;
- rallonge noire ;
- salle bruyante ;
- messages courts dans le même lieu ;
- conséquence après séparation.

À supprimer de leur ancienne fonction :

- rôle d’ouverture ;
- première grande superposition ;
- arrivée trop précoce de Nico comme commentateur ;
- dépendance au déroulé J03 legacy.

## Fenêtre A — Après-midi — Préparation

### Mode

`REMOTE_ASYNC`

### Pourquoi le texte

Marie est à La Verrière. Player est au travail, en trajet ou au foyer.

### Nœud de décision

1. rejoindre tôt et participer ;
2. venir plus tard avec une heure précise ;
3. ne pas venir et reconnaître que Marie vivra la soirée sans lui.

La troisième posture ne doit pas être formulée comme permission donnée à Marie.

## Fenêtre B — Soir — Événement

### Si Player rejoint

Mode : `SAME_VENUE_LOGISTICS`.

Seulement :

- position ;
- objet ;
- tâche ;
- aparté bref ;
- image envoyée.

Aucune confrontation de couple par messages alors qu’ils peuvent se parler.

### Si Player ne rejoint pas

Marie peut envoyer :

- une trace publique ;
- un message bref ;
- rien jusqu’à la fin.

Son silence ne signifie pas punition automatique.

## Fenêtre C — Nuit — Après séparation

Mode : `AFTERGLOW_REMOTE`.

Fonctions possibles :

- remercier une présence réelle ;
- distinguer présence physique et attention ;
- signaler une absence sans procès ;
- dire que la soirée a été bonne malgré tout ;
- créer un retour concret vers le couple.

## Choix et effets

### Présence active

- Marie mémorise un acte ;
- future reconquête facilitée ;
- aucune récompense sexuelle automatique.

### Présence distraite

- la venue ne neutralise pas l’absence mentale ;
- conséquence active.

### Absence honnête

- Marie agit indépendamment ;
- possible dérive, sans punition arbitraire ;
- Player doit assumer ce qu’il a choisi de privilégier.

## Slots visuels

```text
minimum: 3
asset_design: deferred_to_user_comfyui
```

Fonctions :

1. Marie dans sa vie sociale/professionnelle ;
2. résultat public ou événementiel ;
3. retour privé, autonomie ou conséquence.

## Sortie J09

- Marie est devenue visiblement désirable et autonome ;
- le couple possède un nouvel élément concret ;
- la route extérieure dominante reste provisoire ;
- une obligation peut être créée pour J10 ou J11.

Un seul candidat :

- Sandra S16 ;

### Communication

- aucun dialogue audio joué ;

J11 ne propose pas un catalogue.

## Variante Sandra — S19 La photographie qu’elle a vraiment choisie

### Fonction

Sandra choisit une représentation et observe si Player comprend la permission.

### Communication

`TRACE_DELIVERY` et messages texte.

### Nœud

1. reconnaître la sélection et demander la règle ;
2. répondre au désir sans supposer la sauvegarde ;
3. minimiser ou réclamer davantage.

La troisième posture peut fermer ou refroidir.

### Sorties

- voir seulement ;
- conservation autorisée pour une image précise ;
- image retirée ;
- future confiance ;
- aucun transfert autorisé par défaut.

## Variante Mathilde — S20 Elle revient dans la pièce

### Fonction

### Communication

- messages avant la proximité ;
- arrêt lors de la co-présence ;
- retour textuel après séparation.

### Nœud

1. nommer l’effet sans agir davantage ;
2. accepter une proximité conditionnelle ;
3. restaurer une distance claire.

### Conséquence

Marie et le foyer restent structurellement concernés.

## Variante Pauline

### Fonction

### Nœud

3. refuser d’entrer dans un compartiment.

Bastien reste réel.

## Variante Raphaëlle

### Fonction

Accès au processus ou préparation inachevée.

Pas encore de roleplay sexuel complet par défaut.

### Nœud

1. s’intéresser au travail ;
2. reconnaître l’attractivité sans confondre permission ;
3. protéger la frontière.

## Variante adulte optionnelle

Une première scène physique explicite peut se produire seulement si :

- une route est réellement éligible ;
- les limites sont claires ;
- aucune conséquence prioritaire n’est due ;
- la scène remplace le pivot normal ;
- J12 reflète obligatoirement l’après-coup ;
- tout dialogue interactif reste dans les messages ;
- aucun audio n’est utilisé.

Structure :

```text
messages de négociation
→ temps hors téléphone / images éventuelles
→ messages d’après-coup
```

## Sortie J11

# 6. J12 — Samedi — La Verrière puis L’Annexe

## Identité

```text

runtime_status: NOT_IMPLEMENTED
```

## Question dramatique

> Que deviennent les lignes privées lorsque plusieurs versions des personnages occupent enfin la même soirée ?

## Fonction

J12 ne révèle pas tout.

Elle crée :

- comparaison ;
- comportement inhabituel ;
- connaissances partielles ;
- images à plusieurs sens ;
- obligations sociales ;
- début de collision.

## Fenêtre A — Préparation La Verrière

### Marie

- pivot de son monde ;
- messages logistiques ;
- choix de présence Player déjà influencé par J09 ;
- aucune discussion conjugale complète sur place.

### Pauline et Bastien

- présence légitime ;
- surface réelle ;
- éventuelle différence d’audience seulement si construite avant.

### Mathilde

- présence possible comme cousine ou absence crédible ;
- aucune obligation de la mettre dans le groupe.

### Sandra

- présence seulement si relation sociale et horaire crédibles ;
- sinon messages avant/après ou absence.

### Raphaëlle

- présence uniquement avec raison professionnelle, créative ou invitation établie ;
- pas de téléportation de route.

## Fenêtre B — Événement

### Communication

`SAME_VENUE_LOGISTICS` et `TRACE_DELIVERY` uniquement.

Messages possibles :

- position ;
- tâche ;
- image ;
- aparté discret ;
- départ ;
- lieu suivant.

Aucune scène audio, aucun dialogue oral jouable.

## Fenêtre C — L’Annexe

Nico devient :

- ami ;
- observateur social ;
- possible rival ;
- jamais propriétaire des femmes ;
- jamais objet de désir de Player.

Il peut remarquer :

- Marie dans son élément ;
- tension Mathilde / Player ;
- version Pauline ;
- absence ou comportement de Player.

Il ne comprend que ce qu’il voit.

## Fenêtre D — Après séparation

Messages textuels de :

- Marie ;
- route dominante ;
- personnage ayant remarqué quelque chose ;
- personne qui demande une clarification.

Une seule conséquence principale doit être foreground.

## Branches d’après-coup

### Si J11 était adulte

- comportement social cohérent ;
- pas de nouvelle scène sexuelle ;
- gêne, proximité ou distance visible ;
- retour textuel obligatoire.

### Si J11 était un retrait

- absence ou surface ordinaire ;
- pas de récompense de substitution.

### Si aucune route extérieure

- J12 reste fort par Marie, Pauline, Bastien, Nico et le réseau.

## Slots visuels

```text
minimum: 4
maximum_recommended: 6
asset_design: deferred_to_user_comfyui
```

Fonctions :

1. Marie / La Verrière ;
2. groupe public avec participants réels ;
3. Pauline / surface ;
4. Nico / L’Annexe ;
5. route dominante ou absence signifiante ;
6. éventuelle image à double sens.

## Sortie J12

- au moins deux personnages possèdent des connaissances partielles différentes ;
- une image ou un comportement devient recontextualisable ;
- la version officielle ne suffit plus complètement ;
- une conséquence prioritaire est due en J13 ;
- acte IV ouvert.

# 11. Résumé

J09 remet Marie au centre par une visibilité sociale choisie.

J10 donne un coût réel à une seule continuité extérieure.

J11 transforme cette continuité en intention, limite, retrait ou premier passage adulte éligible.

J12 réunit les versions publiques et privées dans un événement dont les dialogues jouables restent entièrement textuels.

La convergence ne révèle pas tout.

Elle rend simplement impossible de continuer comme si chaque relation vivait dans un téléphone séparé.
