# Réseau Intime — 13 Bible des voix en messagerie et tests de distinction

## Statut

**Catégorie : Canon**

**Autorité : exécution des voix dans les dialogues textuels**

Ce document transforme les identités, psychologies et marqueurs déjà définis dans les canons complets en règles directement utilisables pour écrire des conversations de messagerie.

Il complète :

- les fichiers `*_CANON_FULL.md` ;
- `CHARACTER_VOICE_DISTINCTION_CANON.md` ;
- `TEXT_ONLY_MESSAGING_CANON.md` ;
- `08_REGLES_DES_SCENES_MODULAIRES.md` ;
- les plans détaillés J01–J21.

Hiérarchie :

```text
canon complet du personnage
→ propriété lexicale et distinction transversale
→ présent contrat opérationnel de messagerie
→ scène et dialogue précis
```

Le présent document ne remplace jamais l’identité du personnage.

Il définit comment cette identité devient reconnaissable dans :

- la longueur des messages ;
- leur ordre ;
- la ponctuation ;
- les silences ;
- les corrections ;
- les emojis ;
- la manière de plaisanter ;
- la manière d’esquiver ;
- la manière de devenir sérieuse ;
- la différence entre groupe et privé ;
- la transformation de la voix avec le désir, la peur ou le conflit.

# 1. Verdict de l’audit documentaire

## 1.1 Ce qui était déjà suffisamment documenté

Les sources actuelles définissent très bien :

- l’histoire personnelle ;
- le métier ;
- les objets ;
- la blessure ;
- la contradiction ;
- le moteur du désir ;
- les limites ;
- les grands marqueurs de voix ;
- certains exemples de répliques ;
- les risques de contamination lexicale entre personnages.

Les voix de Marie, Sandra, Mathilde, Pauline, Raphaëlle, Nico et Player ont donc déjà une base psychologique solide.

## 1.2 Ce qui manquait pour une production de saison complète

Avant ce document, les sources restaient insuffisamment prescriptives sur :

- la silhouette d’une rafale de messages ;
- la proportion entre fragments et phrases complètes ;
- la capitalisation ;
- la ponctuation ;
- la fréquence des emojis ;
- la manière de corriger un message ;
- la voix publique contre la voix privée ;
- la manière dont le style change sous stress ;
- la banalité nécessaire entre deux lignes fortes ;
- les erreurs naturelles ;
- les phrases que chaque personnage ne doit jamais utiliser ;
- les tests permettant de vérifier la reconnaissance sans le nom du locuteur ;
- les voix des partenaires et personnages secondaires.

Conclusion :

```text
identités vocales : fortes
contrat de production longue : incomplet avant le document 13
```

Après application du présent document, la documentation devient suffisante pour commencer la production des dialogues, sous réserve d’une validation scène par scène.

# 2. Principe directeur

> **Une voix reconnaissable ne vient pas d’un tic répété. Elle vient d’une manière stable de remarquer, d’éviter, de décider et de revenir.**

Une ligne ne devient pas Marie parce qu’elle contient du pain.

Une ligne ne devient pas Sandra parce qu’elle contient `haha`.

Une ligne ne devient pas Mathilde parce qu’elle contient `objection`.

Une ligne ne devient pas Pauline parce qu’elle contient `version`.

Une ligne ne devient pas Raphaëlle parce qu’elle contient `cadre`.

Une ligne ne devient pas Nico parce qu’elle commence par `frérot`.

Une ligne ne devient pas Player parce qu’elle est courte.

Les marqueurs lexicaux servent uniquement une logique plus profonde.

# 3. Les douze dimensions d’une voix en messagerie

Toute scène doit savoir comment le personnage se situe sur les dimensions suivantes.

## 3.1 Filtre d’attention

Ce que la personne remarque avant de répondre.

Exemples :

- Marie remarque l’acte qui manque ;
- Sandra remarque ce que l’image permet de croire ;
- Mathilde remarque l’effet qu’elle produit et tente de le corriger ;
- Pauline remarque l’audience et la contradiction ;
- Raphaëlle remarque la décision cachée dans la formulation ;
- Nico remarque le corps, la pièce et le désir social ;
- Player remarque un détail mais pas toujours l’action simple.

## 3.2 Intention immédiate

Pourquoi le message est envoyé maintenant.

- obtenir une action ;
- ralentir ;
- vérifier ;
- plaisanter ;
- garder le contrôle ;
- rouvrir après avoir fermé ;
- éviter ;
- réparer ;
- nommer un désir ;
- protéger une personne.

## 3.3 Taille du message

- fragment ;
- phrase courte ;
- phrase moyenne ;
- message exceptionnellement long.

## 3.4 Forme de la rafale

- un seul message ;
- deux temps ;
- correction ;
- fermeture suivie d’un retour ;
- réponse espacée ;
- cascade paniquée.

## 3.5 Syntaxe

- phrase complète ;
- sujet omis ;
- oralité ;
- impératif ;
- question ;
- juxtaposition ;
- auto-correction.

## 3.6 Capitalisation et ponctuation

- majuscules normales ;
- minuscules privées ;
- point final ;
- absence de point ;
- points de suspension ;
- parenthèses ;
- point d’interrogation ;
- exclamation.

## 3.7 Tempo

- réponse immédiate ;
- délai de travail ;
- hésitation visible ;
- message tardif ;
- silence protecteur ;
- retour après fermeture.

## 3.8 Humour

- pratique ;
- mauvaise foi ;
- sec ;
- social ;
- professionnel ;
- auto-dépréciation ;
- détour défensif.

## 3.9 Défense

- mouvement ;
- minimisation ;
- correction ;
- compartiment ;
- précision ;
- provocation ;
- blague ;
- silence.

## 3.10 Erreur et réparation

- faute ou mot manquant ;
- correction immédiate ;
- deuxième message qui reformule ;
- aveu d’une mauvaise réponse ;
- refus de corriger ;
- message supprimé ou retiré si le runtime le permet un jour.

## 3.11 Différence public / privé

La personne ne doit pas écrire exactement de la même manière :

- dans un groupe ;
- avec Player ;
- avec son partenaire ;
- au travail ;
- après une scène intime.

## 3.12 Transformation sous intensité

Le désir ou la colère ne remplace pas la voix.

Il retire certains filtres et en renforce d’autres.

# 4. Règles globales de naturel

## 4.1 Interdire la symétrie artificielle

Éviter :

```text
A fait une phrase brillante
B répond par une phrase brillante de même longueur
A clôt par une troisième phrase brillante
```

Préférer :

- une réponse partielle ;
- un rire ;
- une question ignorée ;
- un sujet pratique qui revient ;
- une phrase plus faible ;
- un délai ;
- une correction ;
- une maladresse.

## 4.2 Ne pas répondre à chaque élément

Un personnage humain peut répondre au détail le plus facile et laisser le reste sans réponse.

Le silence sur une partie du message peut devenir signifiant.

## 4.3 Autoriser la banalité

Toute scène forte doit contenir des messages qui ne cherchent pas à devenir des citations.

Exemples de fonctions banales :

- confirmer une heure ;
- demander une adresse ;
- dire qu’on arrive ;
- parler du pain ;
- signaler une batterie faible ;
- se plaindre d’un transport ;
- dire `ok` ;
- répondre trop tard.

## 4.4 Limiter les aphorismes

Une formulation très écrite peut être conservée si :

- elle est rare ;
- le personnage l’a gagnée ;
- elle est entourée de langage ordinaire ;
- elle ne ressemble pas à la voix de l’auteur parlant à travers tout le monde.

## 4.5 Limiter les punchlines

Règle recommandée :

```text
une ligne réellement brillante par beat suffit souvent
```

Les autres messages doivent permettre à cette ligne d’exister.

## 4.6 Préserver les répétitions humaines

Une personne peut répéter :

- `d’accord` ;
- `je sais` ;
- `bon` ;
- `attends` ;
- une question ;
- une phrase légèrement reformulée.

La répétition n’est pas automatiquement mauvaise si elle révèle l’état.

## 4.7 Pas de diagnostic permanent

Les personnages ne parlent pas tous comme des thérapeutes ou des narrateurs omniscients.

Ils savent parfois ce qu’ils ressentent.

Ils savent rarement tout ce que la scène signifie.

## 4.8 Pas de métaphore professionnelle automatique

Le métier colore l’attention et quelques mots.

Il ne transforme pas chaque conversation en :

- dossier juridique ;
- audit bancaire ;
- interface UX ;
- ticket informatique ;
- service de restaurant ;
- planification de projet.

## 4.9 Texte uniquement

Aucune voix ne peut être différenciée par :

- timbre audio ;
- accent joué ;
- intonation doublée ;
- note vocale ;
- appel.

La distinction doit exister uniquement dans l’écriture, le rythme et le contexte.

# 5. Conventions communes de français en messagerie

## 5.1 Orthographe

Les personnages restent globalement lisibles.

Les fautes ne doivent pas être ajoutées artificiellement à chaque message pour faire « vrai ».

Une faute ou omission est utile lorsqu’elle révèle :

- vitesse ;
- fatigue ;
- émotion ;
- changement d’avis ;
- auto-correction.

## 5.2 Points de suspension

Ils ne sont pas le langage générique de l’hésitation.

Utilisation :

- Sandra : possible mais rare et motivée ;
- Mathilde : préfère souvent la correction en message séparé ;
- Pauline : préfère la coupure nette ;
- Raphaëlle : préfère le silence ou la phrase précise ;
- Marie : les utilise rarement ;
- Nico : presque jamais ;
- Player : peut les éviter en envoyant une seconde phrase.

## 5.3 Majuscules

Les majuscules intégrales ne servent pas à remplacer l’émotion.

Elles sont réservées à :

- un mot exagéré ;
- une réaction comique ;
- un contexte de groupe ;
- une urgence concrète.

## 5.4 Emojis

Un emoji n’est pas nécessaire à chaque scène.

Il peut :

- alléger ;
- contredire ;
- masquer ;
- confirmer une proximité ;
- rendre un message plus public ou plus privé.

Il ne doit pas répéter exactement le sens de la phrase.

## 5.5 Longs messages

Un message long doit être exceptionnel et motivé.

Il signale :

- une décision préparée ;
- une clarification ;
- une peur contenue ;
- une limite ;
- un événement qui ne peut plus tenir en trois fragments.

Tous les personnages ne deviennent pas soudainement essayistes lors d’un conflit.

# 6. Marie — voix opérationnelle

## 6.1 Source mentale

```text
Qu’est-ce qu’on fait réellement ?
Est-ce que tu viens avec moi dans la vie qu’on a ?
```

Marie pense en :

- actions ;
- horaires ;
- repas ;
- déplacement ;
- personne à rejoindre ;
- chose à préparer ;
- effort visible.

## 6.2 Silhouette de messages

Rythme courant :

```text
1 à 4 messages
phrase courte ou moyenne
réponse rapide lorsqu’elle est chaleureuse
```

Elle peut lancer plusieurs petits messages lorsqu’elle organise quelque chose.

Elle n’est pas une mitraillette permanente.

## 6.3 Syntaxe et ponctuation

- capitalisation normale ;
- phrases complètes ou orales mais lisibles ;
- questions directes ;
- impératifs pratiques ;
- ponctuation normale ;
- exclamation possible dans l’énergie, pas dans chaque message ;
- points de suspension rares.

## 6.4 Humour

Marie utilise :

- exagération domestique ;
- absurdité pratique ;
- contradiction entre un petit problème et son traitement dramatique ;
- taquinerie de couple ;
- impatience chaleureuse.

Elle ne construit pas une blague en trois niveaux intellectuels.

## 6.5 Défense

Quand elle est blessée :

```text
elle raccourcit
→ retire la décoration
→ demande un acte
→ agit sans attendre
```

Elle ne devient pas soudainement froide et administrative.

## 6.6 Groupe / privé

### Groupe

- entraîne ;
- propose ;
- rappelle une heure ;
- rend l’activité attirante ;
- utilise l’humour pour faire bouger.

### Privé avec Player

- plus de détails domestiques ;
- irritation tendre ;
- demande de présence ;
- capacité à devenir sérieuse très vite.

## 6.7 Emojis

Début de saison :

```text
🙂 😅 😂 ❤️
```

Avec parcimonie.

Sous tension, les emojis diminuent avant que le texte ne devienne plus dur.

## 6.8 Voix sous intensité

### Colère

Plus courte, plus concrète.

```text
Je ne te demande pas une explication.
Je te demande si tu viens.
```

### Désir

Marie reste vivante et active.

Elle ne devient pas une femme fatale générique.

### Vulnérabilité

Elle peut dire une phrase forte, puis revenir à une action ou une heure.

## 6.9 Phrases étrangères à Marie

Éviter comme marqueur récurrent :

- `le cadre` ;
- `la version officielle` ;
- `je classe le dossier` ;
- `je vais laisser cette phrase respirer` ;
- métaphores de preuve ou d’audience ;
- diagnostic thérapeutique complet.

## 6.10 Test d’identité

Une ligne Marie doit souvent permettre de répondre :

```text
quelle vie concrète essaie-t-elle de faire exister ?
```

Si la réponse est `aucune`, la ligne risque d’être générique.

# 7. Sandra — voix opérationnelle

## 7.1 Source mentale

```text
Qu’est-ce que je viens réellement de montrer ?
Qu’est-ce que tu vas en faire ?
Est-ce que tu as vu la personne ou seulement la trace ?
```

## 7.2 Silhouette de messages

Rythme courant :

```text
1 à 3 messages
courts à moyens
peu de rafales
```

Quand elle est embarrassée :

```text
un premier message raisonnable
→ un petit ajout
→ éventuellement un troisième qui révèle davantage
```

Elle ne doit pas fragmenter aussi vite et systématiquement que Mathilde.

## 7.3 Syntaxe et ponctuation

- capitalisation normale ;
- ponctuation douce mais nette ;
- phrases souvent complètes ;
- conditionnels et minimisations ;
- une ellipse possible lorsqu’elle n’arrive réellement pas à finir ;
- pas de lyrisme permanent ;
- pas de paragraphes poétiques.

## 7.4 Le `haha`

Un `haha` peut signifier :

- embarras ;
- oui protégé ;
- porte entrouverte ;
- besoin de rendre la phrase moins grave.

Règle :

```text
0 ou 1 haha dans une scène ordinaire
```

Le répéter détruit le marqueur.

## 7.5 Humour

- minimisation ;
- raisonnabilité prétendue ;
- absurdité SentryCore ;
- ironie douce contre elle-même ;
- petite pointe précise en privé.

Elle n’est pas une femme faite de lumière, de silence et de phrases littéraires.

## 7.6 Défense

```text
minimise
→ déplace vers un objet ou une image
→ ferme doucement
→ peut revenir plus tard
```

Le silence de Sandra doit pouvoir durer.

Une absence de message peut être plus Sandra qu’une longue explication.

## 7.7 Groupe / privé

### Groupe

- écrit moins ;
- répond plutôt qu’elle ne lance ;
- une remarque fine ;
- aucune performance sociale continue ;
- contrôle accru si une image circule.

### Privé avec Player

- plus vive ;
- capable de potins et d’ironie ;
- petites formules affectueuses rares ;
- peut choisir une image plus audacieuse ;
- demande implicite ou explicite sur la conservation.

## 7.8 Emojis

Rares.

Un emoji léger peut apparaître dans la chaleur ou pour réduire la gravité.

Sandra ne doit pas concurrencer Mathilde en volume d’emojis.

## 7.9 Voix sous intensité

### Peur

Délai, message plus contrôlé, suppression ou retrait d’accès.

### Désir

La phrase devient plus précise, pas nécessairement plus longue.

### Adulte

Les règles de voir, garder et transférer deviennent directes.

Elle ne devient ni lyrique ni brutalement crue sans transition.

## 7.10 Phrases étrangères à Sandra

Éviter :

- organisation de plusieurs audiences à la Pauline ;
- longues définitions de cadre à la Raphaëlle ;
- rafales de corrections à la Mathilde ;
- injonctions domestiques à la Marie ;
- poésie générique sur le flou, la lumière ou les traces ;
- surnoms affectueux fréquents.

## 7.11 Test d’identité

Une ligne Sandra doit souvent permettre de répondre :

```text
quelle part d’elle-même protège-t-elle encore après avoir presque montré quelque chose ?
```

# 8. Mathilde — voix opérationnelle

## 8.1 Source mentale

```text
Qu’est-ce que j’ai provoqué ?
Comment je corrige sans vraiment fermer ?
Est-ce que tu m’as seulement regardée ou aussi comprise ?
```

## 8.2 Silhouette de messages

Rythme courant :

```text
2 à 6 messages
beaucoup de fragments courts
correction en message séparé
```

Forme typique :

```text
affirmation
→ correction
→ mauvaise foi
→ fermeture tentée
→ petit retour qui rouvre
```

## 8.3 Syntaxe et ponctuation

- oralité rapide ;
- phrases courtes ;
- fragments autonomes ;
- question suivie d’une correction ;
- point final possible pour accent comique ;
- majuscules ponctuelles dans l’exagération ;
- point de suspension moins utile que la correction séparée.

## 8.4 Erreurs et réparations

Mathilde est la plus compatible avec :

- message parti trop vite ;
- mot corrigé ;
- précision inutile ;
- `je précise` ;
- deuxième message qui rend le premier plus suspect.

Les fautes restent occasionnelles et lisibles.

## 8.5 Humour

- mauvaise foi ;
- auto-accusation comique ;
- catastrophe mineure ;
- correction d’image ;
- vocabulaire juridique dosé ;
- commentaire familial.

## 8.6 Dosage juridique

```text
scène ordinaire : 0 ou 1 formule
scène professionnelle : 2 ou 3 possibles
complicité : 1 ou 2 maximum
scène adulte : diminution forte
```

Le désir doit finir par être nommé sans se cacher derrière `objection`.

## 8.7 Groupe / privé

### Groupe

- plus énergique ;
- réagit vite ;
- peut s’exposer elle-même par une blague ;
- participe au mouvement social ;
- ne révèle pas automatiquement la tension Player.

### Privé avec Player

- correction plus visible ;
- embarras ;
- contrôle d’image ;
- plaisir d’être remarquée ;
- poids de Marie.

## 8.8 Emojis

Base :

```text
😅 🙃 🙄 😂
```

Trouble ou panique :

```text
🫠 👀
```

Les emojis adultes apparaissent seulement après reconnaissance de la tension.

## 8.9 Voix sous intensité

### Peur

Plus de corrections, puis retrait brusque.

### Désir

La vitesse reste, mais les euphémismes diminuent.

### Adulte

Le corps et l’intention peuvent être nommés directement.

La voix reste rapide, parfois suivie d’un message de panique.

## 8.10 Phrases étrangères à Mathilde

Éviter :

- calme espacé de Raphaëlle ;
- compartiment maîtrisé de Pauline ;
- retrait doux de Sandra ;
- demande d’action conjugale de Marie ;
- longue phrase parfaitement construite sans correction dans une scène de panique ;
- vocabulaire juridique dans chaque échange.

## 8.11 Test d’identité

Une ligne Mathilde doit souvent permettre de voir :

```text
la différence entre ce qu’elle voulait contrôler et ce qu’elle vient de révéler malgré elle
```

# 9. Pauline — voix opérationnelle

## 9.1 Source mentale

```text
Qui reçoit quelle version ?
Qu’est-ce que cette réponse change officiellement ?
Peut-on conserver le désir sans ouvrir la contradiction ?
```

## 9.2 Silhouette de messages

Rythme courant :

```text
1 à 4 messages courts
contrôle net
retour possible après clôture
```

Forme typique :

```text
fait court
→ correction sèche
→ défi apparemment léger
→ demande précise
→ fermeture
→ retour révélateur éventuel
```

## 9.3 Syntaxe et ponctuation

### Privé

- minuscules fréquentes possibles ;
- messages courts ;
- peu d’exclamations ;
- coupures nettes ;
- impératifs ;
- pas de points de suspension décoratifs.

### Public / organisation

- phrases plus propres ;
- horaires et informations ;
- politesse ;
- formulation qui protège l’audience.

## 9.4 Humour

- sec ;
- timing ;
- faux sérieux ;
- moralité ou auréole très dosée ;
- contrôle qui craque ;
- constat plutôt que sketch.

Elle n’est pas constamment plus intelligente que tout le monde.

## 9.5 Défense

```text
ferme la ligne
→ restaure la surface
→ revient en privé
→ demande à Player de dire ce qu’elle ne veut pas encore dire
```

## 9.6 Groupe / privé

### Groupe

- fiable ;
- efficace ;
- discrète sur les conflits ;
- peut choisir la bonne image publique ;
- ne teste pas sexuellement Player devant tout le monde.

### Privé avec Player

- plus rapide ;
- plus sèche ;
- double adresse ;
- demande de précision ;
- retour après une fermeture ;
- danger de compartiment.

## 9.7 Emojis

Contrôlés :

```text
🙂 🙃 😅 😇 👀 🫠
```

Jamais une cascade.

Un emoji peut servir à rendre une phrase publiquement innocente.

## 9.8 Voix sous intensité

### Peur

Plus organisée, puis une fissure : second message, question trop précise, alibi inutile.

### Désir

Les plaisanteries de morale diminuent.

### Adulte

Impératifs courts, corps nommé, secret reconnu.

Elle ne devient pas lyrique.

## 9.9 Phrases étrangères à Pauline

Éviter :

- droit comique à la Mathilde ;
- longue clarification éthique à la Raphaëlle ;
- douceur hésitante de Sandra ;
- énergie domestique de Marie ;
- `version` dans chaque scène ;
- sainteté, confession ou chorale comme gimmick permanent.

## 9.10 Test d’identité

Une ligne Pauline doit souvent permettre de répondre :

```text
quelle contradiction essaie-t-elle de garder dans deux compartiments distincts ?
```

# 10. Raphaëlle — voix opérationnelle

## 10.1 Source mentale

```text
Quelle décision est cachée dans cette formulation ?
Qu’est-ce que je choisis exactement ?
Que reste-t-il après la version ?
```

## 10.2 Silhouette de messages

Rythme courant :

```text
1 à 3 messages
espacement visible
phrases moyennes et complètes
```

Elle pose souvent :

```text
une observation
→ une question simple
→ laisse répondre
→ formule une invitation ou une limite
```

## 10.3 Syntaxe et ponctuation

- capitalisation normale ;
- ponctuation complète ;
- vocabulaire précis ;
- une question à la fois ;
- peu d’exclamations ;
- peu de slang ;
- pas de torrent de fragments ;
- peut utiliser une phrase très courte lorsqu’une limite tombe.

## 10.4 Humour

- détail exact ;
- absurdité de processus ;
- phrase sèche très calme ;
- objet de travail ;
- rare perte de composure sur les pandas ou chats.

Elle ne doit pas transformer chaque conversation en contrat.

## 10.5 Défense

```text
nomme la question
→ nomme la limite
→ laisse un espace
→ refuse de décider à la place de Player
```

Son contrôle peut devenir un alibi, mais sa voix reste précise.

## 10.6 Groupe / privé

### Groupe

- intervient peu ;
- information utile ;
- précision ;
- ne surjoue pas le rôle de sage ;
- présence uniquement si le contexte l’autorise.

### Privé avec Player

- espace ;
- question réelle ;
- accès au processus ;
- version choisie ;
- responsabilité du lendemain.

## 10.7 Emojis

Très rares dans sa voix ordinaire.

Exception comique assumée pour chats ou pandas.

Cette rupture doit rester rare afin d’être reconnaissable.

## 10.8 Voix sous intensité

### Peur

Plus lente, plus exacte, parfois trop structurée.

### Désir

La précision devient permission ou instruction, sans clinique froide.

### Vulnérabilité

Une phrase moins élégante, explicitement reconnue comme telle.

### Adulte

Impératifs et questions clairs, puis après-rôle.

## 10.9 Phrases étrangères à Raphaëlle

Éviter :

- vocabulaire juridique récurrent ;
- gestion d’audiences multiples à la Pauline ;
- rafales paniquées à la Mathilde ;
- poésie fragile à la Sandra ;
- `cadre`, `version`, `permission` dans chaque échange ;
- perfection grammaticale continue sans banalité ni fatigue.

## 10.10 Test d’identité

Une ligne Raphaëlle doit souvent permettre de répondre :

```text
quelle décision essaie-t-elle de rendre visible sans prétendre résoudre toute la vie de Player ?
```

# 11. Nico — voix opérationnelle

## 11.1 Source mentale

```text
Qu’est-ce qui se passe réellement dans la pièce ?
Qui désire qui ?
Qu’est-ce que tu essaies de rendre plus propre que ça ne l’est ?
```

## 11.2 Silhouette de messages

Rythme courant :

```text
1 à 4 messages courts
oralité directe
peu de paragraphes
```

Il peut envoyer un message plus long seulement lorsque :

- la salle est vide ;
- la blague est tombée ;
- l’amitié est réellement menacée ;
- il pose une limite.

## 11.3 Syntaxe et ponctuation

- contractions françaises naturelles ;
- `t’as`, `j’suis` possibles mais non obligatoires ;
- phrases directes ;
- ponctuation légère ;
- questions franches ;
- peu de points de suspension ;
- pas de jargon administratif.

## 11.4 Humour

- nourriture ;
- chaise ;
- service ;
- pièce ;
- heure ;
- corps ;
- jalousie visible ;
- exagération de pote.

`Frérot` et `mon vieux` restent occasionnels.

## 11.5 Défense

```text
plaisante
→ nomme le désir
→ cache son propre intérêt
→ peut devenir sérieux après la fermeture de la salle
```

## 11.6 Groupe / privé

### Groupe

- anime ;
- décrit la pièce ;
- distribue les petites provocations ;
- ne transforme pas les femmes en inventaire ;
- sait quand arrêter.

### Privé avec Player

- plus direct ;
- peut nommer jalousie et désir ;
- demande ce qu’il couvre ;
- révèle progressivement son intérêt ;
- peut refuser.

## 11.7 Emojis

Rares et simples.

Un rire ou une réaction peut apparaître, mais sa voix repose sur l’oralité, pas sur une collection d’emojis.

## 11.8 Voix sous intensité

### Désir

Plus direct, moins moraliste.

### Jalousie

Il peut provoquer Player tout en révélant son propre intérêt.

### Blessure

Les blagues s’arrêtent.

### Limite

Phrase courte et non négociable.

## 11.9 Phrases étrangères à Nico

Éviter :

- analyse thérapeutique ;
- gestion abstraite de versions ;
- précision contractuelle ;
- métaphores littéraires ;
- vulgarité permanente ;
- `frérot` à chaque ouverture ;
- catalogue des corps dans chaque conversation.

## 11.10 Test d’identité

Une ligne Nico doit souvent permettre de répondre :

```text
quel désir ou rapport social rend-il concret, et quel intérêt personnel essaie-t-il encore de présenter comme conseil ?
```

# 12. Player — voix opérationnelle

## 12.1 Source mentale

```text
J’ai vu quelque chose.
Est-ce que je le dis ?
Est-ce que j’agis maintenant ou est-ce que je gagne encore du temps ?
```

## 12.2 Silhouette de messages

Rythme courant :

```text
1 à 3 messages
très souvent courts
```

Forme typique :

```text
première réponse insuffisante
→ auto-évaluation ou blague
→ seconde réponse plus honnête
```

## 12.3 Syntaxe et ponctuation

- minuscules fréquentes ;
- ponctuation légère ;
- mots seuls possibles ;
- réponses incomplètes ;
- phrase longue rare ;
- pas de prose parfaite ;
- peut écrire correctement sans que chaque message soit poli comme un roman.

## 12.4 Humour

- sec ;
- auto-dépréciation ;
- légèrement raté ;
- retardataire ;
- protection après une phrase trop vraie.

## 12.5 Défense

```text
réponse courte
→ blague
→ délai
→ phrase honnête éventuelle
```

Il peut remarquer exactement et agir trop tard.

## 12.6 Différence selon l’interlocuteur

### Marie

Plus domestique, plus familier, parfois trop plat.

### Sandra

Plus précis, mais risque de surinterpréter ou de se valoriser comme celui qui voit.

### Mathilde

Plus joueur et embarrassé ; doit préserver Marie dans le sous-texte.

### Pauline

Plus rapide et sec ; danger de se croire plus malin qu’il ne l’est.

### Raphaëlle

Plus obligé de formuler ; ne doit pas copier sa précision.

### Nico

Plus oral ; peut nier, provoquer ou être directement honnête.

## 12.7 Emojis

Rares.

Il peut reprendre un emoji de l’autre personnage exceptionnellement, mais ne doit pas adopter leur style complet.

## 12.8 Voix sous intensité

### Honte

`je recommence` ou une phrase courte plus vraie.

### Désir

Maladroit avant de devenir direct.

### Limite

`non`, `pas comme ça`, puis explication minimale si nécessaire.

### Retour vers Marie

Action et heure avant grand discours.

## 12.9 Phrases étrangères à Player

Éviter :

- réponse parfaite à chaque dilemme ;
- imitation immédiate du meilleur idiome de l’interlocuteur ;
- longue culpabilité noble ;
- séduction professionnelle ;
- analyse complète de sa propre psychologie ;
- métaphore d’échecs à répétition.

## 12.10 Test d’identité

Une ligne Player doit souvent permettre de voir :

```text
l’écart entre ce qu’il a remarqué et ce qu’il est prêt à faire maintenant
```

## 13.9 Lucie

# 17. Contrat d’écriture d’une scène dialoguée

Avant d’écrire, renseigner :

```text
principal_speaker
voice_state
public_or_private
message_burst_shape
humor_mode
defense_mode
punctuation_mode
emoji_budget
forbidden_leakage
voice_change_at_choice
voice_change_at_exit
```

## Exemple conceptuel

```text
principal_speaker: Sandra
voice_state: embarrassed_warm
public_or_private: private
message_burst_shape: 2_messages_then_pause
humor_mode: soft_minimization
defense_mode: close_gently
punctuation_mode: normal_one_ellipsis_max
emoji_budget: 0_or_1
forbidden_leakage: no_Raphaelle_frame_language
```

Ces champs sont des outils d’écriture.

Ils ne doivent pas tous devenir des données runtime.

# 18. Tests obligatoires de distinction

## 18.1 Test sans nom

Retirer les noms de douze lignes :

- deux ordinaires ;
- deux humoristiques ;
- deux vulnérables ;
- deux conflictuelles ;
- deux intimes ;
- deux de groupe.

Critère recommandé :

```text
au moins 9 lignes sur 12 attribuables correctement
par deux lecteurs connaissant la Bible
```

Si les lignes ne sont identifiables que grâce aux objets ou noms propres, la voix reste trop faible.

## 18.2 Test de substitution

Remplacer le nom du locuteur par un autre personnage.

La ligne doit produire au moins une friction claire de :

- rythme ;
- intention ;
- vocabulaire ;
- défense ;
- ponctuation.

Si cinq personnages peuvent la dire inchangée, réécrire.

## 18.3 Test de silhouette

Masquer les mots et regarder seulement :

- nombre de messages ;
- longueurs ;
- ponctuation ;
- emoji ;
- délai.

La silhouette doit déjà différencier certains personnages.

## 18.4 Test banal

Chaque personnage doit pouvoir écrire cinq messages ordinaires reconnaissables sans :

- flirt ;
- conflit ;
- métier mis en avant ;
- catchphrase.

## 18.5 Test sous stress

La voix doit rester reconnaissable lorsque :

- le personnage est en colère ;
- il désire ;
- il ment ;
- il a peur ;
- il pose une limite.

## 18.6 Test public / privé

Le même fait écrit dans :

- un groupe ;
- un fil privé ;
- un contexte de travail ;

doit changer de forme sans changer d’identité.

## 18.7 Test anti-punchline

Sur dix messages consécutifs d’un personnage :

- tous ne doivent pas être citables ;
- au moins plusieurs doivent seulement accomplir le rythme ou l’action.

## 18.8 Test de contamination professionnelle

Compter les métaphores de métier.

Une scène ordinaire échoue si le personnage ne peut plus parler sans son domaine professionnel.

## 18.9 Test Player

Player doit envoyer de vrais messages.

La scène échoue s’il devient uniquement :

- un bouton abstrait ;
- un silence entre deux monologues ;
- une réponse parfaite.

## 18.10 Test text-only

La scène échoue si sa distinction dépend d’un futur :

- accent audio ;
- timbre ;
- vocal ;
- appel ;
- intonation doublée.

# 20. Processus de production recommandé

## Étape 1 — Carte de voix

Avant chaque scène :

- état émotionnel ;
- public ou privé ;
- rafale ;
- humour ;
- défense ;
- phrases interdites.

## Étape 2 — Premier jet sans catchphrases

Écrire d’abord la logique humaine sans ajouter immédiatement :

- `haha` ;
- `objection` ;
- `version` ;
- `cadre` ;
- `frérot` ;
- nourriture Marie.

Ajouter ensuite uniquement le marqueur utile.

## Étape 3 — Passe de banalité

Ajouter :

- confirmation ;
- retard ;
- détail pratique ;
- petit message faible ;
- silence.

## Étape 4 — Passe de distinction

Appliquer les tests :

- sans nom ;
- substitution ;
- silhouette ;
- public / privé ;
- stress.

## Étape 5 — Passe anti-AI

Supprimer :

- symétrie ;
- aphorisme de trop ;
- réponse à chaque point ;
- personnage trop conscient du thème ;
- conclusion trop propre ;
- flirt générique.

## Étape 6 — Passe runtime

Vérifier :

- longueur de bulle ;
- rythme de saisie ;
- choix réels ;
- conditions ;
- pas d’audio ;
- arrêt à la co-présence.

# 21. Critères d’acceptation avant intégration

- [ ] la scène utilise au moins deux ancrages du canon complet ;
- [ ] la voix est reconnaissable sans catchphrase ;
- [ ] la rafale correspond au personnage ;
- [ ] le volume d’emojis est cohérent ;
- [ ] la ponctuation ne ressemble pas à celle de tout le casting ;
- [ ] une partie du dialogue est ordinaire ;
- [ ] toutes les lignes ne sont pas brillantes ;
- [ ] le personnage n’explique pas tout le sous-texte ;
- [ ] le groupe et le privé diffèrent ;
- [ ] l’intensité conserve la voix ;
- [ ] Player écrit de vrais messages ;
- [ ] aucune contamination professionnelle excessive ;
- [ ] aucune dépendance audio ;
- [ ] le test sans nom est satisfaisant ;
- [ ] le test de substitution révèle une friction claire.

# 22. Résumé canonique

Marie fait bouger la vie commune.

Sandra protège ce qu’elle vient presque de confier.

Mathilde corrige trop vite l’effet qu’elle produit et le rouvre malgré elle.

Pauline contrôle qui reçoit quelle version et revient lorsque la fermeture ne tient plus.

Raphaëlle rend la décision visible, laisse un espace et nomme une limite.

Nico rend le désir social et concret, puis révèle progressivement ce qu’il veut lui-même.

Player voit juste, répond court, plaisante, diffère et parfois finit par agir.

Les partenaires et secondaires possèdent une voix plus simple mais réelle.

> **Le lecteur doit pouvoir reconnaître la personne avant même de lire son nom, sans avoir besoin d’un tic, d’un métier ou d’un emoji répété.**
