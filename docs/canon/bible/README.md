# 6. Voix des personnages

Le document :

`13_BIBLE_VOIX_MESSAGERIE_ET_TESTS_DISTINCTION.md`

rend opérationnelles les voix déjà définies dans les canons complets.

Il fixe :

- filtre d’attention ;
- longueur ;
- rafales ;
- ponctuation ;
- capitalisation ;
- emojis ;
- erreurs et corrections ;
- humour ;
- défense ;
- public / privé ;
- transformation sous stress ou désir ;
- phrases étrangères à chaque personnage ;
- tests sans nom, substitution et silhouette.

Règle :

```text
une voix doit rester reconnaissable
sans catchphrase, métier ou nom propre
```

# 7. Règles de production

- un pivot principal par journée ;
- Marie reste centrale ;
- une route extérieure dominante maximum par défaut ;
- conséquences avant nouvelles opportunités lorsqu’elles sont dues ;
- trois slots visuels minimum par journée complète ;
- refus sans récompense de substitution ;
- arrêt du chat à la co-présence ;
- aucune route choisie dans un menu ;
- voix testée avant intégration ;
- Nico reste hétérosexuel sans route avec Player.
