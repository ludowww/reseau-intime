# 4. Tests obligatoires

Une journée doit passer :

- North Star ;
- route invisible ;
- fonction principale ;
- agence ;
- conséquence attribuable ;
- voix ;
- text-only ;
- connaissance limitée ;
- fallback ;
- audience visuelle ;
- trois contenus principalement féminins ;
- consentement actuel ;
- retrait possible ;
- après-coup obligatoire après une progression adulte.
