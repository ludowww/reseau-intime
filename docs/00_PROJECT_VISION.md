# 00 — Vision projet

## Titre officiel

**Réseau Intime**

## Genre

Thriller intime érotique en interface smartphone.

Le jeu mélange :

- fiction interactive ;
- messagerie instantanée ;
- roman visuel adulte ;
- enquête relationnelle ;
- galerie narrative ;
- simulation de notifications et de réseaux sociaux.

## Pitch

Ludo est en couple avec Marie. Leur relation est stable, tendre, mais la routine a affaibli le désir. Autour d’eux, plusieurs personnes réveillent des failles différentes : Mathilde, la cousine presque sœur de Marie ; Sandra, la meilleure amie ambiguë de Ludo ; Raphaëlle, la collègue attentionnée ; Pauline, la meilleure amie de Marie qui cache un jeu plus trouble ; Nico, l’ami de Ludo qui fantasme sur Marie.

Le joueur reçoit des messages, des notifications, des photos, des vidéos, des stories et des preuves. Chaque choix peut rapprocher Ludo d’un personnage, dégrader une autre relation, créer un secret ou déclencher une révélation.

## Promesse joueur

Le joueur ne rejoue pas seulement pour débloquer une autre scène ou un autre contenu visuel. Il rejoue pour découvrir une autre dynamique relationnelle.

Selon les choix :

- Marie peut être blessée, réparée, libertine, dominante, distante ou attirée par Nico ;
- Mathilde peut être tentation contenue, liaison interdite, partenaire de trio, limite rouge ou catastrophe familiale ;
- Sandra peut être confidente, obsession, amante impossible, femme qui choisit enfin ou fantôme émotionnel ;
- Raphaëlle peut être refuge, vraie alternative, partenaire poly possible ou femme qui refuse le mensonge ;
- Pauline peut être tentatrice, manipulatrice, initiatrice libertine, complice ou joueuse renversée ;
- Nico peut être rival, miroir, confident de Marie, menace NTR, partenaire consenti ou complice trouble.

## Thème central

> Le désir n’est pas seulement ce que Ludo ressent. C’est ce qu’il choisit d’en faire : le cacher, l’avouer, le partager, le contrôler, le subir ou le perdre.

## Règle d’unité

Toutes les routes doivent raconter une même histoire de base :

> Un couple en routine se retrouve entouré de désirs qui se réveillent. Les mêmes événements se produisent, mais leur sens change selon les choix du joueur.

## Structure validée

Le jeu repose sur la structure suivante :

```text
Tronc commun + route dominante + route secondaire + menace + mode relationnel
```

Cette structure est validée et sert de base pour toute écriture ou implémentation.

## Piliers de design

### 1. Le smartphone comme monde du jeu

Le joueur ne lit pas simplement des dialogues. Il manipule un téléphone : notifications, conversations, stories, galerie, archives, preuves, messages supprimés.

### 2. Les contenus visuels sont narratifs

Une photo ou une vidéo n’est jamais une simple récompense. Elle peut devenir :

- une preuve ;
- une menace ;
- une tentation ;
- un secret ;
- un outil de pouvoir ;
- un déclencheur de jalousie ;
- une trace impossible à effacer.

### 3. Les personnages sont liés

Un choix avec Sandra peut modifier Marie. Un secret avec Mathilde peut donner du pouvoir à Pauline. Une négligence envers Marie peut renforcer Nico. Une relation avec Raphaëlle peut rendre Sandra jalouse.

### 4. Rejouabilité par dynamiques

Le jeu doit encourager plusieurs parties, non par accumulation mécanique, mais parce que les personnages révèlent des facettes différentes selon les modes relationnels.

## Ce que le jeu n’est pas

- Ce n’est pas un simple jeu de conquête.
- Ce n’est pas une succession de scènes isolées.
- Ce n’est pas un simulateur libre infini.
- Ce n’est pas une galerie déconnectée de l’histoire.
- Ce n’est pas un jeu où toutes les combinaisons sont possibles sans conséquence.

## Ton

- intime ;
- adulte ;
- tendu ;
- sensuel ;
- parfois cruel ;
- psychologique ;
- jamais parodique.

## Règle de contenu

Tous les personnages impliqués dans les routes adultes sont majeurs. Les dynamiques sexuelles ou libertines doivent être traitées comme des choix fictionnels adultes et consentis. Les situations de manipulation peuvent exister comme tension narrative, mais les scènes intimes assumées doivent rester cadrées par le consentement fictionnel des personnages.
