# 09 — Guide d’écriture

## Objectif

Garantir une voix cohérente pour chaque personnage et éviter que les routes deviennent interchangeables.

## Style général

Le jeu se déroule par smartphone. L’écriture doit donc respecter :

- messages courts ;
- silences ;
- sous-entendus ;
- fautes légères possibles mais pas systématiques ;
- emojis rares et caractérisés ;
- notifications comme tension ;
- messages supprimés comme événement ;
- horaires et délais de réponse comme langage.

## Règle centrale

> Ce qui n’est pas dit doit être aussi important que ce qui est écrit.

## Voix de Player

Player varie selon les choix, mais ses réponses doivent rester crédibles :

- évitant ;
- tendre ;
- ironique ;
- troublé ;
- parfois lâche ;
- parfois sincère ;
- rarement frontal au début.

Le joueur doit pouvoir l’orienter, mais Player n’est pas un avatar vide.

## Voix de Marie

Marie connaît Player. Elle n’a pas besoin de grandes phrases pour sentir qu’il ment.

Style :

- naturel ;
- intime ;
- parfois sec quand elle doute ;
- blessé sans être naïf ;
- capable de reprendre le pouvoir.

Exemples :

```text
Tu es bizarre en ce moment.
Je ne te demande pas si tu as fait quelque chose. Je te demande si tu es encore vraiment là.
Je peux entendre tes désirs. Je ne veux plus découvrir tes secrets.
```

## Voix de Mathilde

Mathilde est joueuse, séduisante, consciente de son effet, mais déchirée par sa loyauté envers Marie.

Style :

- taquin ;
- proche ;
- provocant ;
- capable de reculer ;
- culpabilisé quand la limite devient réelle.

Exemples :

```text
Tu ne devrais pas me regarder comme ça.
Je disais ça pour rire. Enfin… je crois.
Dis-moi d’arrêter. Mais dis-le vraiment.
```

## Voix de Sandra

Sandra est discrète, sensible, ambiguë, difficile à saisir. Elle suggère plus qu’elle n’affirme.

Style :

- doux ;
- intime ;
- hésitant ;
- taquin par moments ;
- fuyant quand elle se sent exposée.

Exemples :

```text
Je ne sais pas pourquoi je t’écris à toi.
Tu vas encore dire que je te cherche.
Oublie. Je n’aurais pas dû.
Je ne veux pas te perdre, mais je ne sais pas si je peux te choisir.
```

## Voix de Raphaëlle

Raphaëlle est claire, douce, adulte. Elle ne joue pas longtemps avec le flou.

Style :

- attentionné ;
- direct quand il le faut ;
- respectueux ;
- ferme sur ses limites.

Exemples :

```text
Tu n’avais pas l’air bien aujourd’hui.
Je peux t’écouter, mais je ne veux pas devenir ton secret.
Tu dis que tu veux être aimé, mais tu cours surtout après celles qui te laissent douter.
```

## Voix de Pauline

Pauline provoque, teste, observe. Elle aime avoir un coup d’avance.

Style :

- double sens ;
- contrôle ;
- fausse innocence ;
- phrases courtes ;
- capacité à retourner une situation.

Exemples :

```text
Ne réagis pas.
Je voulais juste voir si tu allais regarder.
Dis-le clairement. J’aime bien quand les hommes arrêtent de se cacher derrière l’humour.
Tu pensais recevoir une photo. Tu m’as surtout donné une preuve.
```

## Voix de Nico

Nico est le miroir masculin. Il peut être lourd, sincère, provocant ou lucide.

Style :

- amical en surface ;
- compétitif ;
- parfois brutalement honnête ;
- phrases qui semblent anodines mais piquent.

Exemples :

```text
Marie était rayonnante hier.
Tu sais la chance que tu as, au moins ?
Tu me reproches de la regarder, mais toi aussi tu regardes ailleurs.
```

## Direction J2 — Mathilde dans la maison

Le J2 introduit Mathilde par une photo envoyée par Marie, puis par une micro-conversation directe avec Player.

Mathilde ne découvre pas la maison : elle y revient. Ce n’est pas sa première présence longue chez Marie/Player, ni sa première nuit possible sur le canapé. La familiarité doit sembler normale pour Marie.

Le trouble ne vient donc pas d’une situation exceptionnelle, mais du regard de Player : Mathilde est chez eux, Marie trouve ça normal, et Player commence à la voir autrement.

En J2, Mathilde doit être familière, taquine, légèrement insolente et vivante, sans devenir frontalement séductrice trop tôt.

## Écriture des choix

Chaque choix doit être lisible dans son intention.

Types de choix :

```text
éviter
rassurer
mentir
avouer
flirter
provoquer
contrôler
montrer
supprimer
conserver
```

Éviter les choix qui semblent identiques mais ont des effets très différents.

## Écriture du désir

Le désir doit monter progressivement.

Règles :

- pas d’explicite trop tôt ;
- chaque contenu intime doit avoir un contexte ;
- la tension vient souvent de l’attente ;
- les conséquences doivent compter ;
- les personnages ne doivent pas tous céder au même rythme.

## Écriture des routes adultes

Les routes adultes peuvent inclure tromperie, libertinage, polyamour, NTR, harem ou domination consentie.

Règles :

- les personnages sont majeurs ;
- les scènes intimes assumées doivent être fictionnellement consenties ;
- la manipulation peut être un enjeu narratif, mais elle doit avoir des conséquences ;
- les contenus explicites sont liés à une progression et à un risque ;
- ne pas réduire les personnages à leur contenu visuel.

## Écriture des scènes communes

Une même scène doit pouvoir changer de sens selon la route.

Exemple : soirée chez Pauline.

- Pauline : piège.
- Nico : menace sur Marie.
- Mathilde : jalousie et regard.
- Sandra : absence présente par messages.
- Raphaëlle : contrepoint le lendemain.
- Marie : lucidité croissante.

## Derniers messages

Chaque fin doit se conclure par un dernier message fort.

Ce message doit résumer :

- ce que Player a choisi ;
- ce qu’il a perdu ;
- ce que le personnage a compris ;
- la tonalité de la fin.

## Erreurs à éviter

- Faire parler tous les personnages avec la même voix.
- Transformer toutes les routes en simple conquête.
- Donner du contenu visuel sans conséquence.
- Oublier Marie dans les routes secondaires.
- Faire de Nico un simple méchant.
- Faire de Raphaëlle une simple fille gentille.
- Faire de Sandra une conquête facile.
- Faire de Mathilde une trahison sans poids familial.
- Faire de Pauline une manipulatrice sans vulnérabilité.
