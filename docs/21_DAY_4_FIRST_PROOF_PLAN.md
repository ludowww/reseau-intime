# 21 — Jour 4 : Lendemain et première vraie preuve

> V0.57 (`docs/V0_57_Route_Character_Rework_Blueprint.md`) guide la hiérarchie actuelle.
> Ce plan J4 reste utile, mais la “première preuve” doit être lue comme conséquence d’une hiérarchie route/personnage déjà orientée par Marie.

## Objectif

Cadrer le Jour 4 comme le moment où la soirée du Jour 3 cesse d’être seulement une tension et devient une trace.

Le Jour 4 ne doit pas encore résoudre les routes, mais il doit prouver au joueur que :

- les choix de priorité ont été vus ;
- les signaux passifs peuvent ressortir ;
- une photo, une capture, une story ou un message supprimé peut devenir une preuve ;
- Marie commence à relier des indices ;
- Pauline peut devenir dangereuse ;
- Sandra peut regretter ;
- Mathilde peut culpabiliser ;
- Nico peut prendre de la place auprès de Marie.

## Règle centrale

> Le Jour 4 transforme une soirée ambiguë en conséquence lisible.

## Structure recommandée

```text
1. Matin — Marie sent le décalage
2. Fin de matinée — preuve conditionnelle selon route nourrie
3. Midi — Raphaëlle comme contraste / lucidité extérieure
4. Après-midi — réseau social : Nico, Marie, Pauline ou groupe
5. Soir — pression de route probable
6. Debug / bilan invisible : route dominante probable, menace active
```

## Types de preuves possibles

```text
Pauline : capture ou phrase qui montre qu’elle a vu quelque chose
Sandra : message supprimé / regret / trace émotionnelle
Mathilde : photo domestique ou avertissement après la soirée
Nico : story où Marie est visible avec lui
Marie : contradiction entre ce que Player dit et ce qu’elle a vu
Fond d’écran : malaise ou soupçon si révélé / caché au Jour 3
```

## Règle anti-explosion

La première preuve ne doit pas provoquer immédiatement une rupture ou une fin.

Elle doit :

- créer une menace ;
- clarifier une route probable ;
- donner envie de continuer ;
- rendre les choix précédents compréhensibles ;
- rester récupérable selon les choix futurs.

## Fin du Jour 4

À la fin du Jour 4, le joueur doit sentir qu’une route commence à dominer, même si le jeu ne l’affiche pas en mode normal.

Exemples de fins de journée :

```text
Marie : Je ne sais pas ce qui me dérange le plus. Ce que j’ai vu, ou le fait que tu fasses comme si je n’avais rien vu.
Pauline : Tu m’as donné plus que tu ne crois.
Sandra : Supprime cette conversation si tu veux être raisonnable.
Mathilde : On aurait dû arrêter hier.
Nico a publié une nouvelle story avec Marie.
```

## Règle finale

> Le Jour 4 est le premier jour où le joueur comprend que son téléphone garde une mémoire.
