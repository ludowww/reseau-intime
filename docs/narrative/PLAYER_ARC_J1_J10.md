# Arc Player — J1 à J10

## Statut

Document de cadrage dédié à Player, à intégrer ensuite dans :

```text
SCENARIO_SPINE_J1_J10.md
CHARACTER_ARCS_J1_J10.md
PLAYER_RESPONSE_RULES.md
```

Objectif : définir Player comme personnage actif, pas seulement comme réceptacle de choix abstraits ou absence commentée par les autres.

## Principe central

```text
Player aime être désiré.
Il est curieux de ses propres limites.
Mais il n’est pas vraiment sûr de lui.
```

Deuxième principe :

```text
Player cherche sincèrement ce qu’il veut,
mais plus il découvre son pouvoir de désir,
plus ses limites reculent.
```

Troisième principe :

```text
Player peut être fleur bleue,
mais il peut aussi devenir dangereux quand il se sent désiré et capable d’obtenir une réponse.
```

## Fonction dramatique

Player représente :

```text
- la curiosité ;
- le besoin d’être désiré ;
- l’incertitude ;
- le romantisme parfois naïf ;
- le pouvoir découvert progressivement ;
- la tentation de tester ses limites ;
- le conflit entre sincérité et confort du mensonge ;
- la peur de choisir trop tôt ;
- le plaisir de sentir qu’il compte pour plusieurs femmes.
```

Il ne doit pas être écrit comme :

```text
- pur prédateur ;
- pur innocent ;
- homme parfaitement lucide ;
- simple spectateur ;
- manipulateur sans faille ;
- héros romantique irréprochable.
```

## Besoin émotionnel

Player aime être désiré.

Ce désir reçu le valorise, le rassure, puis le grise.

Il cherche :

```text
- confirmation qu’il plaît ;
- preuve qu’il peut encore troubler ;
- intensité ;
- attention ;
- complicité ;
- possibilité d’être choisi ;
- espace où il peut tester une version différente de lui-même.
```

Mais il n’est pas totalement sûr de lui.

Il peut douter :

```text
- de ce qu’il veut ;
- de ce qu’il mérite ;
- de ce qu’il peut assumer ;
- de ce qu’il provoque réellement ;
- de la différence entre désir et besoin d’être rassuré.
```

## Fleur bleue / romantisme

Player peut être fleur bleue par moments.

Il ne cherche pas seulement des photos ou une excitation immédiate.

Il peut être touché par :

```text
- une attention douce ;
- une confidence ;
- une phrase qui dit qu’il compte ;
- un souvenir ;
- une vulnérabilité ;
- l’idée d’être vu profondément ;
- la possibilité d’un lien qui dépasse le jeu.
```

Ce côté fleur bleue est important pour Sandra et Raphaëlle.

Mais il peut aussi servir à justifier ses glissements :

```text
Il peut appeler “lien” ce qui est aussi du désir.
Il peut appeler “sincérité” ce qui est aussi une manière d’obtenir plus.
```

## Pouvoir découvert

Player n’est pas sûr de lui au départ.

Mais il découvre qu’il peut :

```text
- faire sourire Sandra ;
- troubler Mathilde ;
- intéresser Pauline ;
- être vu par Raphaëlle ;
- rendre Marie jalouse ;
- provoquer des réponses ;
- obtenir des photos ;
- ouvrir des portes.
```

Chaque fois que ce pouvoir fonctionne, ses limites peuvent reculer.

Règle :

```text
Le pouvoir de Player doit l’attirer autant qu’il l’inquiète.
```

Il ne doit pas devenir maître de la situation trop vite.

Il découvre le pouvoir, mais perd aussi du contrôle.

## Curiosité de ses limites

Player teste parfois :

```text
- jusqu’où une phrase peut aller ;
- jusqu’où une photo peut être demandée ;
- jusqu’où un flirt peut rester défendable ;
- jusqu’où Marie peut accepter ;
- jusqu’où Sandra attend ;
- jusqu’où Mathilde joue ;
- jusqu’où Pauline contrôle ;
- jusqu’où Raphaëlle reste présente.
```

Mais il ne sait pas toujours nommer ce qu’il teste.

Il peut se raconter :

```text
je veux comprendre ;
je veux être honnête ;
je ne fais que répondre ;
ce n’est pas si grave ;
je n’ai rien fait de concret ;
je cherche ce que je ressens.
```

Ces phrases doivent pouvoir être vraies et fausses en même temps.

## Ses grandes postures

### Vérité partielle

Player peut dire quelque chose de vrai, mais incomplet.

```text
Je suis troublé.
Je ne sais pas encore ce que ça veut dire.
```

Effet : ouvre la clarté, mais peut aussi protéger l’ambiguïté.

### Mensonge doux

Player minimise pour éviter le coût.

```text
Ce n’est rien.
Je crois que tu interprètes trop.
```

Effet : protège à court terme, déclenche méfiance ou vengeance.

### Romantisme défensif

Player transforme le désir en lien noble.

```text
Je ne te regarde pas comme une bêtise.
C’est peut-être justement ça qui me fait peur.
```

Effet : très fort avec Sandra, dangereux avec Pauline, ambigu avec Marie.

### Provocation

Player teste son pouvoir.

```text
Tu dis ça comme si tu n’avais pas envie que je regarde.
```

Effet : augmente tension, mais peut abîmer confiance.

### Retenue

Player s’arrête ou pose une limite.

```text
Non. Là, si je continue, je vais appeler ça un jeu juste parce que ça m’arrange.
```

Effet : peut fermer une opportunité, mais créer une vraie confiance.

### Retour vers Marie

Player choisit de revenir au couple ou de clarifier.

```text
Je ne peux pas continuer à transformer tout ça en coins d’ombre.
```

Effet : route plus claire, mais pas forcément plus simple.

## Rapport à Marie

Marie est le centre et le miroir.

Player peut :

```text
- vouloir préserver le couple ;
- mentir pour éviter de le perdre ;
- proposer une ouverture ;
- être jaloux si Marie utilise Nico ;
- découvrir qu’il supporte mal chez elle ce qu’il s’autorise ;
- être attiré par une Marie plus active, plus libre, plus dangereuse.
```

Son conflit avec Marie :

```text
Il veut parfois une liberté qu’il n’est pas sûr de pouvoir accorder à Marie.
```

C’est une tension majeure.

## Rapport à Sandra

Sandra active le Player fleur bleue.

Avec elle, il aime :

```text
- être attendu ;
- être important ;
- être doux ;
- flatter ;
- valoriser ;
- croire que le trouble est presque romantique.
```

Danger :

```text
Il peut transformer son besoin d’être désiré en promesse implicite.
```

Règle : il doit pouvoir être sincère avec Sandra, mais jamais gratuitement innocent.

## Rapport à Mathilde

Mathilde active le Player joueur.

Avec elle, il aime :

```text
- chamailler ;
- provoquer ;
- tester ;
- prendre son parti ;
- sentir que le quotidien devient dangereux ;
- jouer avec les limites de la maison.
```

Danger :

```text
Il peut appeler “jeu” ce qui devient une intimité tactile réelle.
```

Règle : ses gestes doivent toujours garder une possibilité de recul ou de limite, sinon il devient trop agressif.

## Rapport à Pauline

Pauline active le Player grisé par le pouvoir.

Avec elle, il aime :

```text
- être vu dans son désir ;
- être testé ;
- être provoqué ;
- reprendre ou perdre le contrôle ;
- sentir qu’il peut négocier avec le danger.
```

Danger :

```text
Il peut confondre consentement au jeu et absence de conséquence.
```

Règle : Pauline doit parfois lui montrer qu’il n’est pas le seul à contrôler.

## Rapport à Raphaëlle

Raphaëlle active le Player qui veut devenir clair.

Avec elle, il aime :

```text
- respirer ;
- être vu sans être poussé ;
- répondre honnêtement ;
- se sentir moins dispersé ;
- imaginer une relation plus propre.
```

Danger :

```text
Il peut vouloir utiliser la clarté de Raphaëlle comme refuge au lieu de clarifier vraiment.
```

Règle : Raphaëlle ne doit avancer que si Player arrête de faire du flou une excuse.

## Progression cible J1-J10

```text
J1 — Player se croit encore dans une curiosité défendable.
J2 — Il découvre qu’il aime être attendu / remarqué.
J3 — Il commence à tester sans s’avouer qu’il teste.
J4 — Il perd le contrôle de ce qui est visible socialement.
J5 — Il découvre que ses réactions créent des dettes.
J6 — Il fait un acte concret : mentir, avouer, effacer, provoquer, ouvrir.
J7 — Il ressent les effets de son pouvoir : une femme recule, une autre avance, Marie agit.
J8 — Il découvre qu’il ne supporte pas toujours chez Marie ce qu’il autorise chez lui.
J9 — Il doit accepter un coût : vérité, jalousie, ouverture, perte, limite.
J10 — Il ne peut plus se croire simple spectateur de ses désirs.
```

## Garde-fous

À éviter :

```text
- Player prédateur ;
- Player complètement passif ;
- Player qui obtient tout sans coût ;
- Player qui sait parfaitement ce qu’il veut ;
- Player qui ment sans culpabilité ;
- Player qui devient incohérent selon les scènes ;
- choix abstraits sans réponse visible.
```

À préserver :

```text
- envie d’être désiré ;
- curiosité ;
- manque d’assurance ;
- romantisme par moments ;
- capacité de sincérité ;
- capacité de mensonge confortable ;
- pouvoir qui grise ;
- désir qui augmente ;
- limites qui reculent ;
- possibilité de se reprendre.
```

## Réparation des problèmes J5/J6 actuels

Player ne doit plus être seulement :

```text
un choix UI ;
une absence ;
un objet d’analyse des autres ;
un homme à qui on répète de poser son téléphone.
```

Il doit devenir :

```text
un homme qui parle ;
un homme qui se trompe ;
un homme qui aime être désiré ;
un homme qui teste ;
un homme qui peut être sincère ;
un homme qui ment parfois pour conserver plusieurs possibles ;
un homme qui découvre que son pouvoir le transforme.
```

## Phrase de référence

```text
Player n’est pas sûr de lui, mais il découvre qu’il peut troubler.
Et chaque fois qu’il trouble quelqu’un, il devient un peu plus curieux de savoir jusqu’où il pourrait aller.
```

Autre phrase :

```text
Le danger de Player n’est pas qu’il sait exactement ce qu’il veut.
C’est qu’il aime de plus en plus découvrir ce qu’il peut obtenir.
```
