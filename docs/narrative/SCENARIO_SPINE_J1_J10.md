# Colonne vertébrale scénario — J1 à J10

## Statut

Document de cadrage narratif.

Il sert à arrêter la progression au fil de l’eau et à fixer une trajectoire avant toute réécriture de J5/J6 ou création de J7.

État actuel :

```text
J1-J4 : base narrative utilisable.
J5-J6 : runtime techniquement intégré, mais narrativement provisoire.
J7 : à ne pas produire tant que J5/J6 ne sont pas recadrés.
```

## Problème identifié

À partir de J5/J6, les scènes commencent à tourner autour des mêmes motifs :

```text
- Player est absent / ailleurs ;
- Marie veut qu’il pose son téléphone ;
- Sandra attend “plus tard” ;
- Mathilde relance la trace ;
- Pauline garde des photos ;
- Raphaëlle refuse d’être une cachette.
```

Ces motifs sont bons, mais ils doivent changer de fonction chaque fois qu’ils reviennent.

Règle :

```text
Un sujet déjà traité ne peut revenir que s’il produit une conséquence nouvelle.
```

## Question dramatique globale

```text
Player cherche-t-il à comprendre ce qu’il désire,
ou devient-il seulement meilleur pour cacher ce qu’il provoque ?
```

Le jeu ne doit pas seulement accumuler des tensions. Chaque jour doit déplacer quelque chose : une relation, une preuve, une habitude, une image de soi, ou une capacité à mentir.

## Structure cible J1-J10

### J1 — Réactivation douce

Fonction : installer le couple et réactiver Sandra sans danger immédiat.

Événement principal : Sandra revient dans le téléphone par un prétexte photo / ancien lien.

Ce qui change : Player sent qu’une conversation extérieure au couple lui manque.

Personnages actifs : Marie, Sandra.

Personnages absents ou indirects : Mathilde indirecte via Marie.

Fin de journée : Player est encore dans une zone défendable : curiosité, nostalgie, trouble léger.

À ne pas répéter ensuite : simple reconnexion nostalgique sans conséquence.

### J2 — Ouverture du réseau

Fonction : élargir le monde sans le rendre chaotique.

Événement principal : apparition plus nette de Mathilde, Raphaëlle et du cadre social/travail.

Ce qui change : Player n’est plus seulement entre Marie et Sandra ; il est pris dans plusieurs regards.

Personnages actifs : Marie, Sandra, Mathilde, Raphaëlle.

Fin de journée : les canaux existent ; aucun n’est encore dangereux seul.

À ne pas répéter : introduction de personnages sans fonction dramatique.

### J3 — Retour domestique et première trace

Fonction : faire entrer le trouble dans la maison.

Événement principal : Mathilde crée une trace ou une situation ambiguë dans l’espace domestique.

Ce qui change : le désir n’est plus seulement à distance ; il habite un lieu partagé avec Marie.

Personnages actifs : Marie, Mathilde, Raphaëlle selon rythme.

Fin de journée : Player comprend qu’une trace peut survivre à une scène.

À ne pas répéter : simple photo ambiguë sans conséquence.

### J4 — Exposition sociale

Fonction : sortir le trouble du privé et le rendre observable.

Événement principal : soirée chez Pauline ; photos, regards, Nico autour de Marie, Sandra au mauvais moment.

Ce qui change : Player n’est plus seul à contrôler ce qui est vu.

Personnages actifs : Pauline, Marie, Nico, Mathilde, Sandra.

Fin de journée : les images existent, les regards existent, le contrôle de Player diminue.

À ne pas répéter : soirée/photo comme pur décor.

### J5 — Dette sociale des images, à réécrire

Fonction cible : montrer qu’une image crée une dette sociale ou intime.

Problème de la version actuelle : elle redistribue les motifs J4 sans transformation assez nette.

Ce que J5 doit vraiment accomplir :

```text
Player découvre qu’une image n’est pas seulement une preuve,
mais une dette : envers Marie, envers Mathilde, envers Sandra, envers lui-même.
```

Événement principal cible : un choix de Player autour d’une image doit produire un effet visible sur une relation.

Fin de journée cible : Player ne peut plus se dire simplement “j’ai regardé une photo”. Il a laissé une trace de sa réaction.

À éviter :

```text
- les personnages répètent qu’ils ont vu/attendu/gardé ;
- Marie répète uniquement le téléphone ;
- Pauline devient distributrice de photos ;
- Sandra répète seulement le plus tard ;
- Raphaëlle refait la même limite abstraite.
```

### J6 — Premier acte concret de Player, à réécrire

Fonction cible : Player agit concrètement, pas seulement il est analysé.

Problème de la version actuelle : les personnages parlent encore beaucoup de ce que Player devrait faire, mais Player ne répond pas assez directement.

Ce que J6 doit vraiment accomplir :

```text
Player fait un geste réel : effacer, mentir, avouer partiellement, organiser un moment, cacher mieux, ou se trahir.
```

Événement principal cible : une action de Player change l’état d’au moins une relation.

Fin de journée cible : un personnage ne revient pas au même état qu’au début du jour.

À éviter :

```text
- nouvelle variante de “pose ton téléphone” ;
- répétition du plus tard ;
- scènes où Player n’existe qu’à travers des choix abstraits ;
- Raphaëlle qui formule encore la même frontière ;
- Pauline qui augmente le mystère sans coût.
```

### J7 — Réaction du monde, à ne pas écrire encore

Fonction cible : un personnage agit en dehors de Player.

J7 ne doit pas commencer tant que J5/J6 ne sont pas réécrits.

Idée directrice : le monde ne doit pas seulement attendre les choix de Player.

Exemples possibles :

```text
- Marie fait quelque chose pour elle, pas contre Player ;
- Sandra se rend moins disponible ;
- Mathilde retire une possibilité ;
- Pauline perd momentanément le contrôle ;
- Raphaëlle impose une limite concrète de disponibilité ;
- Nico reste un miroir social, pas un rival actif caricatural.
```

### J8 — Première vraie contradiction

Fonction cible : Player ne peut plus maintenir deux versions de lui-même sans friction.

Événement possible : une phrase, un horaire, une image ou une absence contredit ce qu’il a laissé croire.

Fin cible : le joueur sent que la stratégie de dissimulation a un prix.

### J9 — Choix de coût

Fonction cible : choisir ne signifie pas finir une route, mais accepter un coût.

Exemples :

```text
- être plus honnête avec Marie mais perdre une opportunité ailleurs ;
- protéger Sandra mais créer une distance ;
- supprimer une trace Mathilde mais perdre une tension ;
- refuser le jeu de Pauline mais rater une information ;
- accepter la clarté Raphaëlle mais perdre le confort du refuge.
```

### J10 — Premier verrou narratif léger

Fonction cible : première vraie conséquence structurante.

Pas nécessairement route lock définitif, mais un état nouveau :

```text
- Marie sait qu’un problème existe ;
- Sandra a changé sa disponibilité ;
- Mathilde a nommé ou retiré la trace ;
- Pauline n’est plus seulement en contrôle ;
- Raphaëlle n’est plus un espace neutre ;
- Player a choisi une méthode : corriger, cacher, provoquer ou fuir.
```

## Règle de production pour chaque jour

Avant d’écrire un jour, remplir :

```text
Jour :
Titre :
Événement principal :
Question dramatique :
Ce qui change réellement :
Personnages actifs :
Personnages silencieux :
Nouvelle information :
Conséquence concrète :
Motifs interdits car déjà utilisés :
Fin de journée :
```

Tant que “ce qui change réellement” n’est pas clair, ne pas écrire la journée.
