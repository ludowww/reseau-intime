# Consentement, risque et scènes limites

## Statut

Document transversal de cadrage narratif.

Objectif : préserver le jeu, la tension, le risque, les photos et les dynamiques de provocation sans basculer dans des scènes malsaines ou non consenties.

Ce document complète :

```text
DIALOGUE_CONTENT_TEXTURE_RULES.md
PROOF_AND_SECRET_MAP.md
MATHILDE_ARC_J1_J10.md
PAULINE_ARC_J1_J10.md
ROUTE_COMPATIBILITY_MATRIX.md
```

## Principe central

```text
Le jeu peut être risqué.
Le récit peut mettre en scène la transgression, la gêne, le défi, le voyeurisme joué, la jalousie et la peur d’être vu.
Mais les contenus intimes, la nudité, les photos limites et les défis doivent rester consentis entre les personnages concernés.
```

Le risque recherché est narratif et émotionnel :

```text
être vu ;
être surpris ;
aller trop loin ;
regretter une phrase ;
laisser une trace ;
rendre jaloux ;
perdre le contrôle du cadre ;
se découvrir capable de désirer plus.
```

Ce n’est pas :

```text
exposer quelqu’un sans accord ;
voler une intimité réelle ;
forcer une photo ;
transformer le chantage en contrainte brutale ;
érotiser une absence de consentement.
```

## Photos dites “volées”

Le terme peut exister dans les dialogues comme jeu ou provocation, mais la scène doit montrer que ce n’est pas une captation non consentie durable.

Règle :

```text
Une photo surprise devient acceptable seulement si la personne concernée récupère très vite du contrôle.
```

Exemples de contrôle :

```text
- elle demande à voir ;
- elle demande l’envoi ;
- elle ordonne de supprimer ;
- elle choisit de garder ;
- elle répond par une photo ;
- elle transforme la gêne en jeu ;
- elle pose une limite claire.
```

Mauvais usage :

```text
Player prend une photo intime sans accord et la garde comme trophée.
```

Bon usage :

```text
Player prend une photo surprise dans un jeu déjà complice.
Il l’envoie immédiatement.
Mathilde proteste, exige de voir, reprend le contrôle et décide si le jeu continue.
```

## Nudité et contenu intime

Les contenus plus intimes peuvent exister dans l’évolution du jeu, mais doivent être cadrés par :

```text
- relation suffisamment avancée ;
- consentement clair ;
- contrôle de la personne qui envoie ou accepte ;
- progression émotionnelle ;
- possibilité de refus ou recul ;
- conséquence narrative.
```

Une photo intime ne doit pas être juste une récompense mécanique.

Elle doit être liée à :

```text
confiance ;
défi ;
jalousie ;
preuve de pouvoir ;
reprise de contrôle ;
moment de vulnérabilité ;
ouverture consentie ;
relation plus directe.
```

## Voyeurisme joué / risque public

Le voyeurisme dans le projet doit être compris comme un jeu de risque consenti, pas comme l’observation non consentie d’autrui.

Exemple Pauline :

```text
Pauline se prend elle-même en photo dans un lieu public ou semi-public.
Elle joue avec l’idée qu’il y a du monde autour.
Mais elle ne se fait pas réellement voir.
Le risque est dans le cadre, pas dans l’exposition non consentie.
```

Règle :

```text
Le risque peut venir du lieu, de l’heure, du cadrage ou de la possibilité d’être surprise.
Il ne doit pas venir du fait d’impliquer des inconnus non consentants dans une scène intime.
```

## Chantage Pauline

Pauline peut jouer avec le contrôle, les preuves et le silence.

Mais ce chantage doit rester :

```text
- ludique au départ ;
- ambigu ;
- relationnel ;
- réversible ;
- non brutal ;
- reconnu comme dangereux par le récit.
```

Elle peut dire :

```text
Je suis curieuse de savoir ce que tu crois que mon silence vaut.
```

Elle ne doit pas devenir :

```text
un maître-chanteur réaliste qui force Player sous menace explicite.
```

Si la dynamique devient plus dure, le récit doit la traiter comme un problème, pas comme un simple jeu sexy.

## Photos compromettantes

Il ne doit pas y avoir de photos compromettantes sans accord des personnes directement concernées par l’intimité.

Les photos sociales ou ambiguës peuvent exister :

```text
regards ;
proximité ;
groupe ;
angle mal interprété ;
photo défendable mais troublante.
```

Mais les contenus intimes ou humiliants ne doivent pas être pris ou diffusés sans consentement.

## Consentement dans les voies ouvertes

Pour les voies polyamour, trios, quatuors, relation libre ou expériences partagées :

```text
- les personnes concernées doivent savoir de quoi il s’agit ;
- Marie ne doit pas être placée devant un fait accompli ;
- une ouverture ne doit pas excuser rétroactivement un mensonge ;
- chacun doit pouvoir refuser, ralentir ou poser une limite ;
- le désir doit avancer avec un cadre, pas seulement par opportunisme.
```

## Vulnérabilité émotionnelle

Une scène peut utiliser une dispute, une solitude ou une blessure comme déclencheur.

Mais il faut éviter que Player exploite mécaniquement une vulnérabilité.

Exemple Sandra après dispute avec son copain :

```text
Elle peut chercher une présence, une valorisation ou une soirée.
Mais Player doit pouvoir répondre avec douceur, retenue, ou désir assumé avec précaution.
```

Règle :

```text
Plus un personnage est vulnérable, plus Player doit avoir une option de retenue ou de soin.
```

## Consentement et contrôle par personnage

### Sandra

Sandra contrôle ce qu’elle offre.

Elle peut jouer parce qu’elle se sent vue, pas parce qu’elle est poussée.

### Mathilde

Mathilde doit récupérer du contrôle dans les scènes de photo surprise ou de chamaillerie.

Son conflit avec Marie doit rester présent.

### Pauline

Pauline peut se mettre elle-même en risque.

Elle contrôle son image, son selfie, son défi.

Son pouvoir ne doit pas devenir coercition non reconnue.

### Raphaëlle

Raphaëlle exige clarté et limite.

Elle ne doit pas être sexualisée par surprise ou utilisée comme refuge sans que le récit le questionne.

### Marie

Marie peut provoquer, mentir, jalouser, ouvrir ou se venger.

Mais son agence doit rester forte : elle choisit, elle sait, elle teste, elle pose des règles.

## Règles de validation

Avant de valider une scène limite, vérifier :

```text
1. Qui contrôle l’image ou le défi ?
2. Qui peut dire non ?
3. La personne concernée sait-elle ce qui se passe ?
4. Le risque est-il narratif plutôt que non consenti ?
5. La scène offre-t-elle une option de retenue ?
6. Le récit reconnaît-il quand une dynamique devient dangereuse ?
7. La scène évite-t-elle d’impliquer des tiers non consentants dans l’intime ?
```

## Application J5/J6

J5 Pauline :

```text
Selfie provoque contrôlé par Pauline.
Elle choisit l’image.
Le risque vient de son contrôle et de son ambiguïté, pas d’une exposition forcée.
```

J6 Mathilde :

```text
Photo surprise possible seulement si Mathilde reprend immédiatement le contrôle : voir, supprimer, garder, renvoyer, punir ou relancer.
```

J7+ :

```text
Toute montée vers plus direct doit inclure consentement, jeu partagé, limite ou conséquence.
```

## Règle finale

```text
La tension peut venir du risque d’être vu.
Elle ne doit pas venir du fait de voler à quelqu’un le contrôle de son intimité.
```
