# 1. Ce qui reste utile historiquement

Les garde-fous suivants sont conservés dans le document `11` :

- une journée doit sembler vivante mais rester lisible ;
- le nombre de conversations sert le scénario ;
- une journée possède idéalement un pivot principal ;
- tous les personnages ne doivent pas écrire chaque jour ;
- une absence peut être une conséquence ;
- une journée de respiration n’est pas une journée vide ;
- plusieurs images peuvent exister, mais une seule progression visuelle doit généralement dominer ;
- une journée catalogue où toutes les routes avancent doit être évitée.

# 3. Règles historiques conservées

```text
Une journée doit sembler vivante,
mais elle doit rester lisible.
```

```text
Le nombre de conversations doit servir le scénario,
pas l’inverse.
```

```text
Une absence doit créer du manque, une limite,
une conséquence ou une vie propre.
```

```text
Une journée possède un pivot principal clair.
```

```text
Toutes les routes ne progressent pas en même temps.
```

Le document `11` définit désormais comment appliquer ces règles à la saison complète.

# 4. Trois contenus visuels

La cible historique de trois photos devient une règle canonique de :

```text
3 contenus visuels distincts minimum
par journée narrative complète
```

Lire le document `09` pour :

- photos diégétiques ;
- images de scène ;
- audiences ;
- sauvegarde ;
- galerie ;
- progression propre aux personnages.

Les objets ou décors seuls ne remplacent pas les contenus centrés sur les personnages.
