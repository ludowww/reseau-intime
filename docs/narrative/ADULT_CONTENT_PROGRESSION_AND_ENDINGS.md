# Progression adulte, photos et familles de fins

## Statut

Document de cadrage narratif.

Objectif : préciser l’évolution du contenu photo et les grandes familles de fins possibles, sans encore écrire les scènes finales ni figer les routes.

Ce document complète :

```text
DIALOGUE_CONTENT_TEXTURE_RULES.md
CONSENT_AND_RISK_RULES.md
DAILY_RHYTHM_AND_CONTENT_DENSITY.md
ROUTE_COMPATIBILITY_MATRIX.md
ROUTE_STATE_MODEL.md
MARIE_ARC_J1_J10.md
PLAYER_ARC_J1_J10.md
```

## Principe central

```text
Le contenu photo doit évoluer avec les relations.
Au début, il sert à connaître les personnages.
Ensuite, il devient plus intime, plus sexy, puis potentiellement explicite selon les routes, le consentement et la compatibilité des personnages.
```

La progression adulte ne doit pas être séparée de l’histoire.

Elle doit traduire :

```text
- confiance ;
- désir ;
- jeu ;
- pouvoir ;
- jalousie ;
- consentement ;
- compatibilité ;
- évolution du couple Marie / Player ;
- limites qui reculent ou sont réaffirmées.
```

## Progression des photos

### Phase 1 — Soft / découverte

Fonction : faire connaissance avec les personnages.

Types de photos :

```text
- photo quotidienne ;
- selfie simple ;
- photo souvenir ;
- photo sociale ;
- photo restaurant ;
- photo travail ;
- photo maison ;
- détail légèrement troublant mais défendable.
```

Objectif :

```text
Le joueur apprend à reconnaître les personnages, leurs voix, leurs corps sociaux, leurs habitudes, leurs manières de poser ou de ne pas poser.
```

### Phase 2 — Suggestif / sexy léger

Fonction : montrer que les personnages savent qu’ils sont regardés.

Types de photos :

```text
- détail plus assumé ;
- tenue plus choisie ;
- cadrage joueur ;
- selfie envoyé pour provoquer une réaction ;
- photo de confiance ;
- photo qui peut être commentée à double sens.
```

Objectif :

```text
La photo devient un langage relationnel.
```

### Phase 3 — Sexy assumé

Fonction : la relation devient plus directe.

Types de photos :

```text
- photo envoyée comme récompense ;
- photo de défi ;
- photo liée à une confidence ;
- photo de jalousie ;
- photo où le personnage assume l’effet recherché ;
- photo que Player doit choisir de garder, supprimer, montrer ou cacher.
```

Objectif :

```text
Les personnages ne font plus seulement semblant de ne pas comprendre le trouble.
Ils jouent avec lui.
```

### Phase 4 — Explicite / sexuel adulte

Fonction : contenu adulte avancé lié à une route et à un cadre clair.

Conditions obligatoires :

```text
- personnages adultes ;
- consentement clair ;
- progression relationnelle suffisante ;
- compatibilité de route ;
- contrôle de la personne qui envoie ou accepte ;
- conséquences narratives ;
- possibilité de limite ou de recul.
```

Objectif :

```text
Le contenu explicite doit être une conséquence d’une relation, pas une récompense déconnectée.
```

### Phase 5 — Contenu pornographique / route adulte avancée

Fonction : aboutissement de certaines routes adultes très avancées.

Conditions :

```text
- route clairement adulte ;
- consentement explicite des personnages concernés ;
- cadre relationnel établi ou transgression reconnue par le récit ;
- compatibilité forte avec les caractères ;
- conséquences sur Marie, Player, jalousie, ouverture ou pouvoir ;
- pas de contenu imposé, volé ou non consenti.
```

Règle :

```text
Le contenu pornographique peut exister dans les routes avancées,
mais il doit être préparé, consenti, compatible avec les personnages et narrativement situé.
```

## Progression du dialogue avec les photos

La parole doit suivre la progression visuelle.

```text
Soft → allusion légère.
Suggestif → taquinerie et double sens.
Sexy léger → demi-aveu et gêne assumée.
Sexy assumé → désir nommé, limites testées.
Explicite adulte → cadre, consentement, conséquences.
Pornographique avancé → route mature, pacte ou transgression pleinement assumée par le récit.
```

## Compatibilité par personnage

### Sandra

Progression :

```text
soft → valorisation → confiance → sexy par confidence → plus explicite si elle se sent vue et choisit d’oser.
```

Compatible avec :

```text
romantique ;
passionné ;
relation émotionnelle forte ;
ouverture si elle n’est pas cachée ;
photos intimes comme signe de confiance.
```

Incompatible avec :

```text
exposition brutale ;
être utilisée comme simple contenu secret ;
pression alors qu’elle est vulnérable ;
escalade sans valorisation.
```

### Mathilde

Progression :

```text
soft domestique → détail sexy → chamaillerie → photo jeu → sexy assumé → route plus explicite si le jeu est partagé et la loyauté envers Marie traitée.
```

Compatible avec :

```text
jeu ;
provocation ;
proximité domestique ;
transgression contrôlée ;
photos récompenses ;
route passionnée / interdite / maison dangereuse.
```

Incompatible avec :

```text
non-consentement ;
oubli de Marie ;
photo volée gardée sans reprise de contrôle ;
intégration trop facile à une ouverture sans conflit.
```

### Pauline

Progression :

```text
photo sociale → preuve faible → selfie provoque → défi → sexy assumé → explicite selon contrôle, confiance ou domination consentie.
```

Compatible avec :

```text
contrôle ;
provocation ;
défi ;
domination / soumission consentie selon dynamique ;
libertinage ;
NTR ou jalousie si cadre adapté ;
contenu risqué contrôlé par elle.
```

Incompatible avec :

```text
chantage brutal non reconnu ;
exposition de tiers non consentants ;
preuve gratuite sans coût ;
perte totale de sa fragilité.
```

### Raphaëlle

Progression :

```text
photo rare → confiance calme → désir nommé sobrement → intimité adulte seulement si clarté totale.
```

Compatible avec :

```text
romantique clair ;
passion maîtrisée ;
relation exclusive ;
relation libre très honnête ;
intimité adulte lente.
```

Incompatible avec :

```text
mensonge ;
cachette ;
chaos ;
contenu explicite trop rapide ;
utilisation comme refuge.
```

### Marie

Progression :

```text
couple quotidien → reconquête → jalousie → ouverture / vengeance / règle → contenu adulte selon route choisie.
```

Compatible avec :

```text
romantique ;
passionné ;
reconquête ;
ouverture ;
trio / quatuor / libertinage / polyamour si cadre clair ;
NTR si route jalousie / vengeance / accord le permet ;
domination ou soumission si cela correspond à sa reprise de pouvoir et au consentement.
```

Incompatible avec :

```text
être exclue du réel ;
ouverture utilisée pour excuser le mensonge ;
contenu adulte qui efface son agence ;
polyamour sans règles.
```

## Familles de fins possibles

Les fins ne sont pas encore écrites, mais les familles suivantes sont ouvertes.

### Fin romantique

Player choisit ou reconstruit un lien amoureux fort.

Variantes :

```text
- couple Marie reconstruit ;
- relation Sandra émotionnelle ;
- route Raphaëlle claire ;
- amour apaisé après vérité ;
- amour plus adulte après crise.
```

### Fin passionnée

Le désir domine davantage que la stabilité.

Variantes :

```text
- liaison intense ;
- couple qui se consume ;
- route Sandra plus ardente ;
- route Mathilde interdite ;
- Marie et Player dans une reconquête très physique ;
- passion avec coût émotionnel.
```

### Fin NTR / jalousie

La jalousie et le fait de voir ou savoir que l’autre désire ailleurs deviennent centraux.

Variantes :

```text
- Marie avec Nico ;
- Player confronté à son double standard ;
- jalousie subie ;
- jalousie acceptée ;
- jalousie transformée en jeu consenti ;
- couple fissuré par ce que chacun autorise à l’autre.
```

Condition :

```text
La route doit cadrer clairement si la jalousie est subie, provoquée, consentie ou intégrée à une dynamique adulte.
```

### Fin trio

Une configuration à trois devient possible.

Conditions :

```text
- consentement explicite ;
- compatibilité des personnages ;
- Marie informée si elle est concernée ;
- pas de tiers placé devant un fait accompli ;
- tension préparée par la route.
```

Variantes possibles :

```text
Marie / Player / autre personnage ;
Player / Sandra / cadre ouvert ;
Player / Mathilde / Marie seulement si conflit de loyauté traité ;
Player / Pauline / dynamique de défi ;
Player / Raphaëlle uniquement dans cadre très clair, si compatible.
```

### Fin quatuor / groupe restreint

Configuration plus ouverte ou libertine.

Conditions renforcées :

```text
- route adulte très avancée ;
- consentement explicite de tous ;
- compatibilité forte ;
- règles claires ;
- conséquences émotionnelles reconnues ;
- pas d’accumulation gratuite.
```

### Fin libertinage

Le couple ou certaines routes s’ouvrent à une logique d’expérimentation adulte.

Compatible surtout avec :

```text
Marie ouverture négociée ;
Pauline défis consentis ;
Nico / jalousie si cadre clair ;
certaines routes Sandra ou Mathilde selon consentement et maturité.
```

Incompatible avec :

```text
Raphaëlle si le cadre n’est pas très clair ;
Marie exclue du réel ;
mensonge persistant présenté comme ouverture.
```

### Fin domination / soumission

Dynamique adulte centrée sur le pouvoir consenti.

Compatible surtout avec :

```text
Pauline contrôle transformé en cadre consenti ;
Marie reprise de pouvoir ;
Player curieux de ses limites ;
certaines variantes passionnées si les limites sont nommées.
```

Conditions :

```text
- consentement explicite ;
- limites reconnues ;
- possibilité de recul ;
- compatibilité psychologique ;
- pas de confusion avec coercition ou chantage non reconnu.
```

### Fin polyamour / ouverture durable

Le couple ou réseau relationnel se transforme en cadre plus durable.

Conditions :

```text
- vérité ;
- règles ;
- consentement ;
- jalousie nommée ;
- places émotionnelles distinctes ;
- compatibilité entre personnages.
```

Compatible avec :

```text
Marie ouverte à un cadre ;
Sandra si non cachée ;
Raphaëlle uniquement si clarté totale ;
Mathilde très délicat, à cause de Marie ;
Pauline si le contrôle devient confiance ou jeu consenti.
```

### Fin chaos / rupture

Player ne clarifie pas, les routes se contredisent, les preuves s’accumulent.

Variantes :

```text
Marie vengeance ;
Pauline contrôle ;
Sandra blessée ;
Raphaëlle retrait ;
Mathilde culpabilité ;
Player seul avec son pouvoir devenu ingérable.
```

## Règle de compatibilité des fins

```text
Toutes les fins ne doivent pas être compatibles avec tous les personnages.
```

Chaque fin doit dépendre de :

```text
- caractère du personnage ;
- état du couple ;
- clarté ou mensonge ;
- niveau de confiance ;
- rapport au pouvoir ;
- rapport aux photos ;
- limites posées ou franchies ;
- consentement ;
- compatibilité avec les autres routes.
```

## Règle finale

```text
Le jeu peut aller vers du romantique, du passionné, du NTR, du trio, du quatuor, du libertinage, de la domination, de la soumission ou du polyamour.
Mais chaque direction doit émerger des personnages, de leurs choix, de leur consentement et de leurs compatibilités.
```
