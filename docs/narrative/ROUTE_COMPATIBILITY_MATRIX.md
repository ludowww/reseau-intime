# Matrice de compatibilité des routes

## Statut

Document de cadrage narratif.

Il sert à relier les arcs personnages au nouvel axe central :

```text
état du couple Marie / Player
+ posture de Player
+ degré de vérité ou mensonge
= routes compatibles, routes fragilisées, routes bloquées.
```

Ce document ne crée pas encore de variables runtime. Il sert à guider la réécriture J5/J6 et la future structure J7+.

## Principe central

```text
Les routes secondaires ne sont pas indépendantes.
Elles deviennent possibles, toxiques, ouvertes ou impossibles selon l’état du couple Marie / Player.
```

Marie n’est plus un simple obstacle. Elle devient le centre de compatibilité des routes.

## États majeurs du couple

### 1. Couple stable / reconquête

Player reconnaît ses désirs ou ses troubles, mais choisit de revenir vers Marie.

Effets :

```text
- Marie peut redevenir active et désirable ;
- Player doit poser des limites ailleurs ;
- les routes secondaires ralentissent ou changent de fonction ;
- la vérité vaut plus que l’intensité immédiate.
```

Routes compatibles :

```text
- Marie reconquête ;
- Raphaëlle comme alliée de clarté ;
- Sandra recul doux ou amitié troublée mais contenue ;
- Mathilde limite domestique ;
- Pauline refusée ou révélée comme danger.
```

Routes incompatibles ou très coûteuses :

```text
- Pauline contrôle / chantage ;
- Mathilde secret tactile dans la maison ;
- promesse forte à Sandra ;
- mensonge actif ;
- chaos multi-routes.
```

### 2. Couple fissuré / mensonge actif

Player cache, minimise, ment ou laisse Marie hors du réel.

Effets :

```text
- Marie devient méfiante ;
- Marie peut utiliser Nico pour reprendre du pouvoir ;
- Pauline gagne du pouvoir avec les preuves ;
- Sandra peut se sentir cachée ;
- Mathilde devient plus dangereuse dans la maison ;
- Raphaëlle recule si elle sent qu’elle sert de cachette.
```

Routes compatibles :

```text
- Pauline contrôle / preuves ;
- Mathilde jeu domestique risqué ;
- Sandra secret / blessure / valorisation cachée ;
- Marie vengeance avec Nico ;
- chaos désir.
```

Routes incompatibles ou fragiles :

```text
- Raphaëlle avancée ;
- relation libre claire ;
- polyamour consenti ;
- couple apaisé.
```

### 3. Vérité difficile / ouverture en discussion

Player avoue qu’il est troublé, aime être désiré, ou ne sait pas encore ce qu’il veut.

Effets :

```text
- Marie souffre mais n’est pas exclue du réel ;
- Marie peut proposer ou exiger une nouvelle règle ;
- les autres routes peuvent devenir plus assumées ;
- la jalousie existe mais peut être nommée ;
- Raphaëlle devient plus compatible ;
- Pauline perd une partie du pouvoir de chantage.
```

Routes compatibles :

```text
- Marie discussion d’ouverture ;
- Raphaëlle clarté ;
- Sandra confidence, si elle n’est pas cachée ;
- certaines variantes Mathilde, si Marie est reconnue dans le cadre ;
- ouverture progressive avec règles.
```

Routes délicates :

```text
- Pauline contrôle par preuve ;
- Mathilde dans la maison si Marie reste émotionnellement fragile ;
- Nico vengeance, si la vérité n’est pas assez assumée.
```

### 4. Ouverture consentie / relation libre claire

Marie et Player nomment une ouverture ou une exploration avec règles.

Effets :

```text
- les routes secondaires peuvent coexister plus proprement ;
- la tension vient du cadre, de la jalousie et du consentement, pas seulement du secret ;
- Marie peut participer ou poser des limites ;
- Raphaëlle devient compatible uniquement si tout est clair ;
- Pauline perd son pouvoir de chantage brut ;
- Sandra peut être rassurée ou intimidée ;
- Mathilde reste délicate à cause du lien avec Marie.
```

Routes compatibles :

```text
- Marie ouverture active ;
- Raphaëlle relation libre claire ;
- Sandra valorisée sans être cachée ;
- certaines expériences polyamour / trio / quatuor avec consentement explicite ;
- Pauline défis consentis plutôt que chantage ;
- Mathilde seulement si son conflit de loyauté est traité, pas effacé.
```

Routes incompatibles ou à transformer :

```text
- chantage Pauline comme moteur principal ;
- photos compromettantes non consenties ;
- secret Mathilde dans la maison ;
- mensonge persistant ;
- vengeance Nico non nommée.
```

### 5. Marie vengeance / Nico

Marie sent ou découvre le mensonge et reprend du pouvoir en utilisant Nico ou un regard extérieur.

Effets :

```text
- Player découvre qu’il supporte mal chez Marie ce qu’il s’autorise ;
- Nico devient miroir social, pas rival caricatural ;
- Pauline peut observer et photographier ;
- Marie devient active, pas seulement blessée ;
- le couple entre en guerre froide ou en négociation forcée.
```

Routes compatibles :

```text
- Nico miroir / jalousie ;
- Pauline preuve sociale ;
- chaos désir ;
- Player mensonge ou pouvoir grisé ;
- Marie face cachée.
```

Routes fragilisées :

```text
- couple stable ;
- Raphaëlle clarté ;
- Sandra douce si Player utilise Sandra comme compensation ;
- ouverture apaisée.
```

### 6. Chaos désir / multi-routes non clarifiées

Player nourrit plusieurs routes sans cadre clair.

Effets :

```text
- toutes les tensions montent ;
- Pauline contrôle ;
- Marie agit ;
- Sandra peut souffrir ;
- Mathilde devient risquée ;
- Raphaëlle recule ;
- les preuves et contradictions prennent de plus en plus de place.
```

Routes compatibles :

```text
- Pauline contrôle ;
- Mathilde secret ;
- Sandra attente blessée ;
- Marie vengeance ;
- contradictions / preuves.
```

Routes incompatibles :

```text
- vraie stabilité ;
- Raphaëlle avancée ;
- relation libre saine ;
- confiance durable Sandra ;
- ouverture polyamour propre.
```

## Compatibilité par personnage

### Sandra

Moteur : valorisation, confiance, regard ancien.

Compatible avec :

```text
- vérité difficile ;
- ouverture consentie si elle n’est pas cachée ;
- Player fleur bleue ;
- confidence progressive ;
- couple fissuré si cela crée manque et blessure, mais avec coût.
```

Incompatible avec :

```text
- être maintenue comme message caché trop longtemps ;
- promesse floue sans conséquence ;
- chaos où Player ne lui donne jamais de vraie place ;
- chantage Pauline qui l’expose sans qu’elle sache.
```

Conditions d’avancée :

```text
- Player la valorise précisément ;
- Player lui donne un moment réel ;
- elle se sent assez vue pour oser jouer ;
- elle n’est pas seulement une attente.
```

Conditions de recul :

```text
- Player la remet toujours à plus tard ;
- elle découvre qu’elle est cachée ;
- Player promet sans pouvoir assumer ;
- Marie devient trop blessée et Sandra le sent.
```

### Mathilde

Moteur : jeu domestique, chamaillerie, proximité tactile, loyauté conflictuelle envers Marie.

Compatible avec :

```text
- couple fissuré / mensonge actif ;
- route vengeance Marie, si la maison devient instable ;
- ouverture consentie très tardive et très cadrée ;
- Player joueur et curieux de ses limites.
```

Incompatible avec :

```text
- couple fermé reconquis ;
- Raphaëlle exclusive ;
- ouverture qui efface trop vite la culpabilité envers Marie ;
- voyeurisme malsain ou non partagé.
```

Conditions d’avancée :

```text
- Player prend son parti ;
- la complicité domestique monte ;
- le jeu reste partagé ;
- Mathilde récupère du contrôle après les transgressions ;
- Marie reste présente moralement.
```

Conditions de recul :

```text
- Player devient trop agressif ;
- Mathilde sent qu’elle va perdre Marie ;
- la photo/trace devient trop dangereuse ;
- Player pose une limite claire.
```

### Pauline

Moteur : contrôle, preuves, côté sulfureux, fragilité cachée.

Compatible avec :

```text
- mensonge actif ;
- chaos désir ;
- Marie vengeance / jalousie ;
- Player grisé par son pouvoir ;
- défis consentis si le cadre devient plus clair ;
- confiance fragile si Player voit autre chose que le danger.
```

Incompatible avec :

```text
- relation libre totalement claire qui retire tout pouvoir aux preuves ;
- couple fermé reconquis ;
- Raphaëlle clarté exclusive ;
- absence de coût pour les preuves.
```

Conditions d’avancée :

```text
- Player accepte d’être testé ;
- Pauline possède ou provoque des preuves ;
- le jeu de contrôle reste excitant mais risqué ;
- sa fragilité peut apparaître derrière le contrôle.
```

Conditions de transformation :

```text
- si le mensonge continue, Pauline devient dangereuse ;
- si Player la confronte avec finesse, elle peut révéler sa fragilité ;
- si l’ouverture devient claire, ses défis doivent devenir consentis plutôt que chantage.
```

### Raphaëlle

Moteur : clarté, douceur, présence, sécurité, cadre.

Compatible avec :

```text
- vérité difficile ;
- relation libre claire ;
- route exclusive après clarification ;
- Player qui accepte d’être moins flou ;
- Marie informée ou respectée.
```

Incompatible avec :

```text
- mensonge actif ;
- cachette ;
- chaos multi-routes ;
- chantage Pauline ;
- secret Mathilde non traité ;
- Player qui veut seulement un refuge calme.
```

Conditions d’avancée :

```text
- Player dit une phrase vraie ;
- Player accepte un cadre ;
- Raphaëlle voit un geste concret, pas seulement une confession ;
- le désir avance dans la lumière.
```

Conditions de recul :

```text
- Player la transforme en refuge ;
- Marie est exclue du réel ;
- Player ment pour préserver plusieurs possibles ;
- Raphaëlle doit répéter la même limite.
```

### Marie

Moteur : couple en danger, vérité, pouvoir repris, ouverture ou vengeance.

Compatible avec :

```text
- toutes les routes, mais jamais sous la même forme ;
- ouverture si elle sait et consent ;
- vengeance si Player ment ;
- couple fermé si Player revient avec coût ;
- Nico comme miroir si elle reprend du pouvoir.
```

Incompatible avec :

```text
- être passive trop longtemps ;
- être exclue du réel ;
- accepter rétroactivement le mensonge ;
- rester simple obstacle aux autres routes.
```

Conditions d’ouverture :

```text
- Player dit la vérité avant d’être pris ;
- Marie garde une place active dans les règles ;
- la jalousie est nommée ;
- les personnes concernées consentent ;
- l’ouverture ne sert pas à excuser le passé.
```

Conditions de vengeance :

```text
- mensonge répété ;
- minimisation ;
- preuve découverte ;
- sentiment d’être prise pour une idiote ;
- Player veut une liberté qu’il refuse de lui accorder.
```

## Routes principales proposées

### Route A — Reconquête Marie / couple fermé

Description : Player reconnaît ses troubles et revient vers Marie.

Conditions :

```text
vérité partielle ou complète ;
limites posées ailleurs ;
preuve ou photo traitée proprement ;
Marie active mais pas vengeresse.
```

Routes secondaires :

```text
Sandra : ralentissement ou amitié troublée.
Mathilde : limite domestique.
Pauline : refus ou danger exposé.
Raphaëlle : alliée de clarté possible.
```

### Route B — Mensonge actif / couple fissuré

Description : Player garde plusieurs possibles et ment.

Conditions :

```text
minimisation ;
photos gardées ;
preuves faibles ignorées ;
Marie hors du réel.
```

Routes secondaires :

```text
Sandra : blessure / secret.
Mathilde : jeu risqué.
Pauline : contrôle fort.
Raphaëlle : recul.
Marie : vengeance ou Nico.
```

### Route C — Marie vengeance / Nico

Description : Marie reprend du pouvoir en utilisant Nico ou un regard extérieur.

Conditions :

```text
mensonge Player ;
preuve ou intuition forte ;
Marie refuse la passivité.
```

Routes secondaires :

```text
Pauline observe et amplifie.
Nico devient miroir social.
Player découvre son hypocrisie.
```

### Route D — Ouverture négociée

Description : Marie et Player nomment une ouverture possible.

Conditions :

```text
vérité avant explosion ;
consentement ;
règles ;
place active de Marie ;
coût assumé.
```

Routes secondaires :

```text
Sandra : possible si elle n’est pas cachée.
Raphaëlle : possible si clarté totale.
Mathilde : possible mais très délicate à cause de Marie.
Pauline : défis consentis, pas chantage.
```

### Route E — Route exclusive Raphaëlle

Description : Player choisit une clarté plus radicale avec Raphaëlle.

Conditions :

```text
séparation ou clarification avec Marie ;
recul des routes sulfureuses ;
vérité ;
coût fort.
```

Routes incompatibles :

```text
Pauline contrôle ;
Mathilde secret domestique ;
Sandra promesse floue ;
mensonge actif.
```

### Route F — Chaos désir / routes multiples non clarifiées

Description : Player nourrit plusieurs routes sans cadre.

Conditions :

```text
mensonge ;
flou ;
curiosité de pouvoir ;
photos et preuves ;
Marie active mais non informée.
```

Effets :

```text
Pauline devient dangereuse ;
Marie agit ;
Sandra souffre ;
Mathilde devient risquée ;
Raphaëlle recule ;
Player perd le contrôle.
```

## Règles de compatibilité fortes

```text
Raphaëlle avancée exige vérité ou cadre clair.
Pauline contrôle prospère dans le mensonge mais doit coûter quelque chose.
Sandra avance par valorisation, mais recule si elle devient uniquement cachée.
Mathilde avance par jeu domestique, mais Marie doit rester une limite active.
Marie peut ouvrir la relation, mais n’accepte pas d’être exclue du réel.
Nico est un miroir de Marie, pas une route parallèle de Player.
```

## Règle de conception pour J5/J6

Avant de réécrire une scène, choisir explicitement :

```text
Quel état du couple cette scène nourrit-elle ?
Quelle route devient plus compatible ?
Quelle route devient plus coûteuse ?
Quelle route doit reculer ?
```

Si la scène ne modifie aucune compatibilité, elle risque de tourner en rond.
