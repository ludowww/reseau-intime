# Arc Mathilde — J1 à J10

## Statut

Document de cadrage dédié à Mathilde, à intégrer ensuite dans :

```text
SCENARIO_SPINE_J1_J10.md
CHARACTER_ARCS_J1_J10.md
PROOF_AND_SECRET_MAP.md
```

Objectif : remplacer la simple répétition “photo canapé / trace gardée” par une vraie trajectoire de jeu domestique, complicité, proximité tactile et conflit de loyauté envers Marie.

## Principe central

```text
Mathilde commence par jouer avec la complicité.
Le danger, c’est que cette complicité devient un langage intime dans la maison de Marie.
```

Deuxième principe :

```text
Mathilde ne donne pas d’abord de grandes confidences.
Elle laisse entrer Player par le jeu, la chamaillerie et la proximité.
```

## Fonction dramatique

Mathilde représente :

```text
- la maison ;
- le quotidien rendu dangereux ;
- la chamaillerie ;
- la complicité avec Player ;
- le désir qui se cache derrière le jeu ;
- le tactile qui devient naturel ;
- la loyauté conflictuelle envers Marie.
```

Elle n’est pas une simple tentatrice.

Elle est intéressante parce qu’elle aime le jeu, mais sait que ce jeu touche directement Marie.

## Besoin / conflit intérieur

Mathilde aime :

```text
- plaire ;
- être regardée ;
- sentir que Player prend son parti ;
- le faire réagir ;
- gagner des petites batailles de mauvaise foi ;
- transformer la maison en terrain de complicité.
```

Mais elle ne veut pas :

```text
- perdre Marie ;
- devenir “celle qui prend la place” ;
- reconnaître trop vite que le jeu dépasse le jeu ;
- être regardée seulement comme une opportunité sexuelle ;
- faire exploser la maison.
```

Son conflit :

```text
elle aime provoquer,
elle aime que Player réponde,
mais Marie reste la limite vivante de chaque scène.
```

## Couleur de voix

Mathilde doit rester :

```text
taquine
vive
insolente
spontanée
mauvaise foi assumée
capable de gêne rapide
capable de recul si Marie devient trop présente
```

Elle peut être plus expressive que Sandra.

Elle utilise l’humour pour garder le contrôle quand elle est gênée.

## Complicité avec Player

La relation Mathilde / Player avance par petits alignements.

Exemples :

```text
- Player prend son parti dans une dispute légère avec Marie ;
- Player la taquine sur une habitude domestique ;
- Mathilde lui envoie une photo innocente mais piégée ;
- ils se chamaillent sur une preuve ;
- un contact physique devient trop naturel ;
- Mathilde remarque quand Player la choisit symboliquement.
```

Le jeu ne doit pas être seulement visuel.

Il doit être relationnel :

```text
Qui défend qui ?
Qui gagne la blague ?
Qui ose nommer le détail ?
Qui recule quand Marie revient dans la phrase ?
```

## Importance de Marie

Marie est la limite morale et affective de la route Mathilde.

Mathilde peut jouer, mais elle ne doit jamais oublier totalement Marie.

Marie peut apparaître par :

```text
- une dispute légère avec Mathilde ;
- une phrase de Mathilde ;
- un bruit dans la maison ;
- une absence qui rend la maison dangereusement libre ;
- une culpabilité après une montée ;
- une limite que Mathilde pose elle-même.
```

Règle :

```text
Si une scène Mathilde devient trop chaude sans que Marie existe quelque part dans l’angle,
elle perd sa spécificité.
```

## Photos innocentes avec détail sexy

Mathilde peut envoyer quelques photos qui semblent innocentes, mais ont toujours un détail troublant.

Exemples :

```text
- t-shirt trop large ;
- jambe visible sur canapé ;
- posture relâchée ;
- miroir de salle de bain ;
- détail de peau ou d’épaule ;
- tenue de nuit justifiée par le contexte ;
- cadrage domestique non officiellement provocant.
```

Principe :

```text
photo innocente en surface,
impossible à regarder comme complètement innocente.
```

Ces photos ne doivent pas être immédiatement explicites.

Le détail fait le trouble.

## Prise de parti dans les disputes avec Marie

Mathilde peut profiter de petites disputes avec Marie.

Exemple :

```text
Marie reproche quelque chose à Mathilde.
Player prend le parti de Mathilde, même légèrement.
Mathilde s’en amuse et garde le point en mémoire.
```

Fonction :

```text
faire sentir que Player la choisit parfois symboliquement,
sans qu’il puisse encore appeler ça une vraie trahison.
```

Mathilde peut ensuite relancer :

```text
J’ai noté que tu étais de mon côté.
C’était probablement une erreur stratégique.
Mais une erreur flatteuse.
```

## Photo volée / cadrage de consentement ludique

Idée de scène : Player prend une photo de Mathilde dans un moment domestique, par exemple dans la salle de bain pendant qu’elle se brosse les dents.

Attention : cette idée doit être cadrée pour rester dans le jeu de chamaillerie et ne pas devenir une vraie violation.

Version recommandée :

```text
Player la surprend dans un moment domestique.
Il prend une photo rapidement.
Il la lui envoie tout de suite.
Mathilde proteste, le traite de voleur, demande à voir, puis reprend du pouvoir dans le jeu.
```

Règle :

```text
photo volée = acceptable seulement si la scène montre très vite que Mathilde récupère du contrôle.
```

Exemple de ton Mathilde :

```text
T’es sérieux ? 😳
Supprime ça.
... enfin attends, envoie-la-moi d’abord.
Je veux voir à quel point tu abuses.
```

Puis :

```text
Tu es insupportable.
Et cette photo est beaucoup trop bien pour quelqu’un qui devrait se faire gronder.
```

Le jeu doit être :

```text
“je te tiens” ludique,
pas “je t’ai piégée” malsain.
```

## Jeu de chamaillerie

La route Mathilde avance par :

```text
- petites provocations ;
- photos piquées ;
- revanche ;
- mini-menaces ludiques ;
- défis ;
- récompenses ;
- contacts physiques qui commencent comme des blagues ;
- mauvaise foi partagée.
```

La chamaillerie doit cacher le trouble, pas le remplacer.

Plus le jeu continue, plus il devient difficile de dire qu’il n’est “qu’un jeu”.

## Scène araignée la nuit

Idée pivot très forte.

Mathilde envoie un message la nuit parce qu’il y a une araignée et demande à Player de descendre.

Fonction :

```text
- prétexte concret ;
- scène domestique crédible ;
- proximité nocturne ;
- vulnérabilité légère ;
- tenue plus intime justifiée ;
- montée tactile naturelle ;
- possibilité de photo prise dans le jeu.
```

Structure possible :

```text
Mathilde appelle Player à l’aide.
Player descend.
Elle est en petite tenue / tenue de nuit.
Il se moque gentiment de son panique.
Il prend ou reçoit une photo de la scène.
Elle proteste, rit, récupère le contrôle.
Plus tard, elle le remercie avec une photo récompense.
```

Garde-fou : la scène doit rester légère et complice, pas menaçante.

## Photo récompense

Après l’aide de Player, Mathilde peut envoyer une photo récompense sexy.

Fonction :

```text
- elle reprend le pouvoir ;
- elle choisit de jouer ;
- elle récompense une action ;
- elle augmente la tension sans se rendre disponible sans limite.
```

La photo récompense signifie :

```text
tu m’as amusée ;
tu m’as aidée ;
tu as gagné un point ;
tu m’as regardée comme j’avais envie.
```

Important :

```text
Mathilde ne subit pas le jeu.
Elle l’alimente, le cadre, le punit ou le récompense.
```

## Montée tactile

Le tactile doit progresser progressivement.

Exemples :

```text
- épaule contre épaule sur le canapé ;
- se pousser en riant ;
- récupérer un téléphone ;
- attraper un plaid ;
- toucher involontaire puis assumé par une blague ;
- distance réduite pendant une scène de peur ou de chamaillerie.
```

Le tactile Mathilde doit commencer comme quelque chose de défendable.

Puis il devient trop naturel.

## Soirée télé quand Marie est absente

Idée de scène pivot.

Marie doit partir ou s’absenter de la maison.

Mathilde propose une soirée télé.

Fonction :

```text
- la maison change d’équilibre sans Marie ;
- la tension quitte le téléphone ;
- la proximité physique devient centrale ;
- des photos complices et tendancieuses apparaissent ;
- le jeu devient presque une scène de couple, ce qui est précisément dangereux.
```

Éléments possibles :

```text
canapé
plaid
popcorn / tasse / série nulle
photo complice
jambe proche
épaule contre épaule
blague sur le fait que Marie se moquerait d’eux
silence un peu trop long
```

Garde-fou : Marie doit être absente physiquement mais présente moralement.

Mathilde peut dire quelque chose comme :

```text
On dirait qu’on fait une bêtise alors qu’on regarde juste une série médiocre.
C’est probablement mauvais signe.
```

## Progression cible J1-J10

```text
J1-J2 — Mathilde indirecte ou présence domestique légère.
J3 — Première photo / premier trouble / scène canapé.
J4 — Tension sociale ; Player prend parfois son parti dans une petite opposition avec Marie.
J5 — Photos innocentes avec détail sexy ; jeu plus net.
J6 — Photo volée domestique cadrée comme chamaillerie ; limite ludique.
J7 — Scène araignée la nuit ; petite tenue ; photo récompense.
J8 — Proximité tactile plus naturelle ; chamaillerie plus chargée.
J9 — Marie absente ; soirée télé ; photos complices et tendancieuses.
J10 — Moment où le jeu ne peut plus être appelé seulement jeu.
```

## Garde-fous

À éviter :

```text
- Mathilde simple tentatrice ;
- Player voyeur trop agressif ;
- photo non consentie traitée comme normale ;
- oubli de Marie ;
- progression trop frontale ;
- même photo ambiguë répétée plusieurs jours ;
- Mathilde qui ne fait que demander si une trace existe.
```

À préserver :

```text
- jeu partagé ;
- humour ;
- récupération de contrôle par Mathilde ;
- culpabilité envers Marie ;
- montée tactile lente ;
- photos à double lecture ;
- possibilité pour Player de poser une limite.
```

## Réparation des problèmes J5/J6 actuels

La route Mathilde ne doit plus tourner autour de :

```text
Tu as gardé la photo ?
La trace est dangereuse.
C’est un jeu ou une limite ?
```

Elle doit avancer vers :

```text
chamaillerie concrète ;
prise de parti ;
photo piquée avec reprise de contrôle ;
récompense ;
appel nocturne ;
proximité physique ;
soirée en l’absence de Marie.
```

## Phrase de référence

```text
Mathilde ne tombe pas dans une confession.
Elle construit un jeu dans lequel chaque blague rapproche un peu trop les corps.
```
