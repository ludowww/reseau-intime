# Arc Pauline — J1 à J10

## Statut

Document de cadrage dédié à Pauline, à intégrer ensuite dans :

```text
SCENARIO_SPINE_J1_J10.md
CHARACTER_ARCS_J1_J10.md
PROOF_AND_SECRET_MAP.md
```

Objectif : donner à Pauline une vraie trajectoire de contrôle, preuve, jeu sulfureux et fragilité cachée, sans la réduire à “celle qui prend des photos”.

## Principe central

```text
Pauline donne l’impression de contrôler les preuves des autres,
mais son vrai danger vient du fait qu’elle est elle-même déjà compromise.
```

Deuxième principe :

```text
Pauline expose les autres pour ne pas être la seule à avoir quelque chose à cacher.
```

Troisième principe :

```text
Pauline joue au contrôle parce qu’elle sait déjà ce que ça coûte de perdre le sien.
```

## Fonction dramatique

Pauline représente :

```text
- l’apparence sage ;
- la maîtrise sociale ;
- l’observation ;
- les preuves ;
- la tentation du contrôle ;
- le côté sulfureux caché ;
- la fragilité derrière la provocation ;
- le plaisir de faire apparaître ce que les autres cachent.
```

Elle n’est pas seulement une menace.

Elle doit être à la fois :

```text
dangereuse parce qu’elle voit,
désirable parce qu’elle ose,
fragile parce qu’elle a déjà franchi une limite,
troublante parce qu’elle comprend trop bien Player.
```

## Apparence sage / côté sulfureux

En public, Pauline donne une impression de contrôle propre :

```text
organisée
souriante
sage en apparence
maîtresse de maison
celle qui cadre les soirées
celle qui semble savoir garder les formes
```

En privé, elle révèle progressivement :

```text
un goût du risque ;
un plaisir de provoquer ;
un rapport trouble aux preuves ;
une expérience de la tromperie ;
une capacité à voir le désir chez les autres ;
une fragilité qu’elle masque par le contrôle.
```

Le contraste est central.

## Secret : Pauline a déjà trompé son copain

Pauline a déjà trompé son copain.

Marie le sait.

Cela crée plusieurs effets :

```text
- Marie n’est pas naïve sur Pauline ;
- Pauline n’est pas extérieure au thème de l’infidélité ;
- Pauline peut comprendre Player sans le juger frontalement ;
- Pauline a une faille personnelle ;
- Pauline peut exposer les autres parce qu’elle sait ce que signifie être exposée.
```

Attention : Marie sait une partie du passé de Pauline, mais pas nécessairement ce que Pauline fait avec Player.

## Ce que Marie sait

Marie peut savoir :

```text
- Pauline a déjà trompé son copain ;
- Pauline n’est pas aussi sage qu’elle en a l’air ;
- Pauline aime parfois jouer avec le feu ;
- Pauline est capable de garder des choses pour elle.
```

Marie ne doit pas forcément savoir :

```text
- que Pauline teste Player directement ;
- que Pauline envoie un selfie provoque ;
- que Pauline accumule des preuves sur Player ;
- que Pauline propose des défis ;
- ce que Pauline attend vraiment de Player.
```

Cette asymétrie est utile : Marie connaît Pauline, mais pas toute la relation Pauline / Player.

## Soirée orchestrée

Pauline propose ou organise une soirée.

Fonction :

```text
- réunir le groupe ;
- observer les tensions ;
- prendre des photos ;
- créer des occasions ;
- semer une zizanie sociale légère ;
- regarder comment Player réagit quand plusieurs désirs sont visibles en même temps.
```

Pauline ne doit pas seulement “prendre des photos”.

Elle doit orchestrer :

```text
placement des personnes ;
phrases qui rapprochent ;
photos prises au bon moment ;
petites remarques innocentes ;
espace social où tout peut rester défendable.
```

## Photos / preuves / observation

Pauline accumule des preuves.

Mais chaque preuve doit avoir une fonction.

Types de preuves Pauline :

```text
photo sociale ambiguë ;
photo qui montre un regard ;
photo où Player est visible au mauvais endroit ;
photo où Marie est regardée par quelqu’un d’autre ;
photo où Pauline apparaît trop clairement dans le jeu ;
trace que Pauline garde sans l’envoyer ;
preuve qui peut se retourner contre elle.
```

Règle :

```text
Pauline sait parce qu’elle a vu, cadré, reçu ou provoqué.
Elle ne doit pas être omnisciente.
```

## Confrontation douce du lendemain

Le lendemain de la soirée, Pauline contacte Player.

Elle ne doit pas menacer frontalement au départ.

Fonction :

```text
- confronter Player gentiment ;
- lui dire qu’elle comprend qu’il puisse être attiré par d’autres ;
- créer une complicité dangereuse ;
- déplacer la culpabilité vers une forme de désir partagé ;
- montrer qu’elle n’est pas si sage.
```

Tonalité possible :

```text
Je ne te juge pas.
Je crois même que je comprends très bien.
C’est peut-être ça le problème.
```

Ou :

```text
Tu regardes comme quelqu’un qui essaie encore de se convaincre qu’il ne regarde pas.
C’est assez touchant, en fait.
```

## Selfie provoque

Après la confrontation douce, Pauline peut envoyer un selfie provoque.

Fonction :

```text
- elle devient elle-même tentation ;
- elle montre qu’elle ne parle pas depuis une position morale pure ;
- elle teste si Player regarde aussi quand il sait qu’il est testé ;
- elle fait glisser le rapport de force vers un jeu plus sulfureux.
```

Le selfie doit être contrôlé par Pauline.

Il n’est pas une récompense de soumission.

Il dit plutôt :

```text
Tu n’es pas le seul à savoir regarder ailleurs.
Et je sais très bien ce que je fais quand je t’écris.
```

## “Que ferais-tu pour que je garde le silence ?”

Pauline peut ensuite contacter Player pour savoir ce qu’il serait prêt à faire pour qu’elle garde le silence.

Cette idée doit rester dans un cadre de chantage ludique / test de pouvoir, pas de coercition brutale.

Version recommandée :

```text
Je pourrais garder ça pour moi.
Mais je suis curieuse de savoir ce que tu crois que ça vaut.
```

Ou :

```text
Rassure-toi, je ne suis pas méchante.
Je suis seulement très mauvaise quand je m’ennuie 😇
```

Règle :

```text
La menace doit être jouée comme un test sulfureux au début,
mais le récit doit toujours laisser sentir qu’un vrai danger pourrait apparaître si Player nourrit trop ce jeu.
```

## Contrôle vs confiance

La route Pauline doit pouvoir bifurquer entre deux directions.

### Direction contrôle

Pauline garde l’avantage.

Elle :

```text
accumule les preuves ;
propose des défis ;
montre qu’elle peut cadrer les autres ;
teste Player ;
augmente l’intensité ;
transforme le secret en jeu de pouvoir.
```

Risque : elle devient trop dominante ou toute-puissante.

Correction : le contrôle doit créer un risque pour elle aussi.

### Direction confiance / fragilité

Pauline laisse apparaître une sensibilité fragile.

Elle peut révéler :

```text
qu’elle n’a pas trompé par simple cynisme ;
qu’elle s’est sentie invisible ou enfermée dans l’image de la fille sage ;
qu’elle contrôle les preuves parce qu’elle a peur d’être exposée ;
qu’elle veut être vue autrement sans être réduite à une faute ;
qu’elle ne sait pas toujours distinguer jeu et fuite.
```

Cette direction rend Pauline plus humaine.

Elle ne doit pas supprimer son danger, mais le rendre compréhensible.

## Défis progressifs

Pauline peut proposer des défis à Player.

Les défis doivent monter en intensité, mais rester cadrés.

Types de défis possibles :

```text
- envoyer une phrase ambiguë en groupe ;
- prendre une photo sociale avec un détail volontaire ;
- reconnaître qu’il a regardé ;
- deviner ce qu’elle garde ;
- accepter une photo d’elle sans pouvoir répondre immédiatement ;
- provoquer une réaction légère en public ;
- choisir entre recevoir une preuve ou protéger quelqu’un ;
- faire disparaître une preuve ou en créer une autre.
```

Les défis doivent tester :

```text
désir ;
culpabilité ;
contrôle ;
mensonge ;
loyauté ;
capacité à refuser.
```

## Voyeurisme / photos osées en public

Pauline peut jouer avec le fait de regarder ou d’être regardée.

Mais il faut éviter de transformer la route en voyeurisme non consenti.

Version acceptable :

```text
- photo d’elle-même prise ou contrôlée par elle ;
- selfie osé mais cadré par Pauline ;
- photo sociale ambiguë ;
- détail public défendable ;
- regard capturé dans une soirée ;
- défi de repérer un détail sans exposer quelqu’un de vulnérable.
```

À éviter :

```text
- photos intimes non consenties ;
- humiliation réelle ;
- chantage violent ;
- photos compromettantes d’autres personnages traitées comme excitation principale ;
- Player forcé à devenir prédateur ;
- Pauline impunie et toute-puissante.
```

## Photos compromettantes de l’entourage

Les photos compromettantes de l’entourage peuvent exister comme preuves narratives.

Elles ne doivent pas devenir un jeu érotique aux dépens de personnes non consentantes.

Fonction acceptable :

```text
- montrer que Pauline observe trop ;
- créer une dette ou un danger ;
- forcer Player à choisir entre curiosité et protection ;
- révéler que Pauline franchit parfois des limites ;
- faire apparaître un risque moral.
```

Fonction à éviter :

```text
- collection de photos compromettantes comme récompense sexy ;
- exploitation de personnages qui ne participent pas au jeu ;
- escalade sans coût.
```

## Fragilité cachée

Pauline doit avoir une vraie zone sensible.

Pistes :

```text
- elle a longtemps joué le rôle de fille sage ;
- sa tromperie passée n’était pas seulement un caprice ;
- elle aime contrôler les preuves parce qu’elle a peur d’être définie par une seule faute ;
- elle teste Player parce qu’elle veut voir s’il est hypocrite ou honnête sur son désir ;
- elle peut être blessée si Player ne voit en elle qu’un risque excitant.
```

Exemple de tonalité :

```text
Tu crois que je fais ça parce que je n’ai pas de morale.
C’est plus pratique de croire ça.
Moi aussi, ça m’arrange certains jours.
```

Ou :

```text
La fille sage, c’est très reposant pour les autres.
Un peu moins pour celle qui doit la jouer tout le temps.
```

## Progression cible J1-J10

```text
J1-J3 — Pauline absente ou sociale en arrière-plan.
J4 — Soirée chez Pauline : elle organise, cadre, observe, prend des photos.
J5 — Confrontation douce : “je comprends que tu regardes ailleurs”.
J6 — Selfie provoque : Pauline devient elle-même tentation.
J7 — Question du silence : que ferait Player pour qu’elle garde certaines choses ?
J8 — Défis progressifs : transgressions sociales, photos contrôlées, jeu de pouvoir.
J9 — Faille : une preuve ou une émotion peut se retourner contre Pauline.
J10 — Choix : Pauline reste dans le contrôle ou accorde une vraie confiance.
```

## Garde-fous spécifiques

À éviter :

```text
- Pauline omnisciente ;
- Pauline simple maître-chanteuse ;
- chantage violent ;
- voyeurisme non consenti traité comme sexy ;
- accumulation de preuves sans conséquence ;
- photos compromettantes de l’entourage comme récompense ;
- absence de coût pour Pauline ;
- révélation complète trop tôt.
```

À préserver :

```text
- apparence sage ;
- contrôle social ;
- côté sulfureux privé ;
- humour faussement innocent ;
- danger des preuves ;
- faille personnelle ;
- possibilité de confiance ;
- possibilité pour Player de refuser le jeu.
```

## Réparation des problèmes J5/J6 actuels

La route Pauline ne doit plus tourner autour de :

```text
j’ai une photo ;
je garde une photo ;
tu veux voir ce que je garde ;
regarde le détail.
```

Elle doit avancer vers :

```text
je comprends ton désir ;
je suis moi-même compromise ;
je peux t’exposer ;
je peux m’exposer aussi ;
je peux transformer le secret en jeu ;
ou je peux te faire confiance avec ma faille.
```

## Phrases de référence

```text
Pauline ne prend pas des photos pour collectionner des preuves.
Elle les prend pour vérifier qui accepte d’être vu autrement.
```

```text
Pauline expose les autres pour ne pas être seule à avoir quelque chose à cacher.
```

```text
Le contrôle est son jeu ; la fragilité est son secret.
```
