## Nouveau principe d’arc

```text
Chaque arc doit partir du canon détaillé du personnage + du potentiel NSFW de route.
```

Cela signifie :

- Marie doit rester vie désirable, active, sociale, La Verrière, reconquête possible, désir direct possible, agency centrale dans toute route partage/NTR.
- Sandra doit partir de traces concrètes, SentryCore, Jeff, horaires décalés en français, exposition choisie, pas pression.
- Player doit être concret, actif, imparfait, travaillé par désir/honte/jalousie/regard, pas seulement spectateur.
- Mathilde doit partir du domestic taboo / proximité / Marie en sous-texte, pas d’une photo canapé legacy.
- Pauline doit partir de vertu publique / contrôle image / faille privée, pas d’omniscience photo.
- Raphaëlle doit partir de clarté, frame, consentement adulte, version choisie, pas refuge thérapeutique.
