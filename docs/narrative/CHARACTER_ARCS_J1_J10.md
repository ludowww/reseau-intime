# Arcs personnages — J1 à J10

> V0.57 (`docs/V0_57_Route_Character_Rework_Blueprint.md`) prime pour la hiérarchie des arcs.
> Les arcs ci-dessous restent utiles, mais toute lecture “symétrique” des personnages doit être réévaluée à travers V0.57.

## Objectif

Ce document évite que les personnages répètent chaque jour la même fonction.

Chaque personnage doit avoir :

```text
- une trajectoire ;
- un point de transformation ;
- une limite ;
- une action propre ;
- un risque de répétition identifié.
```

## Marie

### Fonction dramatique

Marie n’est pas seulement la personne trompée ou jalouse. Elle est le couple vivant : désir, quotidien, sécurité, lucidité, fatigue.

### Trajectoire cible

```text
J1 : couple encore naturel, familiarité, confort.
J2 : Marie existe dans le quotidien et fait entrer Mathilde indirectement.
J3 : elle sent que Player revient physiquement mais pas totalement.
J4 : elle devient visible socialement, notamment via Nico et le groupe.
J5 : elle comprend que le problème n’est pas seulement “une autre”, mais l’attention de Player.
J6 : elle demande ou provoque un acte concret, pas un énième discours.
J7 : elle agit pour elle-même ou modifie son comportement, au lieu de seulement attendre.
J8 : elle met Player devant une contradiction.
J9 : elle demande un coût réel, pas une explication.
J10 : elle sait qu’il existe un problème structurant, même sans tout savoir.
```

### Risque actuel

```text
Marie répète : pose ton téléphone / regarde-moi / tu es ailleurs.
```

### Correction

Le téléphone ne doit plus être le sujet après J5/J6, sauf conséquence nouvelle.

Exemples d’évolution :

```text
- Marie arrête de demander et fait autre chose ;
- Marie se rend elle-même visible ailleurs ;
- Marie nomme une contradiction précise ;
- Marie propose une scène de couple où Player doit être présent autrement ;
- Marie refuse une excuse mais sans refaire le même reproche.
```

## Sandra

### Fonction dramatique

Sandra est le lien ancien, doux, retenu, qui devient troublant parce qu’il n’a jamais eu de place officielle.

### Trajectoire cible

```text
J1 : reconnexion par photo / prétexte concret.
J2 : présence douce, encore défendable.
J3 : manque plus perceptible mais non revendiqué.
J4 : mauvais timing pendant la soirée.
J5 : “plus tard” devient blessure douce.
J6 : elle refuse de devenir un “plus tard” permanent.
J7 : elle change sa disponibilité : recul, silence, vraie proposition ou limite.
J8 : Player doit choisir s’il la protège ou s’il la garde dans l’ombre.
J9 : Sandra cesse d’être seulement attente ; elle formule un coût.
J10 : le lien est encore possible, mais il n’est plus neutre.
```

### Risque actuel

```text
Sandra répète : j’attends / plus tard / je ne veux pas être cachée.
```

### Correction

Après J6, Sandra ne doit plus seulement dire qu’elle attend.

Elle doit agir :

```text
- ne pas répondre immédiatement ;
- proposer un vrai moment avec une limite ;
- retirer une photo ou une confidence ;
- dire “pas comme ça” ;
- passer par une phrase très simple, pas par analyse émotionnelle.
```

## Mathilde

### Fonction dramatique

Mathilde amène le trouble dans le quotidien domestique. Elle est proche, vive, tentante, mais Marie doit rester dans le sous-texte.

### Trajectoire cible

```text
J1 : indirecte via Marie.
J2 : présence domestique installée.
J3 : première trace ambiguë.
J4 : témoin social du trouble.
J5 : question de la photo gardée ou supprimée.
J6 : la trace est nommée comme limite, dette, jeu ou danger.
J7 : elle retire, renverse ou complique le jeu.
J8 : sa loyauté envers Marie devient plus concrète.
J9 : Player doit choisir entre préserver la maison ou nourrir la tension.
J10 : Mathilde n’est plus seulement “risque excitant” ; elle est aussi conséquence domestique.
```

### Risque actuel

```text
Mathilde répète : photo / canapé / tu l’as gardée ?
```

### Correction

Après J6, ne plus centrer Mathilde sur la même photo.

Évolutions possibles :

```text
- Mathilde retire la trace ;
- elle demande à Player de ne plus jouer ;
- elle teste Player dans une scène sans photo ;
- elle devient plus loyale envers Marie ;
- elle montre qu’elle aussi a peur de ce qu’ils fabriquent.
```

## Pauline

### Fonction dramatique

Pauline voit, cadre, archive, teste. Elle ne doit pas devenir omnisciente ou simplement “la femme aux photos”.

### Trajectoire cible

```text
J1-J3 : périphérique ou absente.
J4 : hôte sociale, canal privé rare, photos.
J5 : redistribution partielle des images.
J6 : ce qu’elle garde devient rapport de force.
J7 : une faille chez Pauline apparaît ; elle n’est pas seulement en contrôle.
J8 : Player peut utiliser ou refuser son jeu.
J9 : son secret crée un risque pour elle aussi.
J10 : Pauline est dangereuse parce qu’elle est exposée autant qu’exposante.
```

### Risque actuel

```text
Pauline répète : j’ai une photo / je garde quelque chose / regarde le détail.
```

### Correction

Pauline doit perdre partiellement le contrôle à partir de J7 ou J8.

Évolutions possibles :

```text
- quelqu’un d’autre possède une image d’elle ;
- elle teste Player mais se trahit ;
- elle demande un service ;
- elle refuse d’envoyer une image et ça devient plus parlant ;
- sa puissance vient d’une faille, pas seulement du contrôle.
```

## Raphaëlle

### Fonction dramatique

Raphaëlle est le concret, la clarté, le travail, la limite. Elle voit les effets du flou sans connaître les scènes privées.

### Trajectoire cible

```text
J2 : badge, attention concrète.
J3 : fatigue, dossier, première limite.
J5 : notes incohérentes, refus d’être cachette.
J6 : clarté ou refuge pratique.
J7 : elle impose une limite de disponibilité ou de cadre.
J8 : elle peut être plus chaleureuse, mais seulement si Player choisit la clarté.
J9 : elle refuse une ambiguïté si Player veut la garder dans un rôle pratique.
J10 : elle n’est plus neutre : soit alliée de clarté, soit porte fermée.
```

### Risque actuel

```text
Raphaëlle répète : je peux écouter mais pas cacher.
```

### Correction

Après J6, elle ne doit plus reformuler la même limite.

Évolutions possibles :

```text
- elle refuse une demande concrète ;
- elle impose un cadre horaire ou professionnel ;
- elle aide sur un dossier mais coupe le flou ;
- elle devient plus personnelle sur un détail, pas par grand discours ;
- elle renvoie Player vers une action précise.
```

## Nico

### Fonction dramatique

Nico n’est pas un rival caricatural. Il est un miroir : Marie est visible ailleurs, et Player doit distinguer amour, présence et surveillance.

### Trajectoire cible

```text
J4 : présence sociale autour de Marie.
J5 : story ou commentaire qui relance la visibilité de Marie.
J6 : optionnel, pas nécessaire si le thème est déjà porté par Marie.
J7 : il peut réapparaître comme contexte social, pas antagoniste.
J8 : il révèle la manière dont Player regarde Marie quand quelqu’un d’autre la voit.
J9-J10 : il reste défendable, jamais méchant plat.
```

### Risque actuel

```text
Nico devient un raccourci pour jalousie.
```

### Correction

Nico ne doit pas porter seul la tension Marie.

À faire :

```text
- le garder rare ;
- le rendre socialement plausible ;
- éviter menace directe ;
- le faire exister surtout par l’effet qu’il produit chez Player.
```

## Player

### Fonction dramatique

Player ne doit pas être seulement un vide commenté par les autres.

Il doit redevenir un interlocuteur : il ment, répond, esquive, rassure, provoque, avoue partiellement, ou se trahit.

### Trajectoire cible

```text
J1-J3 : il découvre son trouble.
J4 : il perd le contrôle de ce qui est visible.
J5 : il découvre que ses réactions aux images créent des dettes.
J6 : il fait un acte concret.
J7 : le monde réagit à cet acte.
J8 : il se contredit.
J9 : il paie un coût.
J10 : il ne peut plus se croire simple spectateur.
```

### Règle

Chaque scène importante doit contenir au moins une vraie position de Player, même si l’UI garde les choix courts.

Exemple :

```text
Choix court : Reconnaître le flou.
Réponse réelle Player : Oui. J’étais ailleurs, et je ne peux pas te demander de faire semblant de ne pas l’avoir vu.
```

## Priorité de réécriture

```text
1. Reprendre Player.
2. Couper les boucles de motifs.
3. Donner à Marie une action différente.
4. Donner à Sandra un changement de disponibilité.
5. Donner à Mathilde une scène sans répéter la même photo.
6. Donner à Pauline une faille.
7. Donner à Raphaëlle une limite concrète nouvelle.
```
