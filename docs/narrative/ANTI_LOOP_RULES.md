# Règles anti-boucle narrative

## Objectif

Empêcher les jours de répéter les mêmes motifs sous des formes légèrement différentes.

Un motif peut revenir seulement si sa fonction change.

Règle centrale :

```text
Répéter un sujet n’est autorisé que si la scène produit une conséquence nouvelle.
```

## Boucles identifiées

### Boucle 1 — Téléphone / absence de Player

Motif utile : Player est physiquement là mais intérieurement ailleurs.

Risque : Marie répète chaque jour qu’il doit poser son téléphone.

Règle :

```text
Après qu’un personnage a nommé l’absence, il ne faut plus la renommer pareil.
Il faut montrer une action ou une conséquence.
```

Progression acceptable :

```text
J4 : Player est distrait en soirée.
J5 : Marie remarque l’absence.
J6 : Player fait ou refuse un geste concret.
J7+ : ne plus parler du téléphone sauf si une nouvelle conséquence apparaît.
```

Mauvais :

```text
Marie : Tu es encore sur ton téléphone.
Marie : Pose ton téléphone.
Marie : Je veux que tu me regardes.
```

Meilleur :

```text
Marie ne demande plus. Elle sort sans prévenir, coupe une conversation, ou change sa façon d’être disponible.
```

### Boucle 2 — Sandra / plus tard

Motif utile : Sandra n’a pas de place officielle.

Risque : elle répète qu’elle attend ou qu’elle ne veut pas être cachée.

Progression acceptable :

```text
J4 : mauvais timing.
J5 : “plus tard” blesse.
J6 : elle refuse de devenir un plus tard permanent.
J7 : elle agit autrement : recul, silence, proposition concrète, limite.
```

Mauvais :

```text
Sandra : J’ai attendu.
Sandra : Je ne veux pas être un message caché.
Sandra : Plus tard me reste dans la tête.
```

Meilleur :

```text
Sandra : Je ne t’écris pas ce soir. Pas pour punir. Pour ne pas m’habituer à attendre.
```

### Boucle 3 — Mathilde / photo gardée

Motif utile : une trace domestique devient dangereuse.

Risque : Mathilde demande plusieurs jours si la photo existe.

Progression acceptable :

```text
J3 : trace créée.
J5 : question gardée/supprimée.
J6 : limite ou conséquence.
J7+ : la photo ne doit plus être le centre ; elle devient comportement ou coût.
```

Mauvais :

```text
Mathilde : Tu l’as gardée ?
Mathilde : Tu l’as supprimée ?
Mathilde : C’est dangereux, non ?
```

Meilleur :

```text
Mathilde voit que Player n’a pas supprimé, ou décide elle-même de couper le jeu.
```

### Boucle 4 — Pauline / photos gardées

Motif utile : Pauline contrôle une partie du visible.

Risque : Pauline répète qu’elle garde encore quelque chose.

Progression acceptable :

```text
J4 : elle prend / cadre.
J5 : elle redistribue.
J6 : garder devient rapport de force.
J7+ : une faille apparaît ; elle ne contrôle plus tout.
```

Mauvais :

```text
Pauline : J’ai une photo.
Pauline : J’en garde une autre.
Pauline : Tu veux savoir ce que je garde ?
```

Meilleur :

```text
Pauline refuse d’envoyer une image parce qu’elle-même y apparaît trop.
```

### Boucle 5 — Raphaëlle / refuge-cachette

Motif utile : Raphaëlle refuse d’être un endroit pratique pour le flou de Player.

Risque : elle reformule la même limite en termes abstraits.

Progression acceptable :

```text
J2 : observation concrète.
J3 : première limite.
J5 : refus de cachette.
J6 : clarté ou refuge pratique.
J7+ : limite concrète de disponibilité ou d’action.
```

Mauvais :

```text
Raphaëlle : Je peux écouter mais pas cacher.
Raphaëlle : Je ne suis pas un refuge.
Raphaëlle : Je veux de la clarté.
```

Meilleur :

```text
Raphaëlle : Je ne reste pas après 19h pour un dossier que tu n’ouvres pas vraiment.
```

## Test anti-boucle avant écriture

Avant chaque scène, répondre :

```text
1. Ce sujet a-t-il déjà été traité ?
2. Si oui, qu’est-ce qui change cette fois ?
3. Est-ce une action, une conséquence ou juste une reformulation ?
4. Le personnage dit-il quelque chose qu’il a déjà dit sous une autre forme ?
5. Player répond-il vraiment ou reste-t-il une absence ?
```

Si la réponse à la question 3 est “reformulation”, supprimer ou réécrire la scène.

## Test de fin de journée

À la fin de chaque jour, au moins une phrase doit être vraie :

```text
- un personnage n’est plus disponible comme avant ;
- une preuve a changé de statut ;
- Player a menti ou avoué quelque chose de précis ;
- une relation a gagné ou perdu une possibilité ;
- un personnage a agi sans attendre Player ;
- une porte s’est ouverte ou fermée concrètement.
```

Si aucune n’est vraie, la journée tourne en rond.

## Règles de silence

Tous les personnages ne doivent pas écrire chaque jour.

Le silence peut être une conséquence.

Exemples :

```text
Sandra ne répond pas J7 parce que J6 l’a blessée.
Marie ne relance pas le téléphone ; elle fait autre chose.
Mathilde évite Player au lieu de plaisanter.
Raphaëlle limite la conversation au travail strict.
Pauline disparaît après avoir perdu un peu le contrôle.
```

## Règles de motif

Un motif doit changer de catégorie :

```text
observation → choix → conséquence → coût
```

Exemple téléphone :

```text
observation : tu es ailleurs.
choix : tu poses ou non le téléphone.
conséquence : Marie cesse de demander.
coût : Player perd un moment ou doit réparer autrement.
```

Exemple photo Mathilde :

```text
observation : la photo existe.
choix : garder / supprimer / mentir.
conséquence : Mathilde change de ton.
coût : Marie devient plus présente dans la scène ou la tension baisse.
```

## Application à J5/J6

Avant réécriture :

```text
- lister tous les motifs déjà traités J1-J4 ;
- supprimer les scènes J5/J6 qui ne font que reformuler ;
- choisir un seul axe fort par personnage ;
- ajouter des réponses Player visibles ;
- donner au moins une action propre à Marie ou Sandra ;
- faire apparaître une faille chez Pauline ou repousser son mystère ;
- transformer Raphaëlle en limite concrète, pas en discours de clarté répété.
```

## Règle finale

```text
Si une scène peut être résumée par une phrase déjà vraie la veille,
elle doit être supprimée ou transformée.
```
