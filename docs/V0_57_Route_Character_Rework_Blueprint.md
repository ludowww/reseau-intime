# V0.57 — Route & Character Rework Blueprint

> Document de refonte documentaire.  
> Source de vérité pour préparer la réécriture J1→J9.  
> Aucun fichier runtime, aucun JSON narratif, aucune conversation jouable ne sont modifiés ici.

## 1. Intention de refonte

La refonte vise à recentrer le jeu autour du couple Player / Marie, avec des routes atteignables, moins dispersées, et une cohérence plus forte entre personnages, tensions et choix.

Phrase directrice :

> Le jeu raconte comment Player choisit de gérer les failles de son couple avec Marie : en réparant, en cachant, en ouvrant, en provoquant, ou en laissant les autres entrer dans l’intimité du couple.

Objectifs de cadrage :
- réduire l’effet de panel dispersé ;
- garder des routes lisibles et atteignables ;
- renforcer les conséquences relationnelles ;
- faire de Marie le centre de gravité du jeu ;
- préparer une réécriture progressive de J1 à J9, sans ajouter J10 ici.

## 2. Architecture centrale

Le centre dramatique est le couple Player / Marie.

Marie n’est pas une route parmi d’autres : elle est le baromètre principal de tout le système. Chaque choix doit faire bouger au moins un de ces paramètres :
- réparation du couple ;
- disponibilité amoureuse ;
- ouverture progressive ;
- éloignement ;
- sorties plus fréquentes ;
- tromperie / NTR possible ;
- réaction aux secrets de Player ;
- réaction à Nico, Sandra, Mathilde et Pauline.

Conséquence de cadrage :
- les autres personnages doivent d’abord être lus à travers l’état du couple ;
- les routes secondaires ne doivent pas se comporter comme des histoires indépendantes ;
- un même personnage peut changer de fonction selon l’état du couple.

## 3. Priorités de personnages

### Priorité 1 : Marie, Sandra, Nico, Mathilde

Pourquoi :
- Marie : axe central, mesure la santé du couple et la direction réelle du jeu ;
- Sandra : axe émotionnel le plus intime, utile pour le manque, la confidence et la nostalgie ;
- Nico : axe miroir / tentation / complice, utile pour la pression, la comparaison et les dérives ;
- Mathilde : axe domestique immédiat, parfait pour l’intrusion dans la maison et la tension de proximité.

Ces quatre personnages construisent le noyau dramatique utile à la réécriture J1→J9.

### Priorité 2 : Pauline

Pourquoi :
- Pauline dépend déjà d’un cercle social plus structuré ;
- elle gagne en force quand les traces, les photos, les sorties et les non-dits sont déjà installés ;
- elle fonctionne mieux comme intensification du réseau que comme point d’entrée principal.

### Priorité 3 / différée : Raphaëlle

Pourquoi :
- Raphaëlle est une présence extérieure et professionnelle, donc plus efficace quand le couple et les routes internes sont déjà lisibles ;
- elle sert mieux d’alibi, de clarté ou de contrepoint que de moteur principal du démarrage ;
- sa fonction demande une base narrative plus propre avant d’être pleinement exploitée.

## 4. Rôles dramatiques définitifs

### Marie

Marie est le centre du couple, le baromètre des choix, et le pivot amour / réparation / ouverture / éloignement / tromperie.

Routes Marie :
- route amour / réparation / exclusive ;
- route ouverte / couple ouvert / intégration progressive ;
- route tromperie / NTS si Player la néglige, cache des choses, sort souvent, ou si l’influence Pauline normalise l’idée d’un écart.

### Sandra

Sandra est la route émotionnelle : manque, confidence, retrouvailles, amour inavoué.

Routes Sandra :
- confidence / amour inavoué / fantasme ;
- tentation plus assumée ;
- ouverture possible plus tardive, compatible seulement si confiance et cadre émotionnel.

### Nico

Nico est l’ami / confident masculin, miroir vicieux de Player, tentateur, complice ou rival.

Routes Nico :
- confident / tentateur léger ;
- rival manipulateur si Player perd le contrôle ou néglige le couple ;
- partage / libertinage / trio dans un cadre plus assumé ;
- complice sombre autour des photos, informations, mises en scène ou manipulation.

Nico peut aller loin, y compris dans la manipulation, mais les routes sombres doivent rester risquées, instables et coûteuses.

### Mathilde

Mathilde est la route domestique principale. Elle incarne l’intrusion dans la maison, le territoire intime, la rivalité ou l’intégration.

Routes Mathilde :
- voyeur / vu-pas-vu / ambiguïté ;
- trahison / rivalité / désir de prendre la place de Marie ;
- ouverture / intégration avec Marie / trio / polyamour / quatuor possible plus tard avec Nico.

### Pauline

Pauline est une priorité 2 progressive. Elle est proche du cercle de Marie et porte la tension sociale, le secret, la tentation et un potentiel rôle de confidente / détective.

Déclencheur recommandé :
- première sortie Marie / Pauline ;
- deuxième sortie piscine / spa / sauna ;
- photos légères de Marie et Pauline ;
- un ou deux messages directs de Pauline à Player ;
- rapprochement discret ou semi-discret par messages ;
- recontacts plus fréquents les jours suivants.

Routes Pauline :
- tentation directe ;
- confidence / détective si Marie part vers tromperie ;
- alliée ambiguë ;
- route trio / manipulation avec Nico plus tardive et sombre.

### Raphaëlle

Raphaëlle est une priorité 3 différée, avec une présence extérieure / professionnelle.

Rôle :
- alibi crédible pour travail, sorties, déplacements ;
- présence extérieure qui ne perturbe pas trop le début ;
- quelques photos peu suggestives ou légèrement intrigantes ;
- route romantique / sexuelle plus tardive via séminaire, déplacement, afterwork ou contexte pro ;
- faible impact sur les autres routes au départ.

## 5. Axes globaux de choix

Les axes qui pilotent les routes sont :
- transparence vs secret ;
- réparation du couple vs éloignement ;
- ouverture consentie vs dérive manipulatrice ;
- intimité émotionnelle vs excitation / provocation ;
- influence de Nico ;
- tension domestique Mathilde ;
- proximité Sandra ;
- disponibilité Pauline ;
- extérieur professionnel Raphaëlle.

## 6. États possibles du couple Marie / Player

### 1. Couple réparé

Le couple se recompose autour de la vérité, du soin et d’un recentrage volontaire.

Effet principal :
- Marie reste centrale et lisible ;
- Sandra et Raphaëlle deviennent plus compatibles si leur fonction reste claire ;
- les routes manipulatrices perdent de la puissance ou deviennent plus coûteuses.

### 2. Couple ouvert

Marie et Player nomment une ouverture ou une exploration avec règles.

Effet principal :
- les routes secondaires coexistent plus proprement ;
- le cadre devient aussi important que le désir ;
- Pauline et Nico peuvent intensifier le système sans tout casser.

### 3. Couple fissuré

Le secret, la distance ou le mensonge prennent plus de place que la réparation.

Effet principal :
- Marie devient plus vigilante ;
- Pauline gagne du pouvoir avec les traces ;
- Mathilde devient plus risquée ;
- Sandra peut se sentir cachée ;
- Raphaëlle recule si elle sert de cachette.

### 4. Couple corrompu / tromperie / manipulation

Le mensonge actif, la mise en scène et la dérive de contrôle dominent.

Effet principal :
- Nico devient un moteur de perte de contrôle ;
- Pauline peut devenir dangereuse ;
- les photos et preuves prennent une valeur stratégique ;
- Marie peut basculer vers la reprise de pouvoir, la vengeance ou la rupture froide.

## 7. Règles des routes sombres

Les routes sombres / manipulatrices sont autorisées.

Mais elles doivent respecter ces règles :
- elles ne sont pas des récompenses propres et sans coût ;
- elles sont risquées, instables et avec conséquences possibles ;
- Nico peut être moteur de perte de contrôle ;
- Player peut devenir manipulateur, mais le jeu doit reconnaître cette dérive ;
- un gain immédiat ne doit pas effacer le coût relationnel.

En pratique :
- plus une route sombre avance, plus elle doit laisser des traces ;
- plus une route manipule, plus elle doit fragiliser le couple ou la stabilité sociale ;
- les routes sombres ne doivent jamais ressembler à une simple version “plus sexy” d’une route normale.

## 8. Règles des photos / preuves / traces

Trois catégories doivent être distinguées :

1. **Photos consenties**
   - prises avec accord ou intention claire ;
   - servent la confiance, la mise en relation ou la mémoire ;
   - peuvent nourrir une route lumineuse ou ouverte.

2. **Photos ambiguës / vu-pas-vu / personne qui laisse faire**
   - zone intermédiaire ;
   - elles alimentent le trouble, le non-dit et la tension ;
   - elles peuvent basculer vers le risque si elles circulent ou sont réinterprétées.

3. **Photos non consenties ou diffusées**
   - appartiennent aux routes sombres à risque majeur ;
   - elles doivent avoir des conséquences possibles ;
   - elles ne doivent pas être traitées comme une simple récompense érotique normale.

Règle de fond :
- la photo doit être un fait narratif, pas seulement un contenu visuel ;
- une photo, une capture ou une story doit produire mémoire, tension ou conséquence.

## 9. Début “canon” recommandé

Le feeling par défaut du début doit être :
- Marie fragile mais encore aimante ;
- Sandra réapparaît émotionnellement à distance ;
- Mathilde crée une tension domestique ;
- Nico arrive comme confident / tentateur léger ;
- Pauline apparaît via Marie ;
- Raphaëlle reste en fond extérieur / professionnel.

Ce départ doit donner un centre émotionnel clair sans saturer le casting d’emblée.

## 10. Timing des locks de route

Verrouillage à garder :
- pas de verrouillage fort avant environ J10 / J12 ;
- J1→J9 doivent surtout colorer la partie ;
- les choix précoces préparent les routes mais ne doivent pas trop fermer le jeu ;
- les vrais locks viennent plus tard.

Conséquence :
- la réécriture J1→J9 doit préparer les orientations, pas trancher les issues ;
- les jours précoces doivent rester lisibles, pas saturés de conséquences définitives.

## 11. Compatibilité avec l’existant J1→J9

### Peut être conservé
- la base Marie / Sandra en J1 ;
- la montée du réseau social et domestique en J2→J4 ;
- l’importance des photos, traces et preuves ;
- la logique “une journée = une poussée de tension” ;
- les repères de rythme déjà validés.

### Doit être réécrit
- les sections qui dispersent trop les axes sans hiérarchie claire ;
- les jours où une route secondaire prend trop de place trop tôt ;
- les passages qui traitent les preuves comme de simples récompenses ;
- les scènes où Marie n’est plus le baromètre principal ;
- les formulations qui supposent des routes assumées trop tôt.

### Doit être déplacé
- les révélations fortes sur Pauline ;
- les versions plus lourdes de Nico ;
- les accélérations d’ouverture / tromperie ;
- les configurations plus complexes de trio / quatuor ;
- les enjeux de type “route consumée” plutôt que “route amorcée”.

### Devient obsolète
- les plans qui traitent J1→J9 comme un simple empilement de scènes ;
- les matrices qui gardent toutes les routes à niveau égal ;
- les documents qui laissent Raphaëlle trop centrale au démarrage ;
- les grilles qui n’expriment pas explicitement la priorité de Marie comme centre de compatibilité.

### Doit être clarifié
- la différence entre route amorcée, route compatible, route verrouillée et route risquée ;
- le rôle exact des photos consenties / ambiguës / non consenties ;
- le statut de Nico comme miroir, rival ou complice ;
- la place de Pauline comme montée sociale et non comme moteur initial ;
- le moment à partir duquel l’ouverture du couple devient réellement un cadre.
